# 관리자 페이지에 로그인하는 회사코드를 잊어버린 경우 해결 방법

### 개요 <a href="#a03" id="a03"></a>

관리자 페이지에 로그인할 때 필요한 회사코드를 잊어버린 경우 관리자의 '아이디'와 '이메일 주소'를 입력하여 회사코드를 확인할 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. 로그인 페이지의 왼쪽 하단에서 '회사코드/아이디를 분실하셨나요?'를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (689).png" alt=""><figcaption></figcaption></figure>



2. 회사코드 찾기 탭에서 관리자의 아이디와 등록한 이메일 주소를 입력하고 확인 버튼을 클릭합니다.\
   회사코드가 이메일로 전송됩니다.

<figure><img src="../../.gitbook/assets/image (690).png" alt=""><figcaption></figcaption></figure>

&#x20;    &#x20;

3. 확인 버튼을 클릭하여 화면을 닫습니다.

<figure><img src="../../.gitbook/assets/image (691).png" alt=""><figcaption></figcaption></figure>
