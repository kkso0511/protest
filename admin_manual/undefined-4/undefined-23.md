# 특정 폴더에서 링크 생성 기능을 활성화하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Shared Box 폴더를 선택하고 '폴더 속성'에서 링크를 만드는 기능을 사용하도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼 에서는 환경 설정 메뉴의 Shared Box 및 Connect 관리 메뉴의 설정 방법을 설명하고 있으며, DLP 관리와 문서 관리 메뉴도 동일한 절차로 설정하실 수 있습니다.
* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정'을 사용할 수 있습니다.
* Shared Box와 DLP 폴더의 링크 설정은 기본으로 사용함으로 설정되어 있습니다.
* 링크 및 링크 설정은 최상위 계층의 폴더에 설정하며, 설정한 내용은 하위 폴더에 그대로 적용됩니다.
* 링크 및 링크 설정에는 [관리자 페이지에서 폴더를 생성하는 방법](https://help.directcloud.net/admin_manual/undefined-4/undefined)의 파일 링크 생성 설정이 반영됩니다.
* 링크 생성 기능은 마스터, 전체권한, 편집자 접근 권한이 부여된 사용자가 이용할 수 있습니다.
* 링크 생성 기능은 모든 애플리케이션에서 이용할 수 있습니다.
* Shared Box의 경우, [워크플로우를 활성화하는 방법](https://help.directcloud.net/admin_manual/undefined-7/undefined)의 절차에서 '결재선 미지정 사용자' 항목의 '기능을 제한함'을 선택하면, 신청자를 제외한 사용자와 Guest는 링크를 생성할 수 없게 됩니다.
* DirectCloud-CONNECT를 이용할 수 있는 경우, 관리 페이지에 Connect 관리 메뉴가 표시됩니다.
* Connect 폴더의 폴더 속성에서는 링크를 설정할 수 없습니다.
* DirectCloud-SHIELD DLP를 계약한 경우, 관리 페이지에 DLP 관리 메뉴가 표시됩니다。
*   DLP 폴더에서 링크 생성 기능을 사용하려면, DLP 설정에서 사용자에게 링크 생성을 허용하도록 설정하는 방법의 절차에 따라 SHIELD > DLP 설정 메뉴의 파일 반출 설정 탭에서 링크 설정의 사용함에 체크가 되어 있어야 합니다.

    이 설정이 되어 있지 않은 경우, DLP 관리의 폴더 속성에 링크 설정이 표시되지 않습니다.
* Warm Storage를 이용할 수 있다면 관리 페이지에 Warm Storage 관리 메뉴가 표시됩니다.
* Cold Storage를 계약한 경우, 관리 페이지에 Cold Storage 관리 메뉴가 표시됩니다.
* Warm Storage와 Cold Storage에서는 링크를 생성할 수 없기 때문에, Warm Storage 관리 및 Cold Storage 관리 메뉴에서 폴더를 선택하더라도 폴더 속성에 링크는 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Shared Box 폴더에서 링크 생성 사용 설정**
{% endhint %}

1. '공유 설정' > 'Shared Box' 관리 메뉴를 선택하고 링크 생성을 활성화할 폴더를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1963).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 폴더 속성 설정에서 링크의 수정 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1964).png" alt=""><figcaption></figcaption></figure>



3. '링크'가 '사용 안 함'으로 설정되어 있으면 '사용'을 선택하고 '저장' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1965).png" alt=""><figcaption></figcaption></figure>
