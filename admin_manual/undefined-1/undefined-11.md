# 공유 링크 생성 화면에서 수신자의 수신 방법을 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 ‘링크 옵션 설정’에서 공유 링크의 폴더 및 파일에 대해 공유 링크 수신자가 수행할 수 있는 작업(수신 방법)을 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* ‘링크 옵션 설정’은 Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자에게도 적용되지만, Connect User에는 적용되지 않습니다.
*   공유 링크 수신 화면에서 미리 볼 수 있는 확장자는 다음과 같습니다.

    | 문서     | xls、xlsx、doc、docx、ppt、pptx、pdf、txt |
    | ------ | ---------------------------------- |
    | 이미지    | jpg、jpeg、png、bmp、gif               |
    | 동영상    | mp4、m4a                            |
    | 음악, 음성 | mov、mp3                            |

***

### 절차 <a href="#a05" id="a05"></a>

**\[관리자 페이지에서의 설정]**

1. '보안 정책' > '반출 정책' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1652).png" alt=""><figcaption></figcaption></figure>



2. '링크생성 시 기본값 설정'의 '수신방법 설정항목 제한'에서 공유 링크 생성 화면에서 선택 대상이 되는 접근 권한을 체크합니다.

<figure><img src="../../.gitbook/assets/image (1681).png" alt=""><figcaption></figcaption></figure>

<table><thead><tr><th width="300">항목</th><th>설명</th></tr></thead><tbody><tr><td>링크 수신자가 미리보기/다운로드 가능</td><td>공유 링크 수신자는 파일 미리보기(브라우저) 및 폴더 파일을 다운로드할 수 있습니다.</td></tr><tr><td>링크 수신자에게 다운로드만 제공</td><td>공유 링크 수신자는 폴더 파일 다운로드만 가능합니다.</td></tr><tr><td>링크 수신자에게 미리보기만 제공</td><td>공유 링크 수신자는 파일 미리보기(보기)만 가능합니다.</td></tr></tbody></table>



3. 링크 옵션 설정에서 저장 버튼을 클릭합니다.



**\[사용자 페이지에서 설정]**

1. 공유 링크 생성 화면의 '수신방법'에서 관리 페이지 설정이 반영되는지 확인합니다.
