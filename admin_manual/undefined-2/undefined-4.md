# 그룹을 삭제하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 관리 메뉴에서 등록 되어있는그룹을 삭제하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '계정 관리' 항목에 접근할 수 있습니다.
* 그룹을 일괄 삭제할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '계정 관리' > '사용자' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (503).png" alt=""><figcaption></figcaption></figure>



2. 삭제할 그룹을 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 삭제를 클릭합니다. 또는 삭제할 그룹을 선택하고 상단의 삭제 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (504).png" alt=""><figcaption></figcaption></figure>



3. 그룹 삭제 확인 화면에서 확인 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (505).png" alt=""><figcaption></figcaption></figure>
