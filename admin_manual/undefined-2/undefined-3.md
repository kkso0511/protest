# 그룹에 할당된 사용 권한을 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 관리 메뉴에서 그룹에 할당된 사용 권한 설정을 변경하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '계정 관리' 항목에 접근할 수 있습니다.
* 사용자 단위로 기능 제한을 변경하려는 경우에는 [특정 사용자에게 예외적인 사용 권한을 부여하는 방법](https://help.directcloud.net/admin_manual/undefined-3/undefined-5)의 절차에 따라 설정해 주세요.
*   여러 그룹에 소속된 사용자의 경우, 권한 세트가 합산되어 적용됩니다.

    예를 들어,&#x20;

    * ‘Shared Box’, ‘링크’, ‘파일첨부 메일발송’ 기능 제한이 할당된 ‘A부’
    * ‘Shared Box’, ‘링크’, ‘업로드 요청’ 기능 제한이 할당된 ‘B부’

    두 그룹 모두에 소속된 사용자에게는 ‘Shared Box’, ‘링크’, ‘파일첨부 메일발송’, ‘업로드 요청’ 기능 제한이 적용됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '계정 관리' > '사용자' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (500).png" alt=""><figcaption></figcaption></figure>



2. 변경하려는 그룹을 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 수정을 클릭합니다. 또는 변경할 그룹을 선택하고 상단의 수정 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (501).png" alt=""><figcaption></figcaption></figure>



3. 사용 권한을 변경하고 확인 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (502).png" alt=""><figcaption></figcaption></figure>
