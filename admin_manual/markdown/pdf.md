# 관리 페이지에서 PDF 설명서를 다운로드하는 방법(비표시)

### 개요 <a href="#a03" id="a03"></a>

DirectCloud 관리 페이지의 오른쪽 상단에 표시되는 "?" 버튼(도움말 버튼)에서 웹 매뉴얼을 보거나 튜토리얼, 빠른 매뉴얼, PDF 매뉴얼을 다운로드하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 다이렉트 클라우드 홈페이지에서 PDF 매뉴얼을 다운로드하려면 아래 URL로 이동하세요.\
  [https://directcloud.jp/support/manual](https://directcloud.jp/support/manual)
* DirectCloud 지원 센터를 웹 브라우저에서 직접 보려면 다음 URL을 방문하세요.\
  [https://help.directcloud.jp/s/](https://help.directcloud.jp/s/)

### 절차 <a href="#a05" id="a05"></a>

#### 웹 매뉴얼 표시

1. &#x20;화면 오른쪽 상단에 표시된 "?" 버튼(도움말 버튼)에 커서를 놓고 표시된 메뉴에서 "지원 센터"를 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tTU9)

2. 웹 브라우저의 다른 탭에서 웹 설명서를 제공하는 [DirectCloud 지원 센터](https://help.directcloud.jp/s/) 의 톱 페이지가 표시됩니다.\
   사용자 매뉴얼 또는 관리자 매뉴얼을 클릭하여 기능별 분류에서 웹 매뉴얼을 찾거나 검색 양식에 키워드를 입력하여 웹 매뉴얼을 검색할 수 있습니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tTVl)

#### PDF 매뉴얼, 빠른 매뉴얼, 튜토리얼 다운로드

1. 화면 오른쪽 상단에 표시된 "?" 버튼(도움말 버튼)에 커서를 놓고 표시된 메뉴에서 다운로드할 파일에 맞는 메뉴를 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tFPa)

| 메뉴      | 설명                                                                     |
| ------- | ---------------------------------------------------------------------- |
| PDF 매뉴얼 | DirectCloud의 사양이나 순서 등을 1권에 정리한 PDF 매뉴얼을 제공하고 있습니다.                    |
| 퀵 매뉴얼   | DirectCloud를 사용하기 시작했을 때 도움이 되는 순서 등을 정리한 복수의 메뉴얼을 제공하고 있습니다.          |
| 튜토리얼    | DirectCloud의 기능으로 실현할 수 있는 환경을, 구체적인 이용 장면을 섞어 설명한 문서와 동영상을 제공하고 있습니다. |

메뉴에 따라 페이지에 다운로드할 수 있는 파일의 미리보기 이미지가 목록에 표시됩니다.

2. 탐색하려는 도움말 문서의 링크를 클릭합니다.

❶ PDF 매뉴얼

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tPTm)

❷ 퀵 매뉴얼

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tV1J)

❸ 자습서

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tJY6)

도움말 문서 다운로드 페이지가 표시됩니다.

3. 다운로드하려는 파일을 체크하고 상단의 '다운로드' 버튼을 클릭합니다.

❶ PDF 매뉴얼

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006wB4U)

❷ 퀵 매뉴얼

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006tRSM)

❸ 자습서

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000027IL\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006w1Ba)

도움말 문서가 다운로드됩니다.
