# Guest의 링크 생성 가능 여부를 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Guest 관리 메뉴에서 Guest에게 링크 생성 권한을 허용할지 여부를 설정하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '계정 관리'를 사용할 수 있습니다.
* 링크 생성 기능은 기본적으로 사용함으로 설정되어 있습니다.
* Guest에게 링크 생성이 가능한 접근 권한(전체권한 또는 편집자)이 부여되어 있는 경우라도, 이 매뉴얼의 절차에 따라 Guest 기능 제한의 링크 생성 항목을 사용하지 않음으로 설정하면 Guest는 링크를 생성할 수 없게 됩니다.
* 이 매뉴얼의 절차에 따라 링크 생성을 사용하지 않음으로 설정하면 Guest는 Link History 메뉴를 사용할 수 없게 됩니다. \
  또한 Guest 화면의 다음 위치에서 링크 생성 버튼, 마우스 오른쪽 버튼을 눌렀을 때 표시되는 링크 생성 메뉴, 생성된 링크 수를 나타내는 Link History 아이콘이 표시되지 않게 됩니다.
  * 폴더 및 파일 목록
  * 검색 결과
  * 최근 사용한 파일
* Guest 기능 제한은 Guest 개별로 설정할 수 없습니다.

***

### 절차

1. '계정관리' > 'Guest' 메뉴를 선택한 뒤, Guest 설정 탭을 표시합니다.

<figure><img src="../../.gitbook/assets/image (369).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하여 Guest 기능 예외 설정을 표시합니다.

<figure><img src="../../.gitbook/assets/image (370).png" alt=""><figcaption></figcaption></figure>



3. Guest 기능 제한에서 링크 생성 항목을 설정한 뒤 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (371).png" alt=""><figcaption></figcaption></figure>
