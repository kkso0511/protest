# 일반 관리자 권한을 해제하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 관리자 메뉴에서 사용자에게 설정된 일반 관리자의 권한을 해제하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '계정 관리'를 사용할 수 있습니다.
* ‘관리자’ 메뉴는 ‘계약 담당 관리자’ 권한을 가진 사용자 외에는 표시되지 않습니다.
* 계약 담당 관리자의 계정 정보를 알 수 없어 권한을 추가할 수 없는 경우에는 일반 관리자 계정으로 로그인한 후, ‘사용자’ > ‘계정 관리’ 메뉴에서 사용자의 편집 버튼을 클릭하고, ‘사용자 변경’ 화면에서 ‘URL 복사’를 클릭하여 비밀번호를 변경해 주시기 바랍니다.
* 일반 관리자 권한을 해제하더라도 사용자 ID는 삭제되지 않습니다.
* ‘사용자 변경’ 화면에서는 일반 관리자의 사용자 ID를 변경할 수 없습니다. 사용자 ID를 변경하려면 [사용자 ID를 변경하는 방법](https://help.directcloud.net/admin_manual/undefined-3/id-1)의 절차를 따라 주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '계정 관리' > '관리자 추가' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1803).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 관리자 추가/관리 설정에서 일반 관리자 권한을 해제하려는 사용자의 휴지통 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1804).png" alt=""><figcaption></figcaption></figure>



3. 확인 화면에서 '확인' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (63).png" alt=""><figcaption></figcaption></figure>



4. 일반 관리자 권한을 해제한 사용자가 '관리자 추가/관리' 목록에서 삭제되었는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (1806).png" alt=""><figcaption></figcaption></figure>
