# 사용자가 이용할 수 있는 애플리케이션을 제한하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 '접속 제어 메뉴'의 '접속프로그램 제한' 설정에서 사용자에게 사용을 허용할 DirectCloud 애플리케이션을 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* 애플리케이션 이용 제한은 Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자에게도 적용됩니다.
* 모바일 버전 Web 브라우저는 2024년 12월 10일 업데이트에서 베타 버전으로 제공되기 시작했습니다.
* 애플리케이션 이용 제한 설정의 허가하는 애플리케이션에서는 기본적으로 PC 버전 Web 브라우저, PC Agent, DirectCloud 드라이브, iOS 애플리케이션, Android 애플리케이션을 사용할 수 있도록 설정되어 있습니다.&#x20;

***

### 절차 <a href="#a05" id="a05"></a>

1. '보안 정책' > '접속 제어'를 선택하고 '접속프로그램 제한' 탭을 선택합니다.

2. 사용자가 이용을 허용하지 않을 애플리케이션의 체크를 해제하고 저장 버튼을 클릭합니다.
