# 휴지통을 사용하도록 설정하고, 보관 기간을 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는, '용량 관리' > '스토리지 현황' 메뉴의 '휴지통' 설정에서, 사용자의 휴지통을 사용할 수 있도록 설정하는 방법과, 사용자 휴지통 및 회사 휴지통에서 보관 기간을 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '스토리지' 항목을 사용할 수 있습니다.
* 계약 플랜이 어드밴스드 이상인 경우, 사용자 휴지통의 보관 기간으로 지정할 수 있는 항목은, '1일', '3일', '5일', '7일', '30일', '제한 없음'입니다.
* 계약 플랜이 스탠다드인 경우에는, '제한 없음'을 지정할 수 없습니다.
* 회사 휴지통은 비활성화할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**사용자의 휴지통을 사용 설정하고 보관 기간 설정**
{% endhint %}

1. '용량 관리' > '스토리지 현황' 메뉴를 선택합니다.

<div align="left" data-full-width="false"><figure><img src="../../.gitbook/assets/image (2116).png" alt=""><figcaption></figcaption></figure></div>



2. 화면을 아래로 스크롤하고 휴지통 설정에서 사용자 휴지통에서 사용을 선택하고 드롭다운 목록에서 저장할 기간을 선택합니다.

<figure><img src="../../.gitbook/assets/image (2115).png" alt=""><figcaption></figcaption></figure>



3. 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2114).png" alt=""><figcaption></figcaption></figure>



**\[사용자 페이지의 휴지통 설정 확인]**

1. 사용자 페이지의 휴지통 메뉴에서 설정한 보관 기간이 반영되는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (2113).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**회사 휴지통 보관 기간 설정**
{% endhint %}

1. '용량 관리' > '스토리지 현황' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (2112).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 휴지통 설정에서 '회사 휴지통'의 보관 기간에 있는 드롭다운 목록에서 저장할 기간을 선택합니다.

<figure><img src="../../.gitbook/assets/image (2111).png" alt=""><figcaption></figcaption></figure>



3. 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2110).png" alt=""><figcaption></figcaption></figure>



**\[관리자 페이지의 휴지통 설정 확인]**

1. 휴지통 설정의 왼쪽 하단에 있는 '회사 휴지통' 버튼을 클릭합니다.\
   회사 휴지통 설정이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (2108).png" alt=""><figcaption></figcaption></figure>

&#x20;    &#x20;

2. 설정한 보존 기간이 반영되는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (2109).png" alt=""><figcaption></figcaption></figure>
