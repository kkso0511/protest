# 삭제된 사용자의 내 상자/휴지통 폴더 및 파일을 확인하는 방법 (한국미제공) (감사옵션)

### 개요 <a href="#a03" id="a03"></a>

DirectCloud 감사가 비활성화된 경우 사용자를 삭제하면 내 박스와 공유 및 DLP 폴더의 휴지통에 저장된 폴더와 파일이 관리자의 휴지통으로 이동합니다. 관리자 휴지통으로 이동한 폴더와 파일의 소유자를 식별할 수 없습니다.\
반면에 DirectCloud 감사를 사용하면 사용자를 삭제해도 위의 폴더와 파일이 관리자 휴지통으로 이동하지 않습니다. 폴더와 파일은 내 박스와 사용자의 휴지통에 저장된 상태에서 스토리지 > 삭제된 사용자 내 박스 관리 메뉴로 이동합니다.\
이 섹션에서는 DirectCloud 감사가 활성화된 경우 삭제된 사용자의 폴더와 파일을 확인하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 [관리자의 기능 제한](https://boxmanager.directcloud.jp/setting/manager) 으로 "저장소"를 선택해야 합니다.
* 계약 계획이 엔터프라이즈인 경우 DirectCloud의 무료 옵션인 DirectCloud 감사를 사용할 수 있습니다.
* DirectCloud 감사를 한 번 활성화하면 비활성화된 상태로 되돌릴 수 없습니다.
* DirectCloud 감사를 사용하면 삭제된 사용자의 ID를 다시 활성화할 수 없습니다. 또한 삭제된 사용자 ID로 사용자를 다시 추가할 수 없습니다.
* 게스트를 삭제하면 게스트 휴지통의 "공유" "DLP" 탭에 저장된 폴더와 파일이 관리자 휴지통으로 이동합니다.
* Connect User를 삭제하면 Connect User 휴지통의 Connect 탭에 저장된 폴더와 파일은 관리자 휴지통으로 이동하지만 '내 상자/공유' 탭에 저장된 폴더 및 파일 관리자의 휴지통으로 이동하지 않습니다.
* 삭제된 사용자 내 상자 관리 메뉴에서 확인할 수 있는 사용자의 내 상자 및 휴지통은 각각 저장 용량 메뉴의 내 상자 및 사용자의 휴지통에 추가됩니다.
* 감사 담당자는 관리자의 모든 기능을 이용할 수 있습니다. [일반 관리자가 사용할 수 있는 메뉴를 제한하는 방법](https://help.directcloud.jp/s/article/operation-of-administrator-function11122) 에 따라 감사 담당자의 관리 페이지에 표시되는 메뉴를 제한할 수 없습니다.

### 절차 <a href="#a05" id="a05"></a>

#### 내 상자에 저장된 폴더와 파일 확인

1. 스토리지 > 삭제된 사용자 내 상자 관리 메뉴를 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAKs)

2. 필요한 경우 삭제된 사용자 내 상자 관리에서 검색 또는 정렬을 수행하여 폴더와 파일을 확인할 삭제된 사용자를 표시합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAL2)

| No. | 설명                                                                                                                                 |
| --- | ---------------------------------------------------------------------------------------------------------------------------------- |
| ❶   | <p>사용자의 휴지통에 저장된 폴더와 파일을 다음 내림차순/오름차순으로 정렬합니다.</p><ul><li>신청일(사용자 등록일)</li><li>할당 용량</li><li>내 상자 사용 용량</li><li>버전 이력 용량</li></ul> |
| ❷   | 사용자의 휴지통에 저장된 폴더와 파일을 필터링하려면 '사용자 이름', '사용자 ID' 또는 '휴대폰'을 입력합니다.                                                                   |
| ❸   | 사용자의 휴지통에 저장된 폴더와 파일을 표시할 건수를 "10", "25", "50", "100" 중에서 선택합니다.                                                                   |

3. 폴더와 파일을 확인할 삭제된 사용자의 사용 현황 열에 표시되는 파일 목록 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aA0l)

삭제된 사용자 내 박스 내역 화면이 표시됩니다.

4. 「○○씨의 마이 박스」(〇〇에는 유저명이 들어갑니다)에서 폴더나 파일을 확인합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAL7)

#### 내 상자에 저장된 파일 미리보기

1. 삭제된 사용자 내 상자 내역에서 미리 보려는 파일을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAJC)

파일 미리보기 화면이 표시됩니다.\
왼쪽 하단의 다운로드 버튼을 클릭하면 파일을 다운로드할 수 있습니다.\
파일 미리보기 화면을 닫으려면 오른쪽 상단의 ✕ 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aALR)

#### 내 상자에 저장된 파일의 인터넷 사용기록 확인

1. 삭제된 사용자 내 박스 기록에서 검색 기록 열에 표시되는 숫자(기록 수)를 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000a9i6)

인터넷 사용 기록 화면이 표시됩니다.

2. 브라우징 기록을 CSV 파일로 출력하려면 브라우징 기록 화면에서 CSV 내보내기 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000a9uS)

CSV 파일이 다운로드됩니다.

3. CSV 파일을 열고 문제없이 표시되는지 확인합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aACQ)

#### 내 상자에 있는 폴더 및 파일의 링크 내역 확인

1. 삭제된 사용자 내 상자 내역에서 링크 내역 열에 표시된 숫자(기록 수)를 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAMK)

링크 이력 화면이 표시됩니다.

2. 링크 수신 화면을 보려면 링크 확인 열에 표시되는 아이콘을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000a9Zz)

비활성화된 링크의 수신 화면이 표시됩니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAAe)

#### 휴지통에 저장된 폴더와 파일 확인

1. '삭제된 사용자 마이박스 내역'에서 '○○의 휴지통'(○○에는 사용자 이름이 들어갑니다)을 선택합니다.\
   사용자의 휴지통에 저장된 폴더와 파일이 표시됩니다.\
   사용자의 휴지통에서는 사용 가능한 기능에 따라 '내 상자/공유', '문서 관리', 'Connect', 'DLP', '웜 스토리지' 탭으로 나뉘어져 있지만 '삭제된 사용자 내 박스 내역'으로 이동 폴더와 파일은 탭으로 구분되지 않고 목록으로 표시됩니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aAMZ)

#### 휴지통에 저장된 파일 미리보기

1. '삭제된 사용자 내 박스 내역'에서 '○○ 휴지통'(○○에는 사용자 이름 포함)을 선택하고 미리 보기를 원하는 파일을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000a9xv)

파일 미리보기 화면이 표시됩니다.\
왼쪽 하단의 다운로드 버튼을 클릭하면 파일을 다운로드할 수 있습니다.\
파일 미리보기 화면을 닫으려면 오른쪽 상단의 ✕ 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q800000013DZ\&feoid=00N2w00000JMb2T\&refid=0EMBB000000aACR)
