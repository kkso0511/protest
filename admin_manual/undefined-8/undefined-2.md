# 파일 버전 관리를 사용하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 '용량 관리' > '스토리지 현황' 메뉴의 버전 관리 설정에서 파일 버전 관리를 활성화하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '스토리지' 항목을 사용할 수 있습니다.
* 초기 상태에서는, 버전 관리로 저장할 수 있는 파일 수가 '10개'로 설정되어 있습니다.
* 버전 관리로 저장할 수 있는 파일 수를 줄이더라도, 버전 데이터는 삭제되지 않습니다.\
  버전 데이터는, 아래 시점에 삭제됩니다.
  * 파일이 갱신되었을 때
  * 파일명이 변경되었을 때
* 이 매뉴얼의 절차로 버전 관리를 활성화하여 설정한 경우라도 [특정 폴더에서 버전 관리를 사용하는 방법](https://help.directcloud.net/admin_manual/undefined-4/undefined-19)의 절차에 따라, 지정한 폴더의 버전 관리를 비활성화할 수 있습니다.
* 버전 관리를 이용하려면, <마스터>, <전체권한>의 접근 권한이 부여되어 있어야 합니다.\
  자세한 내용은, [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/web)을 참조해 주시기 바랍니다.
* 버전 관리는 iOS 애플리케이션, Android 애플리케이션에서는 이용할 수 없습니다.

***

1. '용량 관리' > '스토리지 현황' 메뉴를 선택합니다.

<div align="left"><figure><img src="../../.gitbook/assets/image (2106).png" alt=""><figcaption></figcaption></figure></div>
