# 사용자에게 할당된 My Box의 용량을 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 My Box 관리에서 사용자에게 할당된 My Box의 용량을 변경하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '스토리지' 항목을 사용할 수 있습니다.
* My Box의 할당량은 계약 계획의 총 저장용량 한도 내에서 설정할 수 있습니다.
* My Box 사용량에는 버전 관리에서 보유한 파일의 이전 버전도 포함됩니다.
* 할당 용량의 1GB(기가바이트)는 1,024MB(메가바이트)로 변환됩니다.
* 할당량 변경값에 사용량 이하의 수치를 입력할 수 없습니다.
* [사용자 정보를 변경하는 방법](https://help.directcloud.net/admin_manual/undefined-3/undefined-1)의 단계에 따라 할당량을 변경할 수도 있습니다.
* '사용자 할당 용량'은 스토리지의 [계약 용량과 사용 용량을 확인하는 방법](https://help.directcloud.net/admin_manual/undefined-8/undefined)으로 스토리지 용량 계산이 이루어진 시점에 업데이트됩니다.
* [My Box 사용량이 할당량의 90%를 초과하는 경우 알림 메일을 보내는 방법](https://help.directcloud.net/admin_manual/undefined-10/my-box-90)으로, 계약 담당 관리자·일반 관리자·유저에게 통지 메일을 자동 송신할 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '용량 관리' > 'My Box 관리' 메뉴를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2146).png" alt=""><figcaption></figcaption></figure>



2. 'My Box 관리'에서 확인할 사용자의 할당 용량 열에 표시되는 편집 버튼을 클릭합니다.\
   할당 용량 변경 화면이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (2147).png" alt=""><figcaption></figcaption></figure>

&#x20;    &#x20;

3. '변경 할당 용량' 텍스트 상자에 사용자에게 할당할 용량(메가바이트)을 숫자로 입력하고 확인 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2149).png" alt=""><figcaption></figcaption></figure>



4. 관리자 페이지와 사용자 페이지에서 할당량 정보가 변경되었는지 확인합니다.
