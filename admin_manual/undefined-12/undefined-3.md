# 계약 기간 동안 플랜을 변경하는 방법과 차액을 지불하는 방법 (한국에 맞춘 업데이트 필요)

### 개요 <a href="#a03" id="a03"></a>

이 섹션에서는 계약 기간 동안 DirectCloud 플랜을 변경하는 방법과 상위 플랜으로 변경한 경우 요금의 차액을 지불하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 플랜 변경은 플랜 변경 신청 후 5영업일 이내에 적용됩니다.
* 계약 기간 동안 이용 플랜을 변경해도 계약 기간은 변경되지 않습니다.
* 상위 플랜으로 변경했을 경우는, 다음달 상순에 차액분이 청구되어 월말에 지불이 됩니다.
* 하위 플랜으로 변경하면 연간 결제로 인한 묶음이 발생하므로 차액 환불은 불가능합니다.
* 하위 플랜으로 변경할 수 있는 것은 계약 갱신월만입니다.

### 설명 <a href="#a05" id="a05"></a>

#### 플랜을 변경하는 방법

DirectCloud 계획을 변경하는 방법에는 두 가지가 있습니다.

* 「DirectCloud 이용 신청서」에 기재해 메일로 송신\
  이하의 링크를 클릭하면, 「DirectCloud 이용 신청서」를 다운로드할 수 있습니다. [https://directcloud.jp/assets/download/DirectCloud\_Application.xlsx](https://directcloud.jp/assets/download/DirectCloud_Application.xlsx)
* 웹 양식의 '도입 후 문의'를 입력하고 제출\
  아래 링크를 클릭하면 웹 양식에 액세스할 수 있습니다.\
  [https://directcloud.jp/support/qna\_after](https://directcloud.jp/support/qna_after)\
  배포 후 문의 양식은 [관리 페이지에서 제품에 대한 문의를 수행하는 방법](https://help.directcloud.jp/s/article/operation-of-administrator-function11013) 의 단계에서도 액세스할 수 있습니다.

#### 상위 플랜으로 변경한 경우의 지불에 대해서

계약 갱신월까지의 요금의 차액은, 플랜 변경의 다음 달의 청구가 됩니다. 지불은 다음 달 말입니다.\
자세한 내용은 DirectCloud 이용약관의 8조(사용료 청구 및 지불)를 확인하십시오.\
[https://directcloud.jp/terms](https://directcloud.jp/terms)
