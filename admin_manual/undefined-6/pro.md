# 파일을 다운로드하지 않고 미리 볼 수 있도록 설정하는 방법(제작중) pro 출시에 원본에 덮어씌우는 용

### 개요 <a href="#a03" id="a03"></a>

DirectCloud에는 파일을 다운로드하지 않는 미리보기 기능이 있습니다.\
이 섹션에서는 미리보기 설정에서 파일 미리 보기를 사용하도록 설정하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자의 기능 제한으로 "공유 설정"의 권한이 필요합니다.
* "업로더" 권한의 사용자는 파일 미리보기 기능을 사용할 수 없습니다.
* \[미리보기 설정]에서 \[사용함]을 선택하면 My Box 및 Shared Box 폴더에 저장된 파일 외에 링크 수신 화면에서도 전용 뷰어에서 미리 볼 수 있습니다.
* 파일을 지정하여 미리보기를 활성화하거나 비활성화할 수 없습니다.
* 폴더의 공유 링크를 작성·송신해도 로그 현황＞링크 이력에 폴더의 액세스 로그는 남지 않습니다.
* 폴더에 저장된 파일에 대한 액세스 로그는 유지됩니다.
* DirectCloud 드라이브에서 My Box, Shared Box 폴더에 저장된 동영상 및 오디오 파일을 미리 볼 수 없습니다. 그러나 다른 응용 프로그램을 사용하여 파일을 직접 열 수 있습니다.
*   파일 미리보기에 해당하는 확장자는 다음과 같습니다.
