# 폴더 또는 파일을 이동할 때의 확인 화면 표시 설정을 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

폴더나 파일을 드래그 앤 드롭으로 다른 폴더로 이동할 때 확인 화면이 표시되도록 설정하면 잘못된 위치로 이동하는 것을 방지할 수 있습니다.\
이 매뉴얼에서는, 폴더나 파일 이동 시의 확인 화면 표시 설정을 변경하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정'을 사용할 수 있습니다.
* 이 매뉴얼의 설정은 Web 브라우저의 조작에만 적용됩니다.
* 이 매뉴얼의 설정은, 드래그 앤 드롭으로 이동하는 경우에만 적용됩니다.\
  [파일을 이동하는 방법](https://help.directcloud.net/user_manual/undefined-3/web-drive-5) 의 '드래그 앤 드롭으로 파일이동' 절차로 폴더나 파일을 이동하더라도, 확인 화면은 표시되지 않습니다.
* 확인 화면 표시는, 기본적으로 '사용 안함'으로 설정되어 있습니다.
* 'DLP 관리' 메뉴는, DirectCloud의 유료 옵션인 'DirectCloud-SHIELD DLP'를 계약하고 있는 경우에만 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**드래그앤드롭 확인창 화면 표시 설정 변경**
{% endhint %}

1. '공유 설정' > '세부 기능 설정' 메뉴를 클릭합니다.

{% hint style="warning" %}
**폴더나 파일을 이동할 때 확인 화면 표시**
{% endhint %}

'세부 기능 설정' 메뉴의 '드래그앤드롭 확인창'에서 '항상사용' 또는 '사용'이 선택되어 있는 경우, 사용자가 폴더 또는 파일을 이동하면 "선택한 파일/폴더를 드롭한 폴더로 이동하시겠습니까?"라는 확인 화면이 표시됩니다.\
'확인' 버튼을 클릭하면, 폴더·파일이 이동합니다.
