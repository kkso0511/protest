# DirectCloud에서 7일 이상 사용하지 않는 기기를 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 '로그 현황' > '이슈브리핑' 메뉴에서 7일 이상 사용하지 않는 기기를 확인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '로그 현황' 항목에 접근 가능합니다.
* 사용자 ID나 사용자명으로 검색할 수는 없습니다.
* 미사용 기간을 기준으로 필터링하여 표시할 수는 없습니다.
* 표시 결과를 CSV 파일로 다운로드할 수는 없습니다.
* [저장된 로그인 정보로 DirectCloud를 사용하는 방법](https://help.directcloud.net/user_manual/undefined/web-drive-directcloud) 에서 '로그인 상태 유지'에 체크하여 로그인한 사용자는, 이후부터 '자동 로그인'이 되므로, '7일 이상 미사용 사용자', '7일 이상 미사용 기기'에는 기록되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '로그 현황' > '이슈브리핑' 메뉴를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1476).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 '장기 미사용 장치(7일이상)' 설정에서 기기 정보를 확인합니다.

<figure><img src="../../.gitbook/assets/image (1477).png" alt=""><figcaption></figcaption></figure>
