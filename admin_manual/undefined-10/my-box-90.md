# My Box 사용량이 할당량의 90%를 초과하는 경우 알림 메일을 보내는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자의 My Box 사용량이 할당량의 90%를 초과하는 경우 계약 담당자, 일반 관리자, 사용자에게 알림 메일을 보내도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '로그 현황' 항목에 접근 가능합니다.
* My Box 사용률이 90% 이상인 사용자는, 매일 아침 8시에 실행되는 배치 처리에 의해 순차적으로 계산되어, '문제 보고' 메뉴에 반영됩니다.
* 알림 메일의 수신 대상을 지정할 수는 없습니다.
* 사용자는, 알림 메일을 수신하지 않도록 설정할 수는 없습니다.
* 사용자의 My Box 사용 용량이 할당 용량을 초과하는 시점에 알림 메일이 전송되도록 설정할 수는 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '로그 현황' > '이슈브리핑' 메뉴를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1478).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 '사용률 90% 이상의 사용자' 설정에서 내 상자 사용량이 할당 용량의 90%를 초과하는 사용자를 확인합니다.

<figure><img src="../../.gitbook/assets/image (1487).png" alt=""><figcaption></figcaption></figure>

<table><thead><tr><th width="193">항목</th><th>설명</th></tr></thead><tbody><tr><td>이름</td><td>사용자 이름이 표시됩니다.</td></tr><tr><td>할당공간</td><td>My Box의 할당량이 표시됩니다.</td></tr><tr><td>현재 사용량</td><td>My Box 사용량이 표시됩니다.</td></tr><tr><td>파일 개수</td><td>My Box에 저장된 파일 수가 표시됩니다.</td></tr></tbody></table>



3. '90% 이상 사용자에게 메일로 통지합니다. (관리자에게는 대상자명단을 발송합니다.)'를 선택합니다.\
   설정은 즉시 저장됩니다.

<figure><img src="../../.gitbook/assets/image (1488).png" alt=""><figcaption></figcaption></figure>

&#x20;    &#x20;

4. 사용자의 My Box 사용량이 할당량의 90%를 초과하면 매일 아침 8시 이후에 알림 메일이 순차적으로 전송됩니다.\
   계약 담당자·일반 관리자에게는 '회사명 사용량 점검이 필요합니다.'라는 제목의 메일이, 유저에게는 '○○님씨사용량을 체크해보세요.'라는 제목의 이메일이 전송됩니다.<br>
