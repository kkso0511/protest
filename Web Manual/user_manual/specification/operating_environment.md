# DirectCloud의 각 애플리케이션의 동작 환경

### 개요 <a href="#a03" id="a03"></a>

사용 중인 PC 및 모바일 기기가 아래의 요구 사항을 충족해야 합니다.\
충족하지 않는 경우에는 애플리케이션이 정상적으로 동작하지 않을 수 있습니다.
