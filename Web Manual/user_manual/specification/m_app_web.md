# 모바일 애플리케이션과 모바일 Web 브라우저 버전의 차이

### 개요 <a href="#a03" id="a03"></a>

2024년 12월 10일 업데이트로 추가된 '모바일 Web 브라우저 애플리케이션'은 '모바일 애플리케이션'을 기반으로 개발되었습니다.\
이로 인해 기본적으로는 동일한 기능을 사용할 수 있으나, 일부 기능은 '모바일 Web 브라우저 애플리케이션'에 구현되어 있지 않습니다.\
또한 모바일 애플리케이션은 OS 및 라이브러리에 준거한 동작을 하는 반면, Web 브라우저의 구현 방식과는 차이가 있기 때문에 일부 사용자 인터페이스와 조작감이 다르게 느껴질 수 있습니다.\
본 매뉴얼에서는 '모바일 Web 브라우저 애플리케이션'과 '모바일 애플리케이션'의 차이에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* Web 브라우저에 따라서는 화면을 아래로 스크롤하면 주소 표시줄(주소 입력란)이 숨겨지는 기능이 있으나, '모바일 Web 브라우저 어플리케이션'은 해당 기능을 지원하지 않습니다.
*   모바일 Web 브라우저 어플리케이션은 아래 URL로 접속합니다.\
    **https://mobile.directcloud.net**

    모바일(iPhone, Android) Web 브라우저에서 'PC용 Web 브라우저'의 URL(https://web.directcloud.net)에 접속하면 '모바일 Web 브라우저 어플리케이션'으로 리다이렉트됩니다.
* 모바일 Web 브라우저 어플리케이션에서 지원하지 않는 기능이나 존재하지 않는 페이지(URL)에 접속할 경우, 오류 코드 '404'와 함께 '페이지를 찾을 수 없습니다.'라는 오류 메시지가 표시됩니다.
* 모바일 애플리케이션과는 달리, 모바일 Web 브라우저 어플리케이션에서는 커스텀 도메인을 사용하여 로그인할 수 있습니다.

***

### 설명

{% hint style="warning" %}
**모바일 Web 브라우저 어플리케이션에서 지원하지 않는 기능**
{% endhint %}

‘모바일 Web 브라우저 어플리케이션’은 ‘PC 버전 web 브라우저’가 아니라 ‘모바일 애플리케이션’의 기능을 기반으로 개발되었습니다.\
단, 아래의 ‘모바일 애플리케이션’ 기능은 ‘모바일 Web 브라우저 어플리케이션’에서 사용할 수 없습니다.

* **파일 내보내기 및 이미지 저장**\
  ‘모바일 애플리케이션’에서 제공되는 ‘내보내기’ 기능과 ‘이미지 저장’ 기능은 ‘모바일 Web 브라우저 어플리케이션’에서는 지원되지 않습니다.\
  파일을 외부로 전송하려면 공유 링크 기능을 사용해야 합니다.
* **외부 앱에서 파일 공유**\
  외부 앱의 파일 공유 기능을 사용하여 DirectCloud에 파일을 업로드할 수 없습니다.\
  DirectCloud에 파일을 업로드하려면 ‘모바일 Web 브라우저 어플리케이션’의 업로드 기능이나 업로드 요청기능을 사용해야 합니다.
* **캐시**\
  애플리케이션이 아니라 web 브라우저에서 캐시가 관리되기 때문에 캐시 기능을 사용할 수 없습니다.
* **오프라인 저장**\
  ‘모바일 Web 브라우저 어플리케이션’을 사용할 때는 항상 온라인 상태여야 하므로 오프라인 저장 기능은 이용할 수 없습니다.
* **화면 잠금**\
  ‘모바일 Web 브라우저 어플리케이션’은 web 브라우저를 사용하기 때문에 OS의 화면 잠금 기능을 사용할 수 없습니다.



{% hint style="warning" %}
**사용자 인터페이스와 조작성의 차이**
{% endhint %}

‘모바일 Web 브라우저 어플리케이션’은 web 브라우저의 라이브러리를 사용하기 때문에, 아래와 같이 사용자 인터페이스 및 조작성에서 ‘모바일 애플리케이션’과 차이가 있습니다.

* **스와이프하여 이전 페이지로 이동**\
  iOS의 ‘모바일 애플리케이션’에서는 스와이프 동작으로 이전에 표시된 화면으로 이동하지만, ‘모바일 Web 브라우저 어플리케이션’에서는 스와이프 동작으로 이전에 표시된 web 사이트로 이동합니다.\
  모바일 Web 브라우저 어플리케이션에서 이전 화면을 표시하려면 왼쪽 상단의 ‘←’ 버튼을 탭하십시오.
* **날짜 선택**\
  ‘모바일 애플리케이션’에서는 위아래로 스와이프하여 날짜를 선택하지만, ‘모바일 Web 브라우저 어플리케이션’에서는 달력에서 날짜를 선택합니다.
* **미리보기 화면**\
  동영상 등의 미디어를 재생할 때, ‘모바일 애플리케이션’에서는 개발 시 사용된 라이브러리의 플레이어로 재생되지만, ‘모바일 Web 브라우저 어플리케이션’에서는 web 브라우저 상에서 재생됩니다.
