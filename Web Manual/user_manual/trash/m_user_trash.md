# \[Mobile] 사용자 휴지통의 폴더 및 파일을 확인·조작하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션의 사용자 페이지에서 삭제되어 휴지통으로 이동한 폴더와 파일을 확인하고, 복원하거나 삭제하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 휴지통 기능을 사용하려면, 먼저 관리자 페이지 에서 사용자 휴지통을 활성화해야 합니다.
* 사용자 휴지통의 보존 기간은 모든 탭에 공통으로 설정되며, ‘1일’, ‘3일’, ‘5일’, ‘7일’, ‘30일’, ‘제한 없음’ 중 하나로 지정할 수 있습니다.
* iOS 애플리케이션과 Android 애플리케이션에서는 Connect 기능과 DLP 기능을 사용할 수 없습니다.\
  따라서 휴지통 화면에는 My Box 또는 Shared Box의 폴더에서 삭제된 폴더와 파일이 표시됩니다.
* 사용자의 휴지통에서 폴더와 파일을 검색할 수 없습니다. 다만 '이름', '일시', '종류', '크기'로 정렬할 수 있습니다.
* 링크가 설정된 폴더와 파일이 사용자의 휴지통으로 이동하면 링크 정보가 삭제됩니다. 일단 삭제된 링크 정보는 복원할 수 없습니다.
* Shared Box 또는 My Box에서 삭제되고 사용자의 휴지통으로 이동한 폴더 및 파일을 복원하면 폴더 선택 화면이 표시되지 않습니다. 자동으로 삭제되기 전에 저장된 폴더로 복원됩니다. 그러나 삭제되기 전에 저장된 폴더가 없으면 복원할 폴더를 선택하는 화면이 표시됩니다.
* 사용자의 휴지통에서 삭제된 폴더와 파일은 '회사 휴지통'으로 이동합니다.\
  자세한 내용은 [회사 휴지통에 있는 폴더와 파일을 복원하는 방법](https://help.directcloud.net/admin_manual/storage/company_trash_restore)의 휴지통에서 복원할 때 이동 위치를 참조하십시오.
* 사용자의 휴지통으로 이동한 폴더와 파일을 복원할 수 있는 사용자는 아래의 접근 권한이 부여되어 있어야 합니다.
  * 1단계 폴더 복원: 마스터
  * 2단계 이하 폴더 복원: 마스터, 전체권한
  * 파일 복원: 마스터, 전체권한, 편집자

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**사용자의 휴지통 보기**
{% endhint %}

1. 왼쪽 상단의 삼선 아이콘을 탭합니다.
2. 휴지통 버튼을 탭합니다.\
   휴지통 화면이 표시됩니다.<br>

{% hint style="warning" %}
**휴지통 폴더 또는 파일 복원**
{% endhint %}

1. 휴지통 화면을 표시하고 다음 방법 중 하나로 폴더와 파일을 복원합니다.\
   ❶ 복원할 폴더나 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭한 다음 표시된 메뉴에서 '복구' 버튼을 탭합니다.\
   ❷ 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)'을 탭하고 표시된 메뉴에서 '복수 선택'버튼을 탭 여러 폴더 또는 파일을 선택한 다음 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000hAFL)'버튼을 탭합니다.\
   폴더와 파일이 삭제되기 전에 저장된 My Box, Shared Box로 이동합니다.<br>

{% hint style="warning" %}
**휴지통 폴더 또는 파일 삭제**
{% endhint %}

1. 휴지통 화면을 표시하고 다음 방법 중 하나로 폴더와 파일을 삭제합니다.\
   ❶ 삭제할 폴더 또는 파일의 오른쪽 가장자리에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭한 다음 표시된 메뉴에서 '삭제' 버튼을 탭합니다.\
   ❷ 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)'을 탭하고 표시된 메뉴에서 '복수 선택'버튼을 탭 여러 폴더 또는 파일을 선택한 다음 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000hAIP)'버튼을 탭합니다.
2. 확인 화면에서 iOS 애플리케이션의 경우 '확인', Android 애플리케이션의 경우 '예'를 탭합니다.\
   폴더 또는 파일이 삭제되고 관리자의 휴지통으로 이동합니다.<br>

{% hint style="warning" %}
**휴지통 폴더와 파일 모두 삭제**
{% endhint %}

1. 휴지통 화면을 표시합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000Pb0\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)'을 탭한 다음 표시된 메뉴에서 '휴지통 비우기'버튼을 탭합니다.
3. 확인 화면에서 예 버튼을 클릭합니다.\
   폴더나 파일이 모두 삭제되고 관리자 휴지통으로 이동합니다.
