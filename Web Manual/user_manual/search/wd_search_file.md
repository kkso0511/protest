# \[Web, Drive] 폴더나 파일을 이름이나 문서내 텍스트 등으로 검색하는 방법

### 개요 <a href="#a03" id="a03"></a>

Web 브라우저의 검색에서는 검색 조건을 설정하여 My Box나 Shared Box에 저장된 폴더와 파일을 이름이나 태그로 검색할 수 있습니다. 또한 문서에 포함된 텍스트로 전체 검색을 수행할 수도 있습니다.\
DirectCloud 드라이브의 검색은 Windows 탐색기의 검색 방식에 따릅니다. 따라서 폴더와 파일을 이름이나 문서에 포함된 텍스트로 전체 검색할 수는 있지만, Web 브라우저에서 설정한 태그나 코멘트로 검색하는 것은 불가능합니다.\
이 매뉴얼에서는 Web 브라우저와 DirectCloud 드라이브에서 검색하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

**\[Web 브라우저]**

* Web 브라우저에서는, 이 매뉴얼의 절차에 따라 설정한 검색 조건이 DirectCloud에 로그인되어 있는 동안 유지됩니다. DirectCloud에서 로그아웃하면, 설정한 검색 조건은 초기화됩니다.
* 폴더나 파일을 업로드한 사용자 이름으로는 검색할 수 없습니다.
*   전체 검색에 대응하는 파일 확장자는 아래와 같습니다.

    | 분류     | 확장자                             |
    | ------ | ------------------------------- |
    | 문서     | doc, docx, odt, rtf, pages, pdf |
    | 스프레드시트 | xls, xlsx, ods, numbers         |
    | 프레젠테이션 | ppt, pptx, odp, keynote         |
    | 텍스트    | txt, csv, html, xml             |
    | 전자책    | epub                            |
* 다음 조건에 해당하는 파일은 전체 검색 대상에서 제외됩니다.
  * 파일 크기가 30MB를 초과하는 경우
  * 파일에 포함된 텍스트가 1MB를 초과하는 경우
*   검색 대상에서 ‘문서 내 텍스트’를 선택해 전체 검색을 수행하면, ‘AND 검색’을 선택했더라도 아래와 같은 결과가 표시될 수 있습니다.

    예시) ‘클라우드 스토리지 추천’으로 검색한 경우

    * 문서 내에 ‘클라우드 스토리지’와 ‘추천’이 모두 포함된 문서가 검색 결과에 표시됨
    * 문서 내에 ‘클라우드 스토리지’만 포함된 문서가 검색 결과에 표시됨
    * 문서 내에 ‘추천’만 포함된 문서가 검색 결과에 표시됨
    * 문서 내에 ‘클라우드’, ‘스토리지’, ‘추천’이 각각 포함된 문서가 검색 결과에 표시됨
* 검색 결과는 기본적으로 ‘날짜’의 내림차순으로 표시됩니다.\
  단, ‘본문’ 탭의 검색 결과(문서 내 텍스트 검색 결과)는 ‘이름’이 아니라 관련도가 높은 순으로 표시됩니다.
* ‘파일명’, ‘코멘트’, ‘태그’, ‘문서 속성’ 탭의 검색 결과는 헤더 항목의 ‘이름’, ‘크기’, ‘갱신 일시’, ‘종류’를 클릭해 정렬할 수 있습니다.
* ‘파일명’, ‘코멘트’, ‘태그’, ‘문서 속성’ 탭 이외의 검색 결과는 정렬할 수 없습니다.
* 검색 결과에 표시된 폴더 및 파일을 조작할 때에는 다음과 같은 제한이 있습니다.
  * 폴더 및 파일에 대해 수행할 수 있는 작업은 각 메뉴(예: ‘Shared Box’, ‘Warm Storage’ 등)에 따라 다릅니다.
  * 폴더나 여러 개의 파일을 한 번에 선택하여 다운로드하는 것은 불가능합니다.

**\[DirectCloud 드라이브]**

* DirectCloud Drive에서는 계약 플랜이나 계약 옵션에 관계없이 ‘검색 범위’와 ‘검색 대상’을 지정하여 검색할 수 없습니다.
* DirectCloud Drive의 검색 기능은 Windows 탐색기의 검색 방식에 따릅니다.\
  필터 설정이나 기타 옵션에 대한 자세한 내용은 Windows 도움말 또는 Microsoft의 공식 문서를 참고하세요.

3. 검색 대상별로 검색 결과를 전환하고 표시하려면 탭을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (972).png" alt=""><figcaption></figcaption></figure>



4. 검색 설정을 수정하여 검색하려면 '검색' 필드에서 ▼ 버튼을 다시 클릭합니다.

<figure><img src="../../.gitbook/assets/image (973).png" alt=""><figcaption></figcaption></figure>

검색 설정 화면에 2단계에서 설정한 내용이 표시됩니다.



5. 2단계를 참조하여 검색 설정을 수정하고 검색 키워드를 다시 입력하고 Enter 키를 누르거나 검색 버튼을 클릭합니다.



**\[검색 조건 설정 저장]**

* 검색조건을 저장하고 싶은 경우\
  검색결과 상단에 표시되어 있는 '해당 검색어를 Quick Search에 추가하시겠습니까?'의 '추가' 버튼을 클릭하면 검색란 오른쪽에 저장된 검색조건이 추가됩니다.

<figure><img src="../../.gitbook/assets/image (974).png" alt=""><figcaption></figcaption></figure>

표시된 검색 기준을 클릭하기만 하면 검색 결과가 표시됩니다.\
이 설정은 로그아웃해도 삭제되지 않습니다.\
삭제하려면 추가된 검색 조건의 오른쪽 가장자리에 표시되는 ✕ 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (975).png" alt=""><figcaption></figcaption></figure>



* 검색 기준 설정 내용을 유지하려는 경우 설정한 검색\
  조건은 DirectCloud에서 로그아웃할 때 재설정됩니다. 이 설정을 로그아웃해도 유지되도록 하려면 검색 조건 설정 창에서 오른쪽 하단에 표시되는 '저장' 토글 버튼을 켭니다.

<figure><img src="../../.gitbook/assets/image (976).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**DirectCloud 드라이브에서 검색**
{% endhint %}

1. Windows 탐색기에서 DirectCloud 드라이브의 My Box또는 Shared Box 아래 폴더를 선택합니다.

<figure><img src="../../.gitbook/assets/image (977).png" alt=""><figcaption></figcaption></figure>



2. 필요한 경우 Windows 탐색기에서 '검색'을 설정하고 검색 필드에 검색 키워드를 입력한 다음 Enter 키를 클릭합니다. 선택한 폴더 아래에 폴더와 파일이 검색되고 결과가 표시됩니다.

<figure><img src="../../.gitbook/assets/image (978).png" alt=""><figcaption></figcaption></figure>
