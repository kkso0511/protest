# 사용자 페이지의 DLP 폴더에서 파일의 PDF 다운로드를 활성화하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지의 폴더 속성에서 DLP 폴더에 저장된 파일을 PDF로 변환해 다운로드할 수 있도록 설정하는 방법을 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 폴더 속성을 설정하는 사용자에게는 '마스터' 접근 권한이 부여되어 있어야 합니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/w_permission)을 참조해주시기 바랍니다.
* ['DLP 설정'에서 사용자에게 파일을 PDF로 다운로드할 수 있는 기능을 허용하는 방법](https://help.directcloud.net/admin_manual/directcloud-shield-dlp/dlp-pdf)의 절차에서 PDF 다운로드 설정이 사용함으로 설정되어 있어야 합니다.
* PDF 다운로드 설정은 기본으로 사용함으로 설정되어 있습니다.
* PDF 다운로드 설정은 1계층 폴더에 대해 설정하며, 설정된 내용은 하위 폴더에 상속됩니다.
* DLP 폴더의 폴더 속성에는 본 매뉴얼의 설정 항목 외에 아래 항목들이 있습니다.
  * 다운로드
  * 링크
  * 첨부 파일 전송
* DLP 폴더 및 DLP 관리 메뉴는 DirectCloud의 옵션인 DirectCloud-SHIELD DLP를 계약한 경우에만 표시됩니다.
* DLP 폴더의 폴더 속성에서는 워크플로우, SHIELD, 링크를 설정할 수 없습니다.
* DLP 폴더에서 Connect User 초대를 사용할 수 없습니다.

### 절차

1. DLP 메뉴를 클릭한 다음, PDF 다운로드를 활성화하려는 1계층 폴더를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2359).png" alt=""><figcaption></figcaption></figure>

2. 슬라이드 화면의 폴더 속성 설정에서 PDF 다운로드가 사용함으로 표시되어 있는지 확인합니다.\
   사용하지 않음으로 표시되어 있는 경우, 오른쪽에 표시된 설정 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2360).png" alt=""><figcaption></figcaption></figure>

3. 사용함을 선택한 뒤 확인 버튼을 클릭합니다.
