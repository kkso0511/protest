# Shared Box의 폴더 파일이 자동으로 삭제되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지에서 Shared Box 폴더의 폴더와 파일을 자동으로 삭제하도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 폴더 속성을 설정하는 사용자에게는 <마스터> 접근 권한이 부여되어 있어야 합니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/w_permission)을 참조해주시기 바랍니다.
* 자동 삭제는 1단계 폴더 또는 2단계 폴더에 설정할 수 있으며, 설정한 내용은 하위 폴더에 상속됩니다.
* 자동 삭제 폴더 속성은 Connect 폴더와 DLP 폴더에도 설정할 수 있습니다.\
  사양 및 제약 사항은 Shared Box와 동일합니다.
* [\[Web, Drive\] 사용자 페이지에서 파일을 잠그거나 잠금 해제하는 방법](https://help.directcloud.net/user_manual/file_management/wd_lock_file)으로 잠금된 파일도 삭제됩니다.
* 폴더와 파일은 지정한 날짜와 요일의 1시 30분에 자동 삭제됩니다.
* 자동 삭제가 실행되는 시간은 변경하거나 지정할 수 없습니다.
* 저장 기간 기준으로 실행을 선택한 경우 자동 삭제 대상은 파일만 적용됩니다.
* 저장 기간 기준으로 실행에 체크한 경우, 파일 저장일의 다음 날부터 계산하여 계산일을 포함해 지정한 기간이 경과한 뒤 자동 삭제가 실행됩니다.
* 저장 기간 기준으로 실행에 체크한 경우, 설정 이전에 저장된 파일의 자동 삭제는 아래와 같이 실행됩니다.
  * 저장일로부터의 경과 일수가 설정한 경과 일수를 초과한 경우:\
    설정한 당일은 일수로 계산하지 않으며, 다음 날 1시 30분에 실행됩니다.
  * 저장일로부터의 경과 일수가 설정한 경과 일수를 초과하지 않은 경우:\
    설정한 당일은 일수로 계산하지 않으며, 남은 일수가 경과한 뒤, 다음 날 1시 30분에 실행됩니다.
  *   예: 설정 일시가 2022년 1월 11일, 저장일이 2022년 1월 14일, 자동 삭제 설정이 3일 후인 경우

      | 기산일                                | 자동 삭제일                                               |
      | ---------------------------------- | ---------------------------------------------------- |
      | <p>2022년 1월 15일<br>(저장일의 다음 날)</p> | <p>2022년 1월 18일 1시 30분<br>(기산일을 포함하여 3일 후의 다음 날)</p> |
*   저장 기간 기준으로 실행에서 “○개월 후”란, ○개월 후의 같은 날짜를 의미합니다.\
    예: 설정 일시가 2022년 1월 11일, 저장일이 2022년 1월 14일이고 자동 삭제 설정이 1개월 후인 경우

    | 기산일                                | 자동 삭제일                                            |
    | ---------------------------------- | ------------------------------------------------- |
    | <p>2022년 1월 15일<br>(저장일의 다음 날)</p> | <p>2022년 2월 16일 1시 30분<br>(1개월 후 동일 날짜의 다음 날)</p> |
*   저장 기간 기준으로 실행에서 “○개월 후”에 기산일과 같은 날짜가 존재하지 않는 경우, 다음 달 1일에 삭제됩니다.\
    예: 설정 일시가 2022년 1월 11일, 저장일이 2022년 1월 30일이고 자동 삭제 설정이 1개월 후인 경우

    | 기산일                                | 자동 삭제일                                        |
    | ---------------------------------- | --------------------------------------------- |
    | <p>2022년 1월 31일<br>(저장일의 다음 날)</p> | <p>2022년 3월 1일 1시 30분<br>(1개월 후 말일의 다음 날)</p> |
* My Box의 파일은 폴더 속성 설정에서 자동 삭제를 사용할 수 없습니다.\
  My Box 안의 파일을 자동 삭제되도록 설정하려면 [My Box의 파일을 자동으로 삭제하도록 설정하는 방법](https://help.directcloud.net/admin_manual/storage/mybox_auto_delete)에서 설정해주시기 바랍니다.
* 삭제 후 확인 메일 전송에서 “사용자(Guest 제외)”에 체크하면, 설정한 폴더에 접근 권한을 가진 모든 사용자에게 알림 메일이 전송됩니다.
* 아래의 경우에는 사용자에게 자동 삭제 알림이 전달되지 않습니다.
  * [사용자가 사용자 페이지에서 알림 메일 수신 여부를 결정할 수 있도록 설정하는 방법](https://help.directcloud.net/admin_manual/detail_settings/user_email_setting)에서 “알림 메일”이 “사용하지 않음”으로 설정되어 있는 경우
  * [파일에 유효 기간을 설정하는 방법](https://help.directcloud.net/user_manual/file_management/expiration_date)에서 “메일을 받지 않음”에 체크되어 있는 경우
* 폴더 속성 설정으로 자동 삭제된 폴더·파일은 Storage > 관리자 휴지통 메뉴로 이동합니다.\
  관리자 휴지통의 보관 기간 내라면 폴더·파일을 복원할 수 있습니다.
* DirectCloud의 유료 옵션인 Warm Storage를 계약하고 있는 경우, 사용자 페이지에 Warm Storage 메뉴가 표시됩니다.
* 관리자가 DirectCloud AI를 활성화하면 폴더 속성에 DirectCloud AI가 표시됩니다.
* DirectCloud-CONNECT를 사용할 수 있는 경우, 사용자 페이지에 Connect 메뉴가 표시됩니다.
* Warm Storage의 파일은 자동 삭제가 불가능하므로 Warm Storage 폴더의 폴더 속성에도 자동 삭제가 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**자동 삭제 설정 보기**
{% endhint %}

1. 공유, 연결, DLP 중 하나를 클릭한 다음 자동 삭제를 설정하려는 첫 번째 계층 구조의 폴더를 클릭합니다.
2. 슬라이드 화면의 폴더 속성 설정에서 용량 제한에 사용됨이 표시되는지 확인합니다.\
   '사용 안 함'이 표시되면 오른쪽에 표시된 설정 버튼을 클릭합니다.\
   자동 삭제 설정이 표시됩니다.



{% hint style="warning" %}
**날짜를 지정하여 폴더 파일 자동 삭제**
{% endhint %}

1. 자동 삭제에서 한 번만을 선택하고 날짜 표시 필드를 클릭합니다.
2. 표시된 캘린더에서 자동 삭제할 날짜를 클릭합니다.\
   자동 삭제할 날짜가 입력됩니다.
3. 자동 삭제 알림 메일을 보낼 대상으로 '하위 관리자', '사용자(게스트 제외)'에 하나 이상 체크를 넣고 '확인' 버튼을 클릭합니다.



{% hint style="warning" %}
**요일을 지정하여 매주 폴더 파일 자동 삭제**
{% endhint %}

1. 자동 삭제에서 매주를 선택하고 자동 삭제할 요일을 선택합니다.
2. 자동 삭제 알림 메일을 보낼 대상으로 '하위 관리자', '사용자(게스트 제외)'에 하나 이상 체크를 넣고 '확인' 버튼을 클릭합니다.



{% hint style="warning" %}
**날짜를 지정하여 월별 폴더 파일 자동 삭제**
{% endhint %}

1. 자동 삭제에서 매월을 선택하고 자동 삭제할 날짜를 선택합니다.
2. 자동 삭제 알림 메일을 보낼 대상으로 '하위 관리자', '사용자(게스트 제외)'에 하나 이상 체크를 넣고 '확인' 버튼을 클릭합니다.



{% hint style="warning" %}
**지정한 기간이 경과한 후 파일을 자동 삭제**
{% endhint %}

1. 자동 삭제에서 보존 기간을 기준으로 실행을 선택하고 어느 기간이 경과한 후 자동 삭제할지를 선택합니다.
2. 자동 삭제 알림 메일을 보낼 대상으로 '하위 관리자', '사용자(게스트 제외)'에 하나 이상의 체크를 넣고 '저장' 버튼을 클릭합니다.



{% hint style="warning" %}
**자동 삭제 완료 이메일 확인**
{% endhint %}

파일이 삭제되면 "삭제 후 확인 메일 보내기"에서 체크를 한 사용자에게 "자동 삭제 알림"메일이 전송됩니다.
