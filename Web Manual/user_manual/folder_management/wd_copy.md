# \[Web, Drive] 폴더를 복사하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Web 브라우저나 DirectCloud 드라이브를 사용하여 DirectCloud에 저장된 파일을 DirectCloud의 다른 폴더에 복사하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

*   폴더 복사 작업은 내부적으로 복사 원본 폴더가 먼저 복제된 뒤, 복제된 폴더가 복사 대상 폴더로 업로드되는 방식으로 처리됩니다.\
    따라서 폴더를 복사하는 사용자에게는 복사 원본 폴더와 복사 대상 폴더 모두에 아래의 접근 권한이 부여되어 있어야 합니다.

    * **복사 원본 폴더:**\
      마스터, 전체권한, 편집자
    * **복사 대상 폴더:**\
      마스터, 전체권한, 편집자, 편집자－, 프리뷰어＋, 업로더

    자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/w_permission), [접근 권한의 종류와 DirectCloud 드라이브 에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/d_permission)을 참조하십시오.
* 복사 화면에서 복사 대상 폴더에 새 폴더를 생성하려면 '마스터', '전체권한'의 접근 권한이 부여되어 있어야 합니다.
* 다음 조건을 모두 만족하는 경우, DirectCloud 드라이브에서는 폴더를 복사할 수 없습니다.
  1. [워크플로우를 활성화하는 방법](https://help.directcloud.net/user_manual/workflow/create_workflow)에서 워크플로우가 활성화되어 있음
  2. [특정 폴더에서 워크플로우를 활성화하는 방법](https://help.directcloud.net/admin_manual/folder/enable_workflow)에서 복사하려는 폴더가 포함된 폴더 또는 복사 대상 폴더의 워크플로우가 사용하기로 설정되어 있음
  3. 다운로드 워크플로우 또는 업로드 워크플로우의 신청자로 설정되어 있음
* Warm Storage 및 Cold Storage에는 폴더와 파일을 복사하는 기능이 없습니다.\
  따라서 아래 방향으로 폴더 및 파일을 복사할 수 없습니다.
  * Hot Storage → Warm Storage\
    Hot Storage → Cold Storage
  * Warm Storage → Hot Storage\
    Warm Storage → Cold Storage\
    Warm Storage → Warm Storage
  * Cold Storage → Cold Storage
* 여러 개의 폴더·파일에 체크되어 있는 경우에는 위에서부터 순서대로 복사가 실행됩니다.
* 복사 대상 폴더에 동일한 이름의 폴더가 있는 경우에는 복사할 수 없습니다.
* 이미 생성된 링크가 있는 파일을 포함한 폴더를 복사하는 경우, 파일의 링크 정보는 복사되지 않지만, 폴더를 이동하는 경우에는 파일의 링크 정보가 유지됩니다.
* 부하로 인해 처리 속도가 저하될 수 있으므로, DirectCloud에서는 한 번에 1,000개 이상의 폴더 및 파일을 복사하는 것을 권장하지 않습니다.
*   여러 폴더와 파일을 선택하여 복사할 때, 한 번에 복사할 수 있는 하위 폴더 수는 총 10,000개입니다.

    10,001개 이상의 하위 폴더를 가진 부모 폴더를 복사하려고 하면,\
    “복사/이동 제한을 초과하여 더 이상 진행할 수 없습니다. (하위폴더 10,000개 초과).”\
    라는 오류 메시지가 표시됩니다. 이 경우 복사 작업은 중단됩니다.
*   복사 처리 중 상태 화면의 × 버튼을 누르거나, 세션 시간이 10분을 초과한 경우에는,\
    “복사/이동 중 타임아웃이 발생하여 더 이상 진행할 수 없습니다.”\
    라는 오류 메시지가 표시됩니다.

    이때 오류 메시지가 표시되기 전까지 처리된 폴더 및 폴더 내 저장된 폴더·파일은 복사되지만, 처리 대기 상태였던 폴더·파일은 복사되지 않습니다.
*   DirectCloud AI가 활성화된 폴더에서의 복사 가능 범위는 다음과 같습니다.

    * DirectCloud AI가 활성화된 폴더로 복사할 수 있는 폴더:\
      My Box, Shared Box, 문서 관리 폴더
    * DirectCloud AI가 활성화된 폴더에서 복사할 수 있는 폴더:\
      My Box, Shared Box, 문서 관리 폴더, Connect 폴더, DLP 폴더



***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 폴더 복사**
{% endhint %}

1. 복사할 폴더가 저장된 폴더를 클릭합니다.
2. 다음 방법 중 하나로 폴더의 복사 화면을 표시합니다.\
   \
   ❶ 복사할 폴더에 체크를 하고 상단의 '복사' 버튼을 클릭합니다. 여러 폴더를 선택할 수 있습니다.\
   ❷ 복사할 폴더를 체크하고 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 복사를 클릭합니다. 여러 폴더를 선택할 수 있습니다.\
   ❸ 복사할 폴더의 오른쪽에 있는 ▼ 버튼을 클릭하고 표시된 메뉴에서 복사를 클릭합니다. 여러 폴더를 선택할 수 없습니다.\
   \
   복사 화면이 표시됩니다.
3. 복사 화면에서 대상 폴더를 클릭합니다.
4. 필요한 경우 새 폴더를 클릭하여 새 폴더를 만듭니다.
5. 복사 버튼을 클릭합니다.\
   복사가 시작되고 상태 화면이 표시됩니다.\
   복사가 완료되면 화면 하단에 "복사가 완료되었습니다."라는 메시지가 표시됩니다.\
   메시지에 표시되는 취소 버튼을 클릭하여 복사를 취소할 수 있습니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 폴더 복사**
{% endhint %}

1. DirectCloud 드라이브에서 복사할 폴더가 있는 폴더를 클릭합니다.
2. 복사할 폴더를 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 복사를 클릭합니다.\
   여러 파일을 선택하고 복사할 수도 있습니다.
3. DirectCloud 드라이브에서 대상 폴더를 클릭하여 폴더를 엽니다.
4. 대상 폴더에서 폴더를 선택하지 않은 상태에서 마우스 오른쪽 버튼을 클릭하고 표시된 메뉴에서 붙여넣기를 클릭합니다.\
   폴더가 복사됩니다.
