# 모바일 애플리케이션에서 파일을 이메일로 보내는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션의 폴더에 저장된 파일을 이메일로 전송하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 여러 개의 파일을 선택하여 동시에 이메일로 전송할 수는 없습니다.
* 이 매뉴얼의 절차에 따라 파일을 이메일로 전송하려면, 관리자 페이지의 보안 > 모바일 메뉴에서 ‘이메일로 전송’이 ‘사용’으로 설정되어 있어야 합니다.\
  자세한 내용은 모바일 애플리케이션에서 이메일 전송을 허용할지 여부를 설정하는 방법을 참고하시기 바랍니다.
* 관리자 페이지에서 다운로드 워크플로우가 활성화되어 있는 경우, 모바일 애플리케이션에서 ⁝ 버튼을 탭해도 다운로드 메뉴가 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 이메일을 보내는 방법**
{% endhint %}

1. 이메일로 전송할 파일이 저장된 폴더로 이동합니다.
2. 이메일로 전송할 파일의 오른쪽 끝에 있는 ⁝ 아이콘을 탭합니다.
3. 표시된 메뉴에서 다운로드를 탭합니다.
4. 이메일로 전송 메뉴를 탭합니다.\
   선택한 파일이 첨부된 상태로 이메일 전송 화면이 표시됩니다.
5. 받는사람과 내용을 입력한 후 이메일을 전송합니다.



{% hint style="warning" %}
**Android 애플리케이션에서 이메일을 보내는 방법**
{% endhint %}

1. 이메일로 전송할 파일이 저장된 폴더로 이동합니다.
2. 이메일로 전송할 파일의 오른쪽 끝에 있는 ⁝ 아이콘을 탭합니다.
3. 표시된 메뉴에서 다운로드를 탭합니다.
4. 이메일로 전송 메뉴를 탭합니다.\
   앱 선택 화면이 표시됩니다.
5. 파일 전송에 사용할 이메일 앱을 탭합니다.\
   선택한 파일이 첨부된 상태로 이메일 전송 화면이 표시됩니다.
6. 받는사람과 내용을 입력한 후 이메일을 전송합니다.
