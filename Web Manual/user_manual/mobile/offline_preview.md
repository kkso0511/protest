# 네트워크가 끊겨도 파일을 미리보기하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션에서 네트워크 연결이 끊어진 상태에서도 파일을 미리보기하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 사전에 [오프라인 상태에서도 모바일 애플리케이션에서 PDF 파일을 열람할 수 있도록 설정하는 방법](https://help.directcloud.net/admin_manual/security_policy/m_offline_use_pdf)의 절차에 따라, ‘오프라인 시 접근’을 ‘사용’으로 선택해야 합니다.
* <마스터>, <전체권한>, <편집자>, <다운로더> 접근 권한이 부여된 사용자는 오프라인 열람을 이용할 수 있습니다.
*   오프라인 열람이 가능한 파일의 확장자는 '.pdf'만 해당됩니다.

    ‘오프라인에서 이용 가능’ 버튼을 탭하면, ‘오프라인’ 화면에 PDF 파일이 다운로드됩니다.\
    이로 인해 ‘로그 현황' > '파일 이용 내역’ 메뉴의 ‘동작’ 항목에는 ‘파일 다운로드’로 표시됩니다.
* PDF 파일을 한 번 다운로드한 후 파일을 표시하는 방식이므로, ‘오프라인’ 화면에서 PDF 파일을 삭제하더라도 원본 PDF 파일은 삭제되지 않습니다.
* ‘오프라인’ 화면에 PDF 파일이 저장된 상태에서 모바일 애플리케이션에서 로그아웃을 시도하면, ‘오프라인’ 화면에 저장된 모든 PDF 파일이 삭제된다는 안내 메시지가 표시됩니다.\
  ‘확인’ 버튼을 탭하면 PDF 파일이 모두 삭제됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**네트워크 연결이 끊긴 상태에서 파일 미리보기**
{% endhint %}

**\[PDF 파일을 오프라인으로 볼 수 있도록 허용]**

1. 오프라인으로 볼 수 있게 하려는 PDF 파일이 저장된 폴더를 탭합니다.
2. 오프라인 탐색하려는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000NWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 '오프라인으로 사용 가능'을 탭합니다.\
   선택한 파일을 오프라인으로 볼 수 있습니다.



**\[PDF 파일 오프라인 탐색]**

사용하는 모바일 네트워크 환경을 분리합니다.

1. 왼쪽 상단의 삼선 아이콘을 탭한 다음 오프라인 메뉴를 탭합니다.\
   오프라인 화면이 표시됩니다.
2. 오프라인 탐색하려는 PDF 파일을 탭합니다.\
   미리보기 화면이 표시됩니다.



{% hint style="warning" %}
**오프라인 화면에서 PDF 파일 삭제**
{% endhint %}

1. 오프라인 화면에서 삭제할 PDF 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000NWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)' 버튼을 탭합니다.
2. 표시되는 메뉴에서 삭제 버튼을 탭합니다.\
   "모바일 기기에 저장된 파일을 삭제하시겠습니까?"라는 화면이 나타납니다.
3. iOS 애플리케이션의 경우 '확인', Android 애플리케이션의 경우 '예'를 탭합니다.\
   파일이 삭제됩니다.
4. 오프라인 화면에 더 이상 표시되지 않는지 확인합니다.
