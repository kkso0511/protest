# 얼굴 인식으로 애플리케이션 잠금을 해제하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션에서 얼굴 인식을 통해 모바일 애플리케이션의 잠금을 해제하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 사전에 기기에서 Face ID를 설정해 두셔야 합니다.
*   처음 로그인할 때는 회사 ID, 사용자 ID, 비밀번호로 인증한 후 얼굴 인식 시스템이 작동합니다.

    다중 요소 인증을 통해 로그인할 수 있으므로, 로그인 시 보안성을 더욱 높일 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션 잠금 해제**
{% endhint %}

1. 왼쪽 상단에 있는 세 줄 아이콘을 탭한 후, 설정 메뉴를 탭합니다.\
   설정 화면이 표시됩니다.
2. 화면 잠금을 탭합니다.\
   화면 잠금 화면이 표시됩니다.
3. Face ID 및 패스코드 토글 스위치를 ON으로 설정합니다.\
   설정은 즉시 저장됩니다.
4. 다시 애플리케이션에 로그인할 때 Face ID가 실행되는지 확인합니다.



{% hint style="warning" %}
**Android 애플리케이션 잠금 해제**
{% endhint %}

1. 왼쪽 상단에 있는 세 줄 아이콘을 탭한 후, 설정 메뉴를 탭합니다.\
   설정 화면이 표시됩니다.
2. 화면 잠금을 탭합니다.\
   화면 잠금 화면이 표시됩니다.
3. 생체 인증 및 보안에 체크합니다.\
   설정은 즉시 저장됩니다.
4. 다시 애플리케이션에 로그인할 때 Face ID가 실행되는지 확인합니다.
