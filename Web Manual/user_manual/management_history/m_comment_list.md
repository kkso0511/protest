# \[Mobile] 사용자가 등록한 코멘트를 목록으로 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션에서 사용자가 등록한 파일 코멘트를 목록으로 확인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일 코멘트 기능을 이용하려면, 먼저 [파일 코멘트 기능을 활성화하는 방법](https://help.directcloud.net/admin_manual/detail_settings/enable_comment), [특정 폴더에서 코멘트를 사용하도록 설정하는 방법](https://help.directcloud.net/admin_manual/folder/enable_comment)에 따라 파일 코멘트가 활성화되어 있어야 합니다.
* 파일이 삭제되면 해당 파일과 연결된 코멘트도 함께 삭제되며, 파일 삭제로 인해 코멘트도 삭제되었습니다. 라고 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**파일 코멘트 목록 표시**
{% endhint %}

1. 오른쪽 위에 있는 말풍선 아이콘을 탭합니다.\
   코멘트 알림 화면이 표시됩니다.\
   오른쪽 위에 있는 체크 아이콘을 탭하면 모든 코멘트를 읽음 상태로 변경할 수도 있습니다.



{% hint style="warning" %}
**파일 코멘트 세부정보 확인**
{% endhint %}

1. 코멘트 알림 화면에서 상세 내용을 확인하려는 코멘트를 탭합니다.\
   사용자 이름, 사용자 ID, 코멘트 내용, 코멘트가 등록된 날짜와 시간이 표시됩니다.\
   파일을 탭하면 미리보기 화면을 표시할 수도 있습니다.



{% hint style="warning" %}
**파일 코멘트 삭제**
{% endhint %}

1. 코멘트 알림 화면에서 삭제하려는 코멘트를 왼쪽으로 스와이프합니다.
2. 삭제를 탭합니다.\
   확인 화면이 표시됩니다.
3. iOS 애플리케이션에서는 확인을, Android 애플리케이션에서는 예를 탭합니다.\
   파일 코멘트가 삭제됩니다.
