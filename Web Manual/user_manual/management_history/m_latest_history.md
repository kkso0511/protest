# \[Mobile] Home 메뉴에서 최근의 조작 이력을 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션의 Home 메뉴에서 최근 작업 기록을 확인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 폴더에 대한 접근 권한이 삭제된 경우, Home 메뉴에 나열되어 있던 아래 항목의 파일과 폴더 경로 정보는 사라집니다.\
  코멘트\
  즐겨찾기\
  최근 사용한 파일\
  공유된 파일\
  링크로 업로드된 파일
* 폴더의 접근 권한이 업로더로 변경된 경우, 해당 폴더에 저장된 폴더 또는 파일은 Home 메뉴 항목에 표시되지만 파일을 미리보기로 확인할 수는 없습니다.
* 관리 페이지의 Guest 기능 제한에서 Home이 비활성화로 설정된 경우, Guest에게는 Home 메뉴가 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. iOS 앱 또는 Android 앱에 로그인합니다.
2. Home에 다음 항목이 표시됩니다.

<table><thead><tr><th width="210">항목</th><th>설명</th></tr></thead><tbody><tr><td>코멘트</td><td>추가된 파일 코멘트가 표시됩니다. 최대 20건 표시됩니다.</td></tr><tr><td>최근파일</td><td>최근에 사용한 파일이 표시됩니다. 최대 20건 표시됩니다.</td></tr><tr><td>즐겨찾기</td><td>즐겨찾기에 등록한 파일이 표시됩니다. 최대 20건 표시됩니다.</td></tr><tr><td>공유받 파일</td><td>다른 사용자가 업로드한 파일이 표시됩니다. 최대 20건까지 표시됩니다. 업로더 접근 권한이 부여된 사용자에게는 표시되지 않습니다.</td></tr><tr><td>Link History</td><td>사용자가 직접 발행한 공유 링크의 이력이 표시됩니다. 최대 20건까지 표시됩니다.</td></tr><tr><td>링크로 받은 파일</td><td>공유 링크에 업로드 기능이 허용된 경우, 링크 수신 화면에서 업로드된 파일이 표시됩니다. 링크 수신 화면에 파일이 업로드될 때마다 썸네일이 하나씩 생성되며, 썸네일은 최대 20건까지 표시됩니다. 한 번에 여러 파일이 업로드된 경우, 썸네일 왼쪽 하단에 ‘+1’과 같이 업로드된 나머지 파일 개수가 표시됩니다. 숫자를 클릭하면 업로드된 파일 이름을 확인할 수 있습니다.</td></tr></tbody></table>
