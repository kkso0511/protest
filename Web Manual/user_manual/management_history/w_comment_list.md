# \[Web] 사용자가 등록한 코멘트를 목록으로 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자가 등록한 파일의 코멘트를 목록으로 확인하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일 코멘트 기능을 사용하려면, 관리자 페이지에서 파일 코멘트 기능을 활성화 시켜야 합니다.
* 파일이 삭제되면 해당 파일에 연결된 코멘트도 함께 삭제되며, “파일이 삭제되어 코멘트를 확인할 수 없습니다.”라는 메시지가 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. 사용자 페이지에 로그인하고 홈 화면의 오른쪽 상단에 있는 코멘트 아이콘을 클릭합니다.\
   코멘트가 목록에 표시됩니다.

<figure><img src="../../.gitbook/assets/image (2492).png" alt=""><figcaption></figcaption></figure>



2. 코멘트에 커서를 올리면 아래와 같은 작업을 수행할 수 있습니다.\
   각 버튼의 자세한 내용은 아래 표를 참고하세요.
