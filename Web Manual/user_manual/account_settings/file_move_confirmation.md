# 폴더 또는 파일 이동 시 확인 화면을 표시할지 여부를 사용자 페이지에서 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지에서 폴더 또는 파일을 이동할 때 확인 화면을 표시할지 여부를 사용자 페이지에서 변경하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 본 매뉴얼에서는 Web 브라우저에서의 조작에만 적용됩니다.
* 본 매뉴얼에서는 드래그 앤 드롭으로 폴더 또는 파일을 이동하는 경우에만 적용됩니다.\
  [폴더 또는 파일을 이동할 때 확인 화면 표시 설정을 변경하는 방법](https://help.directcloud.net/admin_manual/detail_settings/move_confirm_popup)에서 관리자가 '항상 사용'으로 설정한 경우에는, 본 매뉴얼에서는의 절차에 따라 사용자 페이지에서 설정을 변경할 수 없습니다.
* 본 매뉴얼에서는의 설정은 설정한 사용자에게만 적용됩니다.

***
