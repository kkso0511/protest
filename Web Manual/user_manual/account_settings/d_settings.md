# DirectCloud 드라이브 설정 화면

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 DirectCloud Drive 설정 화면에 표시되는 정보, 설정 버튼, 상세 설정 메뉴에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼에서는 Windows용 DirectCloud Drive를 예시로 설명합니다.
* 관리 페이지에서 오프라인 기능이 활성화되어 있는 경우, DirectCloud Drive의 상세 설정 메뉴에 오프라인 설정이 표시됩니다.
* 관리 페이지의 '계정관리'->'Guest'->'기능 예외 설정'에서 Home이 비활성화되어 있는 경우, Guest 계정으로 DirectCloud Drive에 로그인하면 다음 항목이 표시되지 않습니다.
* My Box 사용량(사용률) / 사용자 할당 용량
  * 코멘트
  * 최근 사용한 파일
  * 즐겨찾기
  * 공유된 파일
  * Link History
  * 링크로 업로드된 파일
  * 자동 이동 이력
* Warm Storage를 사용할 수 있다면 자동 이동 이력이 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**DirectCloud 드라이브 설정 화면 표시**
{% endhint %}

1. DirectCloud 드라이브에 로그인합니다.<br>
2. PC 트레이아이콘에서 DirectCloud 드라이브 아이콘을 클릭합니다.\
   DirectCloud 드라이브 설정 화면이 표시됩니다.
