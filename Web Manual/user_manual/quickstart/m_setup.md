# 모바일 앱 설치 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 DirectCloud 제품 페이지에서 iOS 또는 Android 애플리케이션을 설치하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 스마트폰, 태블릿에서 'App Store'로 이동하여 'DirectCloud' iOS 애플리케이션을 설치할 수 있습니다.
* 스마트폰, 태블릿에서 'Google Play 스토어'로 이동하여 'DirectCloud' Android 애플리케이션을 설치할 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션 설치**
{% endhint %}

&#x20;**DirectCloud** **홈페이지에서 다운로드**

1. web 브라우저에서 DirectCloud 홈페이지의 제품다운로드 페이지로 이동합니다.
2. 화면을 아래로 스크롤하고 iOS에서 다운로드 버튼을 클릭합니다.\
   App Store 설치 화면이 표시됩니다.
3. 가져오기를 탭합니다.\
   DirectCloud 애플리케이션이 설치됩니다.
4. 모바일 홈 화면에 'DirectCloud' 애플리케이션이 표시되는지 확인합니다.



**앱 스토어에서 다운로드**

1. 앱 스토어를 실행합니다.
2. 검색칸에 '다이렉트클라우드'를 입력합니다.
3. 검색 된 목록 중 '기업용 웹하드 DirectCloud' 를 선택합니다.
4. 파일을 설치 합니다.



{% hint style="warning" %}
**Android 애플리케이션 설치**
{% endhint %}

&#x20;**DirectCloud** **홈페이지에서 다운로드**

1. web 브라우저에서 DirectCloud 홈페이지의 제품 다운로드 페이지로 이동합니다.
2. 화면을 아래로 스크롤하고 Android의 다운로드 버튼을 클릭합니다.\
   Google Play 스토어 설치 화면이 표시됩니다.
3. 설치를 탭합니다.\
   DirectCloud 애플리케이션이 설치됩니다.
4. 모바일 홈 화면에 'DirectCloud' 애플리케이션이 표시되는지 확인합니다.



**Google Play 스토어에서 다운로드**

1. Google Play 스토어를 실행합니다.
2. 검색칸에 '다이렉트클라우드'를 입력합니다.
3. 검색 된 목록 중 '기업용 웹하드 DirectCloud' 를 선택합니다.
4. 파일을 설치 합니다.
