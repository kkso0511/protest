# 업데이트 알림을 통해 DirectCloud Drive를 업데이트하는 방법

### 개요 <a href="#a03" id="a03"></a>

업데이트 또는 유지 관리로 인해 애플리케이션이 업그레이드되면 다음과 같은 방법으로 알림을 받게 됩니다.

* 업데이트 알림 대화 상자
* 알림 메시지
* 작업 트레이 아이콘의 배지
* 알림 표시줄 업데이트

업데이트 알림에서 새 버전의 DirectCloud Drive로 업데이트하는 단계는 다음과 같습니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼에서는 윈도우용 DirectCloud 드라이브를 예로 들어 설명합니다.
* DirectCloud 드라이브를 새 버전으로 업데이트하는 사용자에게는 Windows 관리자 권한이 있어야 합니다.
* 자동 설치 프로그램을 사용하여 DirectCloud 드라이브를 설치하면 업데이트 알림이 표시되지 않습니다.
* 같은 PC에서 다른 Windows 사용자가 DirectCloud 드라이브를 사용하는 경우 Windows 관리자 권한이 있는 사용자가 DirectCloud 드라이브를 설치하려고 하면 "다른 사용자가 DirectCloud 드라이브를 사용하고 있습니다"라는 화면이 표시됩니다.\
  화면의예(Y)를 클릭하면 다른 사용자가 사용하는 DirectCloud 드라이브가 종료되고 설치가 실행됩니다.
* 같은 PC에서 다른 Windows 사용자가 DirectCloud 드라이브를 사용하는 경우 Windows 관리자 권한이 없는 사용자가 DirectCloud 드라이브를 설치하려고 하면 관리자 암호 입력 화면이 표시됩니다.\
  암호를 입력하고 예를 클릭하면 다른 사용자가 사용하는 DirectCloud 드라이브가 자동으로 종료되고 설치가 실행됩니다.
* DirectCloud 드라이브를 업데이트하려면 재부팅이 필요하거나 필요하지 않을 수 있습니다.\
  본 매뉴얼에서는 재부팅이 필요한 경우의 단계를 설명합니다.\
  재부팅이 필요하지 않으면 업데이트가 완료된 후 DirectCloud 드라이브가 이 컴퓨터에 설치되었습니다.라는  메시지가 표시됩니다.
* 2025년 9월 26일 업데이트 이후에 릴리스된 DirectCloud Drive의 알림 메시지, 작업 트레이 아이콘 배지 및 업데이트 알림 표시줄 지원 버전입니다.
* Mac에는 알림 메시지가 표시되지 않으므로 최신 버전의 Mac용 DirectCloud Drive가 릴리스되면 푸시 알림에 최신 버전 공지가 표시됩니다.\
  Mac의 푸시 알림은 앱별 설정에 따라 시스템 설정 > 알림에 나타납니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**업데이트 알림 확인**
{% endhint %}

새 버전의 DirectCloud Drive가 출시되면 다음 방법 중 하나로 알림을 받게 됩니다.<br>

1. **업데이트 알림 대화 상자**\
   DirectCloud 드라이브를 시작하면 "새 버전이 있는데 업데이트하시겠습니까?" 라는 메시지가 표시됩니다. 알림 대화 상자에는 현재 버전 및 최신 버전 정보가 표시됩니다. 최신 버전의 DirectCloud Drive로 업데이트하면 이 대화 상자가 더 이상 표시되지 않습니다.\
   "예" 버튼을 클릭하여 업데이트를 시작합니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-26 144207.png" alt="" width="344"><figcaption></figcaption></figure>



2. **알림 메시지**\
   DirectCloud Drive에 로그인한 경우 오전 10시에 화면 오른쪽 하단의 알림 메시지에 "최신 버전 알림"이 표시됩니다. 이 알림 알림은 최신 버전의 애플리케이션이 설치될 때까지 매일 표시됩니다.\
   업데이트 알림 대화 상자를 표시하려면 클릭합니다.&#x20;

<figure><img src="../../.gitbook/assets/스크린샷 2025-12-18 101726.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
**NOTE:**\
Windows 앱이 전체 화면 모드(전체 화면 모드)로 실행되는 경우 "최신 버전 공지" 알림 메시지가 표시되지 않습니다. 20분 후 전체 화면 모드인지 다시 확인하고, 그렇지 않은 경우 알림 메시지가 표시됩니다.
{% endhint %}



3. **작업 트레이 아이콘의 배지**\
   작업 트레이에 표시되는 DirectCloud Drive 아이콘에는 업데이트 알림 배지가 표시됩니다. 최신 버전의 DirectCloud Drive로 업데이트하면 배지가 사라집니다.\
   아이콘을 클릭하면 DirectCloud Drive 설정 화면으로 이동합니다.<br>

<figure><img src="../../.gitbook/assets/image (2509).png" alt=""><figcaption></figcaption></figure>



4. **알림 표시줄 업데이트**\
   DirectCloud Drive 설정 화면의 메뉴 위에 업데이트 메시지가 표시됩니다. '✕' 버튼을 클릭하면 닫을 수 있지만, 설정 화면을 다시 열면 다시 나타납니다. 최신 버전의 DirectCloud Drive로 업데이트하면 업데이트 알림 표시줄이 사라집니다.\
   '업데이트' 버튼을 클릭하여 업데이트를 시작합니다.

<figure><img src="../../.gitbook/assets/image (2510).png" alt="" width="563"><figcaption></figcaption></figure>



{% hint style="warning" %}
**DirectCloud 드라이브 업데이트**
{% endhint %}

다음 방법 중 하나로 업데이트를 시작합니다.



1.  **업데이트 알림 대화 상자**\
    업데이트 알림 대화 상자에서 업데이트할 때 "새 버전이 있는데 업데이트하시겠습니까?" 대화 상자에서 예 버튼을 클릭합니다.<br>

    <figure><img src="../../.gitbook/assets/스크린샷 2025-09-26 144207.png" alt="" width="344"><figcaption></figcaption></figure>





2. DirectCloud Drive 설정 화면에서 시작하려면 '업데이트' 버튼을 클릭합니다.\
   업데이트 준비 화면이 표시되면 설치가 자동으로 시작되고 DirectCloud Drive 설정 화면에 설치 진행률이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (1209).png" alt=""><figcaption></figcaption></figure>

&#x20;        설치가 완료되면 DirectCloud 드라이브 설치 완료 화면이 나타납니다.<br>

2. 지금 재부팅 또는 나중에 재부팅을 선택하고 마침 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1210).png" alt=""><figcaption></figcaption></figure>

"지금 재부팅  하겠습니다." 를 선택하면 즉시 컴퓨터가 다시 시작됩니다.\
나중에 수동으로 재부팅을 선택한 경우 사용자는 언제든지 컴퓨터를 다시 시작하여 DirectCloud 드라이브를 사용할 수 있습니다.<br>

3. 다시 시작한 후 바탕화면에 바로 가기 링크가 생성되었는지 확인합니다.&#x20;

<figure><img src="../../.gitbook/assets/image (1211).png" alt=""><figcaption></figcaption></figure>
