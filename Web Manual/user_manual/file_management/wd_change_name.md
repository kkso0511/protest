# \[Web, Drive] 폴더와 파일의 이름을 바꾸는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 DirectCloud에 저장된 폴더와 파일의 이름을 바꾸는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 다음 조건에 해당하는 사용자는 파일 이름을 변경할 수 있습니다.
  * <마스터>, <전체권한> 접근 권한이 부여된 사용자
  * 파일의 최종 수정자로서 <편집자> 접근 권한이 부여된 사용자
* 폴더 이름을 변경하려는 사용자는 해당 폴더에 대해 <마스터>, <전체권한> 접근 권한이 부여되어 있어야 합니다.
* 폴더 및 파일 이름, 전체 경로의 문자 수는 1바이트 기준이 아니라 반각과 전각을 구분하지 않고 1글자로 계산됩니다.
* 폴더 및 파일 이름은 최대 190자까지 입력할 수 있습니다.
* 폴더 및 파일 이름에 사용할 수 있는 문자와 사용할 수 없는 문자는 아래와 같습니다.
  * 사용 가능한 문자\
    UTF-8의 한자, 히라가나, 영문 대소문자, 숫자, 공백, 다국어 문자, 기호, 이모지 등
  * 사용할 수 없는 문자\
    \ / : \* ? " < > |
*   DirectCloud 드라이브의 경우, 앞서 안내한 문자 외에도 아래의 문자는 폴더 및 파일 이름에 사용할 수 없습니다.\
    CON, PRN, AUX, NUL, COM1, COM2, COM3, COM4, COM5, COM6, COM7, COM8, COM9, LPT1, LPT2, LPT

    또한 공백과 마침표(.)는 이름의 첫 글자로는 사용할 수 있지만, 마지막 글자로는 사용할 수 없습니다.
* 폴더의 전체 경로는 260자 이내로 설정해야 합니다.\
  문자 수에는 폴더 구분 기호, 드라이브 문자, 루트 폴더명(DirectCloud), Shared Box의 문자 수도 포함됩니다.
* Mac 버전의 DirectCloud 드라이브를 사용하는 경우, 아래의 제한 사항이 적용되므로 주의하시기 바랍니다.\
  폴더명: 255바이트\
  폴더 전체 경로: 1024바이트
* 변경하려는 폴더 및 파일 이름에 구형 한자가 포함된 경우, 일부 문자는 자동으로 신자체로 변환될 수 있습니다.
* DirectCloud-SHIELD DLP를 계약한 경우, Web 브라우저에는 DLP 메뉴가 표시되고 DirectCloud 드라이브에는 DLP 폴더가 표시됩니다.
* DirectCloud-CONNECT를 이용할 수 있는 경우, Web 브라우저에 Connect 메뉴가 표시됩니다.
* Warm Storage를 사용할 수 있는경우, Web 브라우저에는 Warm Storage 메뉴가, DirectCloud 드라이브에는 Warm Storage 폴더가 표시됩니다.
* Cold Storage를 계약한 경우, Web 브라우저에 Cold Storage 메뉴가 표시됩니다.
* Cold Storage의 경우 '편집자' 접근 권한을 가진 사용자는 저장 후 90일이 지난 파일의 이름을 변경할 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 이름 바꾸기**
{% endhint %}

1. 사용자 페이지에서 폴더를 클릭하여 이름을 바꾸려는 폴더와 파일을 표시합니다.
2. 다음 방법 중 하나를 사용하여 변경할 이름 입력 화면을 표시합니다.\
   ❶ 이름을 변경하려는 폴더 또는 파일에 체크한 후, 상단에 있는 이름 변경 버튼을 클릭합니다.\
   ❷ 이름을 변경하려는 폴더 또는 파일을 오른쪽 클릭하고 표시된 메뉴에서 이름 변경을 클릭합니다.\
   ❸ 이름을 변경하려는 폴더 또는 파일 오른쪽에 있는 ▼ 버튼을 클릭하고 표시된 메뉴에서 이름 변경을 클릭합니다.
3. 이름을 입력하고 이름 바꾸기 버튼을 클릭합니다.\
   폴더나 파일의 이름이 바뀝니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 이름 바꾸기**
{% endhint %}

1. 사용자 페이지에서 폴더를 클릭하여 이름을 바꾸려는 폴더와 파일을 표시합니다.
2. 이름을 바꾸려는 폴더나 파일을 선택하고 마우스 오른쪽 버튼을 클릭하고 표시된 메뉴에서 이름 바꾸기를 선택합니다.
3. 이름을 입력하고 Enter 키를 누릅니다.\
   폴더나 파일의 이름이 바뀝니다.
