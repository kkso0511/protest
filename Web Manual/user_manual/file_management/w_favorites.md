# \[Web] 폴더 및 파일을 즐겨찾기로 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 DirectCloud에 저장된 폴더와 파일을 즐겨찾기로 설정하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일을 즐겨찾기로 등록하려는 사용자는 해당 파일에 대해 업로더를 제외한 접근 권한이 부여되어 있어야 합니다.
* 모든 사용자는 폴더를 즐겨찾기로 설정할 수 있습니다.
* Shared Box, My Box, Connect, DLP 메뉴 아래의 폴더와 파일을 즐겨찾기에 등록할 수 있습니다.
* DLP 메뉴는 DirectCloud의 옵션인 DirectCloud-SHIELD DLP를 계약한 경우에만 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**즐겨찾기 등록**
{% endhint %}

1. 사용자 페이지에서 폴더를 클릭하여 즐겨찾기에 등록할 폴더와 파일을 표시합니다.
2. 다음 방법 중 하나를 사용하여 폴더 또는 파일을 즐겨찾기에 등록합니다.\
   \
   ❶ 즐겨찾기에 등록할 폴더나 파일을 체크하고 상단의 즐겨찾기 버튼을 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❷ 즐겨찾기에 등록할 폴더 또는 파일을 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 즐겨찾기를 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❸ 즐겨찾기에 등록할 폴더 또는 파일의 오른쪽에 있는 ▼ 버튼을 클릭하고 표시된 메뉴에서 즐겨찾기를 클릭합니다.\
   \
   즐겨찾기에 등록된 폴더나 파일의 시작 부분에는 노란색 ☆ 마크가 표시됩니다.



{% hint style="warning" %}
**즐겨찾기 참조**
{% endhint %}

1. 즐겨찾기 메뉴를 클릭합니다.\
   즐겨찾기에 등록된 폴더 및 파일 목록이 표시됩니다.
2. 즐겨찾기 목록에서 파일을 클릭하면 파일을 미리 볼 수 있습니다.\
   미리보기를 닫으면 즐겨찾기 목록으로 돌아갑니다.
3. 즐겨찾기 목록에서 파일 아래에 표시된 폴더 경로를 클릭하면 즐겨찾기로 등록된 파일이 저장된 폴더로 이동합니다.
4. 즐겨찾기 목록에서 폴더를 클릭하면 즐겨찾기로 등록된 폴더로 이동합니다.



{% hint style="warning" %}
**즐겨찾기 해제**
{% endhint %}

**\[즐겨찾기 메뉴에서 취소]**

1. 즐겨찾기 메뉴를 클릭하여 즐겨찾기를 취소하려는 폴더와 파일을 표시합니다.
2. 즐겨찾기를 해제하려는 폴더나 파일을 선택하고 상단의 즐겨찾기 해제 버튼을 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   즐겨찾기 메뉴 목록에서 삭제한 폴더나 파일의 표시가 사라집니다.\
   또, 해제한 폴더나 파일의 선두로부터, 노란색☆마크가 사라집니다.



**\[폴더에서 해제]**

1. 폴더를 클릭하여 즐겨찾기를 취소하려는 폴더와 파일을 봅니다.
2. 다음 방법 중 하나를 사용하여 폴더 또는 파일 즐겨찾기를 해제합니다.\
   \
   ❶ 즐겨찾기를 해제하려는 폴더 또는 파일을 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 즐겨찾기 해제를 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❷ 즐겨찾기를 해제하려는 폴더 또는 파일의 오른쪽에 있는 ▼ 버튼을 클릭한 다음 표시된 메뉴에서 즐겨찾기 해제를 클릭합니다.\
   \
   즐겨찾기 메뉴 목록에서 삭제한 폴더나 파일의 표시가 사라집니다.\
   또, 해제한 폴더나 파일의 선두로부터, 노란색☆마크가 사라집니다.
