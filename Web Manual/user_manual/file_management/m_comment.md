# \[Mobile] 파일 코멘트를 추가하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud에서는 파일에 연결하여 코멘트을 추가할 수 있습니다.\
이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 DirectCloud에 저장된 파일을 표시하고 코멘트을 추가하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* <업로더>의 접근권한이 부여되고 있는 사용자 이외는, 파일 코멘트 기능을 이용할 수 있습니다.\
  자세한 내용은 [접근 권한의 종류와 Mobile 앱 / Mobile Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/m_permission)을 참고해 주시기 바랍니다.
* 아래의 모든 조건에 해당하는 경우 파일 코멘트를 추가할 때 알림 이메일이 전송됩니다.
  * [파일 코멘트 기능을 활성화하는 방법](https://help.directcloud.net/admin_manual/detail_settings/enable_comment) 및 [특정 폴더에서 코멘트를 사용하도록 설정하는 방법](https://help.directcloud.net/admin_manual/folder/enable_comment)에서 파일 코멘트 기능을 사용으로 설정
  * [파일에 코멘트가 추가되었을 때 알림 메일을 보낼지 여부를 관리 페이지에서 설정하는 방법](https://help.directcloud.net/admin_manual/detail_settings/comment_email_notification)에서 메일 알림을 사용함으로 설정
  * [사용자 페이지에서 알림 메일의 수신 여부를 설정하는 방법](https://help.directcloud.net/user_manual/account_settings/mail_reception)에서 사용자 페이지의 알림 설정을 수신함으로 설정
  * 본 매뉴얼의 절차에 따라 코멘트 입력란 왼쪽 아래에 있는 메일 알림에 체크
* 코멘트 등록 알림 메일에서 링크를 클릭하여 댓글을 표시하면 반드시 Web 브라우저에서 열립니다. 다른 애플리케이션에서는 직접 댓글을 표시할 수 없습니다.
* 댓글 등록 알림 메일의 발신자 정보에 포함되는 이름은 no-reply 메일에 회신 가능 여부에 따라 달라집니다.
  * no-reply 메일에 회신 가능: 사용자 이름no-reply@directcloud.jp
  * no-reply 메일에 회신 불가: DirectCloudno-reply@directcloud.co.jp
* ‘코멘트’ 화면에서 ‘To’를 클릭했을 때 표시되는 선택 화면에는 사용자, 그룹, Guest를 합쳐 최대 3,000건까지 표시됩니다. 표시되지 않은 사용자는 검색하여 추가할 수 있습니다.
* ‘To’에서 코멘트 등록 알림 메일을 보낼 사용자를 설정할 경우, 최대 100명까지 추가할 수 있습니다.
* 미리보기 세션이 만료되면 세션 만료 화면이 표시됩니다. 미리보기를 보려면 화면을 업데이트하세요.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션, Android 애플리케이션에서 파일 코멘트 추가**
{% endhint %}

1. [\[Mobile\] 파일을 다운로드하지 않고 미리보기하는 방법](https://help.directcloud.net/user_manual/file_management/m_preview)의 순서로 미리보기 화면을 표시합니다.
2. 미리보기 화면에서 아래의 코멘트 아이콘을 클릭합니다.\
   코멘트 화면이 표시됩니다.
3. 아래의 코멘트 만들기 메뉴를 탭합니다.\
   코멘트 작성 화면이 표시됩니다.
4. ‘내용’란에 코멘트를 입력합니다.
5. 폴더에 접근 권한을 가진 사용자 중, 접근 권한이 ‘업로더’가 아닌 모든 사용자에게 알림 메일을 보내고 싶은 경우, 이메일 알림’을 탭하고 ‘전체’를 선택합니다.
6. 선택한 사용자에게만 알림 메일을 보내고 싶은 경우, 다음 절차로 설정합니다.\
   ❶ ‘To’ 오른쪽에 표시된 아이콘을 탭하고, 표시된 사용자 목록에서 코멘트의 수신자로 설정하고 싶은 사용자를 모두 탭하여 선택합니다. 이때, 표시되지 않은 사용자는 ‘사용자 이름’ 또는 ‘사용자 ID’로 검색하거나, 표시된 사용자를 필터링하여 볼 수 있습니다.\
   **NOTE:** 사용자가 추가되지 않은 경우, ‘모든 사용자’가 알림 메일 전송 대상이 됩니다.\
   \
   ❷ 오른쪽 상단의 ‘추가’를 탭합니다.\
   사용자가 추가됩니다.\
   \
   ❸ ‘메일 알림’을 탭하고 ‘선택한 사용자’를 선택합니다.\
   **NOTE:** 절차 ❷에서 사용자를 추가했더라도, ‘모든 사용자’가 선택되어 있으면 ‘모든 사용자’가 알림 메일 전송 대상이 됩니다.
7. 완료 버튼을 누릅니다.\
   코멘트가 추가되고 알림 메일이 발송됩니다.



{% hint style="warning" %}
**iOS 앱, Android 앱에서 코멘트를 확인하고 답장하기**
{% endhint %}

파일에 코멘트가 추가되면, ‘○○님이 코멘트를 남겼습니다.’라는 제목의 알림 메일이 전송됩니다. 알림 메일의 링크를 클릭하면 Web 브라우저가 표시되지만, 이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션에 로그인하여 대상 코멘트를 표시하고, 답글을 작성하는 절차에 대해 설명합니다.

1. iOS 애플리케이션 또는 Android 애플리케이션에 로그인한 후, 알림 메일의 ‘파일명’에 기재된 파일을 미리보기하고 ‘코멘트’ 화면을 표시합니다.
2. 답글을 작성하고 싶은 코멘트의 오른쪽 하단에 표시된 답글 버튼을 탭합니다.\
   ‘내용’란에 원래 코멘트가 인용된 상태로 ‘코멘트 작성’ 화면이 표시됩니다.
3. ‘To’에 답글 수신자를 설정하고 코멘트를 입력한 후, ‘완료’ 버튼을 클릭합니다.\
   답글 코멘트가 등록됩니다.
