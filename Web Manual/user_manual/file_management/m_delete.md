# \[Mobile] 파일을 삭제하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 DirectCloud에 저장된 파일을 삭제하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일을 삭제하는 사용자에게는 <마스터>, <전체권한>, <편집자>에 대한 접근 권한이 부여되어야 합니다.\
  '편집자' 접근 권한이 부여된 사용자는 자신이 업로드한 파일만 삭제할 수 있습니다.\
  자세한 내용은 [접근 권한의 종류와 Mobile 앱 / Mobile Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/m_permission)을 참고해 주시기 바랍니다.
*   삭제한 파일은 저장된 위치에 따라 삭제한 사용자의 휴지통 탭으로 이동합니다.

    * 'My Box' 및 'Shared Box' 파일:  사용자 휴지통의 'My Box/Shared Box' 탭으로 이동
    * "Connect"의 파일: 사용자의 휴지통 내 "Connect" 탭으로 이동

    그러나 사용자 휴지통의 탭 이름은 사용 가능한 옵션에 따라 달라집니다.
* 사용자의 휴지통에 저장된 파일을 삭제하면 관리자의 휴지통으로 이동합니다.
* 관리자 페이지에서 '용량 관리'> '스토리지 현황' 메뉴에서 사용자의 휴지통에 저장된 파일을 확인할 수 있습니다.
* 사용자의 휴지통이 사용 안 함으로 설정된 경우 삭제된 폴더 파일은 관리자의 휴지통으로 이동하지 않고 영구적으로 삭제됩니다.
* 사용자의 휴지통을 사용하도록 설정하면 웹 응용 프로그램에서 폴더 파일을 삭제할 때 확인 화면이 표시되지 않습니다.
* 파일을 한 번에 삭제할 때 삭제할 수 있는 파일 수에는 제한이 없습니다.
* 삭제 상태 화면에는 세션 시간이 없으므로 처리 시간이 초과되어 시간이 초과되지 않습니다.
* 삭제를 처리하는 동안 네트워크에 오류가 발생하면 "복사/이동 중 타임아웃이 발생하여 더 이상 진행할 수 없습니다."라는 오류 메시지가 표시됩니다.\
  이 때 오류 메시지가 표시되기 전에 처리 된 파일은 삭제되지만 처리 대기 상태의 파일은 삭제되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 파일 삭제**
{% endhint %}

**\[파일 삭제]**

1. 삭제할 파일이 저장된 폴더를 탭합니다.
2. 삭제할 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000QzD\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 삭제를 탭합니다.\
   확인 화면이 표시됩니다.
4. '확인'을 탭합니다.\
   삭제가 시작되고 상태 화면이 표시됩니다.\
   삭제가 완료되면 "삭제했습니다."라는 화면이 나타납니다.
5. '확인'을 탭합니다.\
   파일이 삭제됩니다.<br>

**\[여러 파일 삭제]**

1. 삭제할 파일이 저장된 폴더를 탭합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000QzD\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)' 버튼을 탭한 다음 표시된 메뉴에서 '복수 선택'을 탭합니다.\
   폴더와 파일의 왼쪽에 선택 버튼이 표시됩니다.
3. 삭제할 파일을 선택하고 오른쪽 하단의 휴지통 버튼을 탭합니다.\
   확인 화면이 표시됩니다.
4. '확인'을 탭합니다.\
   삭제가 시작되고 상태 화면이 표시됩니다.\
   삭제가 완료되면 "삭제했습니다."라는 화면이 나타납니다.
5. '확인'을 탭합니다.\
   여러 파일이 삭제됩니다.

2. 원하는 폴더나 파일을 길게 누릅니다.\
   폴더와 파일을 여러 개 선택할 수 있습니다.
3. 삭제할 여러 파일을 선택하고 오른쪽 하단의 휴지통 버튼을 탭합니다.\
   확인 화면이 표시됩니다.
