# \[Mobile] 파일을 이동하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 DirectCloud에 저장된 파일을 DirectCloud의 다른 폴더로 이동하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

*   파일 이동 기능은 내부적으로 이동 대상 파일을 삭제한 후, 이동할 폴더로 업로드하는 방식으로 처리됩니다. 따라서 파일을 이동하려는 사용자는 이동 전 폴더와 이동 후 폴더 모두에서 아래의 접근 권한을 가지고 있어야 합니다.

    * 이동 전 폴더:\
      <마스터>, <전체권한>
    * 이동 후 폴더:\
      <마스터>, <전체권한>, <편집자>, <편집자－>, <프리뷰어＋>, <업로더>

    자세한 내용은 [접근 권한의 종류와 Mobile 앱 / Mobile Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/m_permission)을 참고해 주시기 바랍니다.
* 이동 대상 폴더에 설정된 접근 권한에 따라 이동 작업이 제한되지는 않습니다.
* 이동 화면에서 대상 폴더에 새 폴더를 만들려면 마스터, 전체권한의 접근 권한이 부여되어야 합니다.
* 여러 개의 폴더와 파일을 선택한 경우, 선택된 목록의 위에서부터 순서대로 이동이 실행됩니다.
* 이동 대상 폴더에 동일한 이름의 폴더 또는 파일이 있을 경우, 이동할 수 없습니다.
* 이미 링크가 생성된 파일을 복사하는 경우에는 링크 정보가 복사되지 않지만, 파일을 이동하는 경우에는 링크 정보가 유지됩니다.
* 한 번에 이동할 수 있는 파일 수에는 제한이 없습니다. 하지만 처리 부하로 인해 속도가 저하될 가능성이 있으므로, DirectCloud에서는 한 번에 1,000개 이상의 폴더 및 파일을 이동하는 작업은 권장하지 않습니다.
* 여러 폴더와 파일을 동시에 이동할 때, 한 번에 이동할 수 있는 하위 폴더의 총수는 10,000개입니다.\
  10,001개 이상의 하위 폴더를 포함한 상위 폴더를 이동하려고 하면 "복사 또는 이동 대상이 최대값(10,000개)을 초과하여 조작을 진행할 수 없습니다." 라는 오류 메시지가 표시되며, 이동 처리는 중단됩니다.
*   이동 처리 중 세션 시간이 10분을 초과한 경우에는 "복사 또는 이동 중에 타임아웃이 발생하여 작업이 중단되었습니다." 라는 오류 메시지가 표시됩니다.

    이 경우 오류 메시지가 표시되기 전까지 이미 처리된 폴더와 대상 폴더에 저장된 폴더·파일은 이동되지만, 처리 대기 중이던 폴더·파일은 이동되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 파일 이동**
{% endhint %}

**\[파일 이동]**

1. 이동하려는 파일이 저장된 폴더를 탭합니다.
2. 이동하려는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000vhR\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 이동을 탭합니다.\
   이동 화면이 표시됩니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 새 폴더를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   이동이 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.<br>

**\[여러 파일 이동]**

1. 이동하려는 파일이 저장된 폴더를 탭합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000vhR\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)' 버튼을 탭한 다음 표시된 메뉴에서 '복수 선택'을 탭합니다.\
   폴더와 파일의 왼쪽에 선택 버튼이 표시됩니다.
3. 이동할 파일을 선택하고 오른쪽 하단의 '' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000vhR\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g5OS)버튼을 탭합니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 새 폴더를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   이동이 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.<br>

{% hint style="warning" %}
**Android 애플리케이션에서 파일 이동**
{% endhint %}

**\[파일 이동]**

1. 이동하려는 파일이 저장된 폴더를 탭합니다.
2. 이동하려는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000vhR\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 이동을 탭합니다.\
   이동 화면이 표시됩니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 폴더 만들기를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   이동이 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.<br>

**\[여러 파일 이동]**

1. 이동하려는 파일이 저장된 폴더를 탭합니다.
2. 원하는 폴더나 파일을 길게 누릅니다.\
   폴더와 파일을 여러 개 선택할 수 있습니다.
3. 이동하려는 여러 파일을 선택하고 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000vhR\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g5OS)' 버튼을 누릅니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 폴더 만들기를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   이동이 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.
