# 온라인 편집 화면에서 시트 보호를 활성화하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 온라인 편집 화면에서 시트 보호를 활성화하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 온라인 편집을 사용하려면, [파일을 다운로드하지 않고 온라인으로 편집할 수 있도록 설정하는 방법](https://help.directcloud.net/admin_manual/detail_settings/online_edit)에 따라 온라인 편집 기능이 활성화되어 있어야 합니다.
* <마스터>, <전체권한>, <편집자>, <편집자-> 접근 권한이 부여된 사용자가 온라인 편집을 이용할 수 있습니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/w_permission), [접근 권한의 종류와 DirectCloud 드라이브 에서 가능한 조작](https://help.directcloud.net/user_manual/access_permission/d_permission)을 참고해 주시기 바랍니다.
* 온라인 편집에서는 100MB 이하의 파일을 최대 5명이 동시에 편집할 수 있습니다.
*   Microsoft Office가 설치되어 있지 않은 PC에서도 Office 파일을 편집할 수 있습니다.

    온라인 편집에 대응하는 확장자는 다음과 같습니다.\
    .xls, .xlsx, .doc, .docx, .ppt, .pptx
* 확장자가 .doc, .ppt, .xls인 구형 형식의 Office 파일을 온라인 편집하려면, 편집 화면을 표시하기 위해 새로운 형식(.docx, .pptx, .xlsx)으로 변환한다는 확인 대화상자에서 확인 버튼을 클릭해야 합니다.
* 텍스트 파일과 매크로 파일은 온라인 편집을 사용할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 시트 보호 사용**
{% endhint %}

**\[시트를 지정하여 보호 기능 사용]**

1. 파일을 다운로드하지 않고 온라인 편집하는 방법 의 "웹 응용 프로그램에서 온라인 편집 화면 열기"에 설명된 절차에 따라 온라인 편집 화면을 표시합니다.
2. 편집 화면의 시트 이름을 선택하고 마우스 오른쪽 버튼으로 클릭 > 보호를 선택합니다.
3. 임의의 암호를 설정한 후 사용자에게 허용할 작업을 선택하고 보호를 클릭합니다.\
   지정된 시트의 보호가 활성화됩니다.<br>

**\[특정 셀 이외를 지정하여 보호 기능 사용]**

1. 상단 메뉴에서 "보호" > "잠긴 셀"의 선택을 취소하고 편집 화면의 시트 이름을 선택하고 마우스 오른쪽 버튼으로 클릭 > "보호"를 선택합니다.
2. 임의의 암호를 설정한 후 사용자에게 허용할 작업을 선택하고 보호를 클릭합니다.\
   특정 셀 이외의 보호가 활성화됩니다.
