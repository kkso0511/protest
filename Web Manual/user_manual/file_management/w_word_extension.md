# \[Web] 표시되는 파일 이름의 문자 수를 늘리는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Web 브라우저의 폴더·파일 목록에 표시된 파일 이름이 너무 길어 전체 이름을 확인할 수 없을 때, 표시 가능한 글자 수를 늘려 파일 이름 전체를 보이도록 설정하는 방법 또는 긴 파일 이름을 확인하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일 이름의 최대 글자 수는 190자입니다.\
  파일 이름 관련 상세한 사양은 [\[Web, Drive\] 폴더와 파일의 이름을 바꾸는 방법](https://help.directcloud.net/user_manual/file_management/wd_change_name)의 제한 및 참고 사항도 함께 확인해주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

다음 방법 중 하나를 사용하여 전체 Web 브라우저에 표시되지 않는 긴 파일 이름을 보거나 확인할 수 있습니다.<br>

1. web 브라우저의 확대율을 축소하여 표시합니다.

{% hint style="info" %}
**NOTE:**\
Ctrl 키를 누른 상태에서 마우스 휠을 위아래로 돌리면 확대 비율을 변경할 수 있습니다.
{% endhint %}



2. PC의 "디스플레이 설정"에서 해상도와 화면 축소율을 설정하여 파일 이름 전체가 표시되도록 합니다.



3. 파일 이름에 마우스 커서를 놓고 툴팁에서 파일 이름을 확인합니다.
