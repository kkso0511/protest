# DirectCloud 드라이브에서 오프라인 설정을 만드는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud 드라이브에는 오프라인 폴더에 파일을 저장하여, 네트워크에 연결되어 있지 않은 상태에서도 파일을 편집할 수 있는 오프라인 기능이 있습니다.\
오프라인 폴더에서 편집한 파일은 네트워크 연결이 온라인 상태로 전환되면 DirectCloud에 반영할 수 있습니다.\
이 매뉴얼에서는 DirectCloud 드라이브에서 오프라인 기능과 관련된 설정을 수행하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

*   오프라인 기능은 DirectCloud 드라이브에서만 이용하실 수 있습니다.

    다음 애플리케이션에서 오프라인 기능을 사용할 수 있습니다.

    * Windows용 DirectCloud 드라이브\
      DirectCloud 드라이브(DirectCloud-SHIELD IRM 통합 버전)\
      DirectCloud 드라이브(DirectCloud-SHIELD DLP 통합 버전)

***

### 절차 <a href="#a05" id="a05"></a>

1. DirectCloud 드라이브에 로그인합니다.



2. PC의 트레이아이콘에서 DirectCloud 드라이브 아이콘을 클릭합니다.\
   DirectCloud 드라이브 설정 화면이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (1576).png" alt=""><figcaption></figcaption></figure>



3. 설정 화면의 오른쪽 상단에 있는 ⁝ 버튼을 클릭하고 표시된 메뉴에서 오프라인 설정을 선택합니다.

<figure><img src="../../.gitbook/assets/image (1577).png" alt=""><figcaption></figcaption></figure>



4. 오프라인 폴더에 같은 이름의 파일을 저장할 때의 동작을 선택합니다.

<figure><img src="../../.gitbook/assets/image (1578).png" alt=""><figcaption></figcaption></figure>

<table><thead><tr><th width="204">항목</th><th>설명</th></tr></thead><tbody><tr><td>덮어쓰기</td><td>파일을 덮어씁니다.</td></tr><tr><td>다른 이름으로 저장</td><td>파일 이름 끝에 숫자가 추가되고 다른 이름으로 파일이 저장됩니다.</td></tr></tbody></table>



5. 설정을 변경한 경우 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1579).png" alt=""><figcaption></figcaption></figure>
