# 폴더 및 파일에 태그를 추가하는 방법

### 개요 <a href="#a03" id="a03"></a>

폴더나 파일에 태그를 추가하면 검색할 때 태그를 지정하여 다른 폴더에 있는 폴더나 파일을 검색하고 원하는 폴더와 파일 목록을 빠르게 표시할 수 있습니다.\
이 매뉴얼에서는 DirectCloud에 저장된 폴더와 파일에 태그를 추가하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 폴더와 파일에 태그를 설정할 수 있는 사용자는 해당 폴더나 파일에 <마스터>, <전체권한>, <편집자>, <편집자－> 접근 권한이 부여되어 있어야 합니다.
* 태그가 설정된 폴더나 파일을 복사/이동한 경우 태그는 다음과 같습니다.
  * 복사:  태그가 삭제됩니다.
  * 이동: 태그가 유지됩니다.
* 최상위(1단계) 폴더에는 태그를 추가할 수 없습니다.
* 서로 다른 태그가 설정된 여러 폴더나 파일을 선택한 후 새 태그를 추가하면, 기존 태그는 새 태그로 덮어씌워집니다.
* 태그의 최대 길이는 10자입니다.
* 하나의 폴더나 파일에는 최대 10개의 태그를 추가할 수 있습니다.
* 여러 폴더나 파일에 동시에 태그를 추가할 경우, 한 번에 태그를 추가할 수 있는 폴더 및 파일의 최대 개수는 50개입니다.
* 태그는 Web 브라우저에서만 표시됩니다.
* DirectCloud-CONNECT를 사용할 수 있는 경우, 사용자 페이지에 Connect 메뉴가 표시됩니다.
* DirectCloud-SHIELD DLP를 계약한 경우, 사용자 페이지에 DLP 메뉴가 표시됩니다.
* Warm Storage를 사용할 수 있는 경우, 사용자 페이지에 Warm Storage 메뉴가 표시됩니다.
* Cold Storage를 계약한 경우, 사용자 페이지에 Cold Storage 메뉴가 표시됩니다.
* Warm Storage 및 Cold Storage의 폴더나 파일에는 태그를 추가할 수 없습니다.
* 태그가 설정된 폴더나 파일을 Warm Storage 또는 Cold Storage로 이동할 경우의 동작은 다음과 같습니다.
  * Warm Storage: 태그 라벨이 표시된 상태로 이동되며, 태그 정보도 유지됩니다.
  * Cold Storage: 이동 시 태그 라벨은 삭제되지만, 태그 정보는 보존됩니다. Cold Storage에서 복원하면 태그 라벨이 다시 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**태그 추가**
{% endhint %}

1. 사용자 페이지에서 폴더를 클릭하여 태그를 추가할 폴더와 파일을 봅니다.



2. 다음 방법 중 하나를 사용하여 폴더 또는 파일에 태그를 추가합니다.\
   ❶ 태그를 추가할 폴더나 파일을 체크하고 상단의 '태그' 버튼을 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❷ 태그를 추가할 폴더나 파일을 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 태그를 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❸ 태그를 추가할 폴더 또는 파일의 오른쪽에 있는 ▼ 버튼을 클릭한 다음 표시된 메뉴에서 태그를 클릭합니다.\
   \
   태그 추가/편집 화면이 표시됩니다.



3. 태그를 입력하고 Enter 키를 누르면 태그가 설정됩니다.\
   계속해서 태그를 입력하고 Enter 키를 누르면 최대 10개까지 태그를 설정할 수 있습니다.



4. 확인 버튼을 클릭합니다.\
   폴더나 파일에 태그가 추가됩니다.



{% hint style="warning" %}
**태그 수정**
{% endhint %}

폴더나 파일에 태그를 추가하거나 제거하고 싶은 경우에도 태그를 추가하면 같은 단계로 편집합니다.

1. 사용자 페이지에서 폴더를 클릭하여 태그를 편집할 폴더와 파일을 봅니다.



2. 다음 방법 중 하나로 폴더 또는 파일의 태그를 편집합니다.\
   ❶ 태그를 편집하고자 하는 폴더나 파일을 체크하고 상단의 '태그' 버튼을 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❷ 태그를 편집할 폴더나 파일을 마우스 오른쪽 버튼으로 클릭하고 표시된 메뉴에서 태그를 클릭합니다.\
   여러 폴더와 파일을 선택할 수 있습니다.\
   ❸ 태그를 편집할 폴더 또는 파일의 오른쪽에 있는 ▼ 버튼을 클릭한 다음 표시된 메뉴에서 태그를 클릭합니다.\
   \
   태그 추가/편집 화면이 표시됩니다.\
   입력란 위에 현재 설정된 태그가 표시됩니다.



3. 태그를 입력하고 Enter 키를 누르면 태그가 설정됩니다.\
   계속해서 태그를 입력하고 Enter 키를 누르면 최대 10개까지 태그를 설정할 수 있습니다.\
   10개 이상의 태그가 설정된 경우 새 태그를 입력할 수 없으므로 다음 단계에서 불필요한 태그를 제거해야 합니다.



4. 태그를 제거하려면 태그 오른쪽에 표시되는 ✕를 클릭합니다.



5. 확인 버튼을 클릭합니다.\
   폴더 또는 파일에 편집한 태그가 반영됩니다.



{% hint style="warning" %}
**태그를 지정하여 검색**
{% endhint %}

DirectCloud 검색에서는 태그를 지정하여 폴더와 파일을 검색할 수 있습니다.

1. 사용자 페이지의 왼쪽 상단에 있는 '검색' 입력란에서 오른쪽 가장자리에 표시되는 ▼ 버튼을 클릭합니다.검색 설정 항목이 표시됩니다.



2. '검색 대상'에서 '태그'를 선택합니다.\
   [\[Web, Drive\] 폴더나 파일을 이름이나 문서내 텍스트 등으로 검색하는 방법](https://help.directcloud.net/user_manual/search/wd_search_file)을 참조하고, 필요에 따라 그 외의 항목을 설정합니다.



3. '검색' 입력란에 검색할 태그를 입력하고 검색 버튼(돋보기)을 클릭합니다.\
   검색 결과 태그 탭에는 입력한 태그와 일치하는 폴더와 파일이 나열됩니다.

* 태그 탭의 검색 결과에서 파일을 클릭하여 파일을 미리 볼 수 있습니다. 미리보기를 닫으면 태그 탭의 검색 결과 목록으로 돌아갑니다.
* 태그 탭의 검색 결과에서 파일 아래에 표시된 폴더 경로를 클릭하면 태그가 추가된 파일이 저장된 폴더로 이동합니다.
* 태그 탭의 검색 결과에서 폴더를 클릭하면 태그가 추가된 폴더로 이동합니다.
