# 사용자 페이지에서 Guest 초대를 수락하는 방법

### 개요 <a href="#a03" id="a03"></a>

접근 권한이 <마스터>로 설정된 사용자는 사용자 페이지에서 Guest 초대를 승인할 수 있습니다.\
이 매뉴얼 에서는 사용자 페이지에서 Guest를 승인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* [관리자 승인 후 Guest에게 초대 이메일을 보내도록 설정하는 방법](https://help.directcloud.net/admin_manual/user/guest_admin_approval)의 절차에서, 관리자 승인 항목이 사용으로 설정되어 있어야 합니다.
* <마스터>가 Guest 초대를 승인하거나 거부한 이력은 관리 페이지의 '로그 현황' > '관리자 페이지 변경 내역'메뉴에 기록됩니다.
* 사용자 ID와 Guest ID에 동일한 문자열을 설정할 수는 없습니다.
* Guest에게 설정할 수 있는 접근 권한은 사용자 및 그룹과 동일합니다.\
  그러나 Guest는 파일첨부 메일발송, 업로드 요청, Guest 초대, Guest 초대 승인, 접근 권한 설정, 폴더 속성 설정 기능을 사용할 수 없습니다.
* Guest는 제1계층 폴더를 조작할 수 없습니다. 따라서 Guest에게 <마스터> 권한을 부여할 수 없습니다.
* 사용자 페이지에서는 Guest를 초대했을 때의 정보를 확인할 수 없습니다.\
  [등록된 Guest를 확인하는 방법](https://help.directcloud.net/admin_manual/user/guest_list)의 절차에 따라 관리 페이지에서 확인할 수 있습니다.

### 절차 <a href="#a05" id="a05"></a>

1. 접근 권한이 <마스터>로 부여된 사용자가 사용자 페이지에 로그인한 후, Guest 초대를 승인하려는 폴더에 접근합니다.

<figure><img src="../../.gitbook/assets/image (1094).png" alt=""><figcaption></figcaption></figure>

2. 화면 오른쪽의 슬라이드 메뉴에 'Guest 초대 승인요청이 있습니다.'가 표시되는지 확인하고 '승인' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1095).png" alt=""><figcaption></figcaption></figure>

&#x20;      'Guest 초대에 승인 신청' 화면이 표시됩니다.

3. Guest 초대에 대한 자세한 정보를 확인한 후 승인을 선택하고 확인 버튼을 클릭합니다.
