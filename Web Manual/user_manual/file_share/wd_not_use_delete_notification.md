# \[Web, Drive] 접근 내역이 없는 링크가 만료되었거나 삭제되었을 때 알림 메일이 전송되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 한 번도 접근이 없었던 링크가 만료거나 삭제된 경우, 링크의 발신자와 수신자에게 알림 메일이 전송되도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 구버전의 링크 생성 화면에서는 이 설정을 사용할 수 없습니다.
* 이 매뉴얼의 설정은 My Box의 파일을 선택하여 생성한 링크에도 적용됩니다.
* 접속 기록이 한 번도 없는 링크라도 알림 메일은 전송됩니다. 자세한 내용은 [\[Web, Drive\] 접근 내역이 없는 링크가 만료되었거나 삭제되었을 때 알림 메일이 전송되도록 설정하는 방법](https://help.directcloud.net/user_manual/file_share/wd_not_use_delete_notification)을 참조해 주시기 바랍니다.
* 아래에 해당하는 경우, 링크는 만료됩니다.
  * [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/w_link_check) 또는 [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/d_link_check)에 기재된 링크 생성 화면 기능에서 설정한 링크의 유효 기간이 만료된 경우
  * [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/w_link_check) 또는 [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/d_link_check)에 기재된 링크 생성 화면 기능에서 설정한 링크 접근 횟수의 상한에 도달한 경우
* 아래 항목 중 하나에 해당하면 링크는 삭제됩니다.
  * [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/w_link_check) 또는 [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/d_link_check)[방법](https://help.directcloud.net/user_manual/undefined-4/dirve)을 통해 사용자가 링크를 삭제한 경우
  * [\[Web, Drive\] 링크가 만료되었을 때 자동으로 삭제되도록 설정하는 방법](https://help.directcloud.net/user_manual/file_share/wd_expired_link_auto_delete)의 경우
  * [\[Web, Drive\] 링크 대상 파일이 업데이트 될 때 링크가 삭제되도록 설정하는 방법](https://help.directcloud.net/user_manual/file_share/wd_update_link_auto_delete)의 경우
* 삭제된 링크를 다시 활성화할 수는 없습니다.
* 링크를 생성한 이후, 대상 폴더 또는 파일이 저장된 폴더에 대한 접근 권한이 삭제되면 링크로 접속하더라도 해당 폴더나 파일에 접근할 수 없습니다.\
  또한 링크 생성자의 접근 권한이 삭제되면, 링크 수신자 역시 해당 폴더나 파일에 접근할 수 없습니다.
* 이메일 옵션 설정은 생성된 링크를 메일로 전송하는 경우에만 적용됩니다.\
  생성된 링크를 복사하여 사용하는 경우, 예를 들어 SNS 등에 붙여넣어 공유하는 경우에는 적용되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서의 설정**
{% endhint %}

1. [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/w_link_check)을 참조하여 '링크 생성'화면을 표시합니다.



2. 이메일 옵션을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1314).png" alt=""><figcaption></figcaption></figure>

&#x20;      이메일 옵션 설정이 표시됩니다.



3. '링크 만료/삭제 시까지 접속자가 없는 경우 알림'을 체크하고 이메일 옵션 설정 화면 이외의 장소를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1315).png" alt=""><figcaption></figcaption></figure>

&#x20;      설정은 즉시 저장됩니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 설정**
{% endhint %}

1. [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/d_link_check)을 참조하여 '링크 생성'화면을 표시합니다.



2. 메일 옵션을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1316).png" alt=""><figcaption></figcaption></figure>

&#x20;      메일 옵션 설정이 표시됩니다.



3. '링크 만료/삭제 시까지 접속자가 없는 경우 알림'을 체크하고 메일 옵션 설정 화면 이외의 장소를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1317).png" alt=""><figcaption></figcaption></figure>

&#x20;      설정은 즉시 저장됩니다.



{% hint style="warning" %}
**링크 종료 이메일 확인**
{% endhint %}

링크가 사용 중지되거나 삭제된 경우 발신자 및 수신자에게 다음과 같은 "접속이력이 없는 링크가 종료되었습니다."라는 알림 메일이 전송됩니다.
