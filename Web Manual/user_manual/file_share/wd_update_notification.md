# \[Web, Drive] 링크 대상 폴더 파일이 업데이트 될 때 알림 메일이 전송되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 전송한 링크의 폴더 또는 파일이 업데이트되었을 때, 링크의 발신자와 수신자에게 알림 메일이 전송되도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 구버전의 링크 생성 화면에서는 이 설정을 사용할 수 없습니다.
*   폴더 링크를 전송한 경우, 아래와 같은 작업으로 대상 폴더 바로 아래에 있는 파일이 업데이트되면 "링크 대상의 파일·폴더가 업데이트되었습니다." 라는 알림 메일이 전송됩니다.

    * 대상 폴더 바로 아래에 파일을 업로드한 경우
    * 대상 폴더 바로 아래에 있는 파일을 덮어쓴 경우

    단, 폴더 바로 아래에 있는 파일이 삭제된 경우에는 알림 메일이 전송되지 않습니다.
* 링크 생성 이후, 대상 폴더 또는 파일이 저장된 폴더에 대한 접근 권한이 삭제되면 링크로 접속하더라도 해당 폴더나 파일에 접근할 수 없습니다.\
  또한 링크 생성자의 접근 권한이 삭제되면 링크 수신자 역시 해당 폴더나 파일에 접근할 수 없습니다.
* 이 매뉴얼의 설정은 My Box에서 생성한 링크에도 적용되므로, 대상 폴더 바로 아래의 파일이 업데이트되면 알림 메일이 전송됩니다.\
  단, 링크의 업로드 기능을 사용하여 파일을 업로드한 경우에는 알림 메일이 전송되지 않습니다.\
  링크 업로드 기능에 대한 자세한 내용은 [링크 수신 화면에서 미리보기, 다운로드, 업로드를 하는 방법](https://help.directcloud.net/user_manual/file_share/management_in_link)을 참조해 주시기 바랍니다.
* 링크 수신 화면에서 파일을 미리보기 할 때 표시되는 날짜와 시간은 파일의 업데이트 시각이며, 숨길 수 없습니다.\
  동일한 파일명으로 덮어쓰는 경우, 링크 수신 화면에 표시되는 파일의 업데이트 시각도 함께 변경됩니다.
* [파일을 다운로드하지 않고 온라인으로 편집하는 방법](https://help.directcloud.net/user_manual/file_management/web_edit)에 따라 파일을 수정하면 파일이 덮어써지면서 업데이트 시각이 변경되지만, 이 매뉴얼의 절차에 따라 알림 메일을 전송하도록 설정할 수는 없습니다.
* 이메일 옵션 설정은 생성된 링크를 메일로 전송하는 경우에만 유효합니다.\
  생성된 링크를 복사하여 사용하는 경우, 예를 들어 SNS 등에 붙여넣어 공유하는 경우에는 적용되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서의 설정**
{% endhint %}

1. [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/w_link_check)을 참조하여 '링크 생성'화면을 표시합니다.



2.이메일 옵션을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1314).png" alt=""><figcaption></figcaption></figure>

&#x20;      이메일 옵션 설정이 표시됩니다.



3. '파일(폴더)이 갱신된 경우 알림'에 체크를 하고, 이메일 옵션 설정 화면 이외의 장소를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1319).png" alt=""><figcaption></figcaption></figure>

&#x20;      설정은 즉시 저장됩니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 설정**
{% endhint %}

1. [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/file_share/d_link_check)을 참조하여 '링크 생성'화면을 표시합니다.



2. 이메일 옵션을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1316).png" alt=""><figcaption></figcaption></figure>

&#x20;      이메일 옵션 설정이 표시됩니다.



3. '파일(폴더)이 갱신된 경우 알림'에 체크를 하고, 이메일 옵션 설정 화면 이외의 장소를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1321).png" alt=""><figcaption></figcaption></figure>

&#x20;      설정은 즉시 저장됩니다.



{% hint style="warning" %}
**알림 이메일 확인**
{% endhint %}

링크 대상 폴더나 파일이 수정되면, 링크를 보낸 사람과 받은 사람에게\
“링크 대상 파일(폴더)이 업데이트되었습니다.”라는 알림 메일이 발송됩니다.
