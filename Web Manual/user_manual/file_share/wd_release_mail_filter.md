# \[Web, Drive] 메일 발송시 발송제한 필터에 등록되어 있는 메일 주소를 해제하는 방법

### 개요 <a href="#a03" id="a03"></a>

공유 링크, 파일첨부 메일발송, 업로드 요청, Guest 초대, Connect User 초대 중 하나의 기능으로 메일을 전송하려 할 때, 발송 제한 중인 메일 주소 화면이 표시되는 경우, 받는 사람에 입력한 메일 주소가 DirectCloud 메일 서버의 필터에 등록되어 있어 해당 메일 주소로의 발송이 제한되고 있는 상태입니다.\
이 매뉴얼에서는 메일을 전송할 때 '이메일 발송 제한' 화면에서 필터를 해제하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* DirectCloud 드라이브에서는 Guest 초대, Connect User 초대, 파일첨부 메일발송을 사용할 수 없습니다.
* 필터에 등록되어 있는 메일 주소가 DirectCloud 사용자 계정의 메일 주소인 경우, [메일 서버의 필터에 등록된 메일 주소를 관리 페이지에서 해제하는 방법](https://help.directcloud.net/admin_manual/user/mail_filter_release)을 통해 계약 담당 관리자 및 일반 관리자가 해제할 수 있습니다.
* 사용자는 필터에 등록된 자신의 메일 주소를 직접 해제할 수 있습니다.\
  자세한 내용은 [이메일 주소가 비활성화 됐을 때 사용자 페이지에서 해제하는 방법](https://help.directcloud.net/user_manual/account_settings/mail_activation)을 참고해주시기 바랍니다.
* 메일 주소가 필터에 등록되는 원인에 대해서는, [알림 조건을 충족하고 있음에도 알림 메일이 도착하지 않는 경우의 대처 방법](https://help.directcloud.net/admin_manual/detail_settings/email_not_arrived)을 참고해주시기 바랍니다.
* 이 매뉴얼의 절차로 메일 주소를 필터에서 해제하여 전송하더라도, 메일 주소가 필터에 등록되는 원인이 해결되지 않는 한 해당 메일 주소는 매번 필터에 다시 등록됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 이메일 주소 해제**
{% endhint %}

1. Web 브라우저에서 공유 링크, 파일첨부 메일발송, 업로드 요청, Guest 초대, Connect User 초대 중 하나의 화면을 표시합니다.



2. 필터에 등록되어 있는 메일 주소를 수신자로 설정하고 전송 버튼을 클릭합니다.\
   전송이 제한된 메일 주소 화면에 필터로 인해 전송이 제한되고 있는 메일 주소가 표시됩니다.



3. 필터에서 해제하고 싶은 메일 주소의 체크박스를 선택한 후 전송 버튼을 클릭합니다.\
   필터에 의해 전송이 제한되고 있던 메일 주소로 메일이 전송됩니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 이메일 주소 해제**
{% endhint %}

1. &#x20;DirectCloud 드라이브에서 공유 링크 또는 업로드 요청 화면을 표시합니다.<br>
2.  필터에 등록된 메일 주소를 수신자로 설정한 후 전송 버튼을 클릭합니다.

    전송이 제한된 메일 주소 화면에 필터로 인해 전송이 제한되고 있는 메일 주소가 표시됩니다.<br>
3.  필터에서 해제하려는 메일 주소의 체크박스를 선택하고 전송 버튼을 클릭합니다.

    필터로 인해 전송이 제한되고 있던 메일 주소로 메일이 전송됩니다.
