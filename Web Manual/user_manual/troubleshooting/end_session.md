# 사용자 페이지에서 세션이 종료되었다고 표시되는 경우 해결 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 '세션이 종료되었습니다'라고 표시되는 경우의 대처 방법에 대해 설명합니다.

***

### 절차 <a href="#a05" id="a05"></a>

'세션이 종료되었습니다'로 표시되는 원인에는 다음 세 가지가 있습니다.

* **접속 중인 글로벌 IP 주소가 변경된 경우**\
  프록시 서버를 사용하고 있으며 접속 중인 IP 주소가 변동되는 설정인 경우, IP 주소가 변경되는 시점에 로그아웃됩니다.\
  로그아웃되지 않도록 하려면 접속 중인 글로벌 IP 주소가 변경되지 않도록 설정해야 합니다.
* **관리자 페이지에서 자동 로그아웃이 유효로 설정된 경우**\
  관리자 페이지의 '보안 정책' > '로그인 설정' 메뉴에서 자동 로그아웃하도록 설정되어 있으면, DirectCloud를 일정 시간 동안 조작하지 않았을 때 강제로 로그아웃됩니다.\
  자동 로그아웃되지 않도록 설정하려면 일정 시간 동안 조작이 없을 경우 자동 로그아웃시키는 방법을 참고해 주시기 바랍니다.
* **액세스 토큰의 유효 기간이 만료된 경우**\
  DirectCloud의 모든 세션은 액세스 토큰으로 관리됩니다.\
  토큰의 유효 기간은 최대 60일이므로, '자동 로그아웃'을 설정하지 않은 경우라도 로그인 후 60일이 경과하면 토큰의 유효 기간이 만료되어 다시 로그인이 필요합니다.\
  액세스 토큰의 유효 기간은 연장할 수 없습니다.
