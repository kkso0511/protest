# 관리 페이지에서 제품에 대해 문의하는 방법(비표시)

### 개요 <a href="#a03" id="a03"></a>

DirectCloud 관리 페이지에 제공되는 메뉴에서 문의 양식을 표시하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 당사 홈페이지에서 문의하는 경우 다음 URL로 이동하십시오.\
  [https://directcloud.jp/support/qna\_after](https://directcloud.jp/support/qna_after)

### 절차 <a href="#a05" id="a05"></a>

1. 관리 페이지에 로그인하고 화면 왼쪽 메뉴에서 관리자 이름을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (716).png" alt=""><figcaption></figcaption></figure>

계약 정보 화면이 표시됩니다.

2. '문의'에서 '고급' 버튼을 클릭합니다.

![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000PCF\&feoid=00N2w00000JMb2T\&refid=0EMBB000000EPio)

문의 양식이 표시됩니다.

![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000PCF\&feoid=00N2w00000JMb2T\&refid=0EMBB000000EOT3)
