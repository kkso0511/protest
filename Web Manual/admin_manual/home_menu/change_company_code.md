# 회사코드를 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud 회사코드는 관리자 페이지에서 변경할 수 없습니다. 다만, 아래의 방법으로 문의해 주시는 것으로 대응 가능합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 회사코드는 영문자숫자 6\~40자로 설정할 수 있습니다.
* 알파벳은 소문자만 사용할 수 있습니다.
* 회사코드의 변경은 원칙적으로 1회까지입니다.

***
