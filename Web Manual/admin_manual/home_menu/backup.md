# 관리 페이지에서 원격지 백업을 신청하는 방법(비표시)

### 개요 <a href="#a03" id="a03"></a>

DirectCloud 관리 페이지에 제공되는 메뉴에서 원격지 백업 옵션에 가입하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 신청서를 제출하고 신청하고 싶다면 아래 URL로 이동하여 파일을 다운로드하세요.\
  [https://directcloud.jp/assets/download/DirectCloud\_Application.xlsx](https://directcloud.jp/assets/download/DirectCloud_Application.xlsx)

### 절차 <a href="#a05" id="a05"></a>

1. 관리 페이지에 로그인하고 화면 왼쪽 메뉴에서 관리자 이름을 클릭합니다.

![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000PCK\&feoid=00N2w00000JMb2T\&refid=0EMBB000000EOlv)

계약 정보 화면이 표시됩니다.

2. '계약 정보'에서 '고급' 버튼을 클릭합니다.

![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000PCK\&feoid=00N2w00000JMb2T\&refid=0EMBB000000EOmA)

3. 화면을 아래로 스크롤해, 「옵션 추가의 신청」메뉴의 「재해 대책·원격지 백업」에 체크를 넣고, 「신청」버튼을 클릭합니다.

![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000PCK\&feoid=00N2w00000JMb2T\&refid=0EMBB000000EOmU)
