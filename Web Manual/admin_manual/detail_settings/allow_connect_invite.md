# DirectCloud를 사용하는 다른 회사로부터의 Connect User 초대를 허용하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는, DirectCloud를 사용하고있는 다른 회사의 사용자로부터의 Connect User 초대를 수락하여, 다른 회사의 Connect User로 연결되는 것을 허용하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정'을 사용할 수 있습니다.
* Connect User는, 초대된 폴더 외에는 접근할 수 없습니다.
* Connect User는 '회사 ID'와 '메일 주소'의 조합으로 관리되기 때문에, 회사가 다를 경우 동일한 메일 주소를 사용할 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '공유 설정' > '세부 기능 설정' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하여, 'Connect 연결' 설정에서 '사용'을 선택하고, '저장' 버튼을 클릭합니다.
