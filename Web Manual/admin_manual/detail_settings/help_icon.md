# 도움말 문서의 메뉴를 표시할지 여부를 설정하는 방법 (한국미적용)

### 개요 <a href="#a03" id="a03"></a>

여기에서는 웹 애플리케이션의 사용자 페이지, DirectCloud 드라이브, 모바일 애플리케이션에서 오른쪽 상단에 표시되는 "?"버튼(도움말 버튼)에서 웹 매뉴얼을 보거나 튜토리얼, 빠른 매뉴얼, PDF 매뉴얼을 다운로드합니다. 할 수 있는지 여부를 설정하는 방법을 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 도움말 문서 표시는 표준으로 사용으로 설정됩니다.
* 응용 프로그램에 따라 "?" 버튼(도움말 단추)의 동작과 참조할 수 있는 도움말 문서의 유형이 다릅니다.\
  자세한 내용은 [도움말 버튼 및 메뉴 표시 예를](https://help.directcloud.jp/s/article/operation-of-administrator-function11259#hyouji_rei) 참조하십시오.
* 도움말 표시 설정의 도움말 문서 표시를 사용 안 함으로 설정하면 DirectCloud 드라이브, 모바일 애플리케이션에 ? 버튼(도움말 버튼)이 표시되지 않습니다.
* "도움말 표시 설정"의 "앱 다운로드 표시"와 "도움말 문서 표시"를 모두 사용하지 않음으로 설정하면 웹 응용 프로그램의 사용자 페이지에 "?" 버튼(도움말 버튼)이 표시되지 않습니다. .
* 관리 페이지의 "?" 버튼(도움말 버튼) 및 메뉴를 숨길 수 없습니다.

### 절차 <a href="#a05" id="a05"></a>

#### 도움말 버튼 및 메뉴 표시 예 <a href="#hyouji_rei" id="hyouji_rei"></a>

**웹 애플리케이션 메뉴**

사용자 페이지의 오른쪽 상단에 표시되는 "?" 버튼(도움말 버튼)에 마우스 커서를 올릴 때 표시되는 메뉴를 클릭하면 다음 도움말 문서를 볼 수 있습니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ8000006h1tR)

| 메뉴      | 설명                                                                                             |
| ------- | ---------------------------------------------------------------------------------------------- |
| 지원 센터   | 웹 브라우저의 다른 탭에서 웹 설명서를 제공하는 [DirectCloud 지원 센터](https://help.directcloud.jp/s/) 의 톱 페이지가 표시됩니다. |
| PDF 매뉴얼 | 사용자 페이지에 PDF 매뉴얼 페이지가 표시됩니다.                                                                   |
| 퀵 매뉴얼   | 사용자 페이지에 빠른 설명서 페이지가 표시됩니다.                                                                    |
| 튜토리얼    | 사용자 페이지에 자습서 페이지가 표시됩니다.                                                                       |
| 앱 다운로드  | 사용자 페이지에 앱 다운로드 페이지가 표시됩니다.                                                                    |

**DirectCloud 드라이브 버튼**

"?" 버튼(도움말 버튼)을 클릭하면 웹 브라우저에서 "웹 매뉴얼"의 " [사용자 매뉴얼](https://help.directcloud.jp/s/user-manual) " 페이지가 표시됩니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ8000005yK4D)

**모바일 애플리케이션 메뉴**

* iOS

지원 센터 메뉴를 누르면 웹 브라우저에서 웹 매뉴얼의 [사용자 매뉴얼](https://help.directcloud.jp/s/user-manual) 페이지가 표시됩니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ80000060DBY)

* 안드로이드

지원 센터 메뉴를 누르면 웹 브라우저에서 웹 매뉴얼의 [사용자 매뉴얼](https://help.directcloud.jp/s/user-manual) 페이지가 표시됩니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ8000005zunB)

#### 도움말 문서 표시 설정 전환

1. 환경설정 > 고급설정 메뉴를 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ8000005yJJS)

2. 화면을 아래로 스크롤하여 도움말 표시 설정을 표시한 다음 도움말 문서 표시에서 다음 중 하나를 선택합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ8000005yKVd)

| 설정      | 설명                                                                                                                                                                          |
| ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 사용하다    | <p><strong>웹 애플리케이션:</strong><br>도움말 버튼 메뉴에 “지원 센터”, “PDF 매뉴얼”, “퀵 매뉴얼”, “튜토리얼”을 표시합니다.<br><br><strong>DirectCloud 드라이브, 모바일 애플리케이션:</strong><br>도움말 버튼을 표시합니다.</p>         |
| 사용하지 않음 | <p><strong>웹 애플리케이션:</strong><br>도움말 버튼 메뉴에 “지원 센터”, “PDF 매뉴얼”, “퀵 매뉴얼”, “튜토리얼”을 표시하지 않습니다.<br><br><strong>DirectCloud 드라이브, 모바일 애플리케이션:</strong><br>도움말 버튼을 표시하지 않습니다.</p> |

3. 저장 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001pEz\&feoid=00N2w00000JMb2T\&refid=0EMQ8000005yKYr)
