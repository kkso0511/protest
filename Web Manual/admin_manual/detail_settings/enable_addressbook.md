# 회사·개인 주소록을 활성화하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud에서는 다음의 주소록을 이용할 수 있습니다.

* 개인 주소록:\
  공유 링크 등의 수신자에 추가한 메일 주소가 자동으로 등록되는 주소록
* 회사 주소록:\
  같은 회사 ID의 사용자의 메일 주소가 자동으로 등록되는 주소록
* 주소그룹:\
  용도 등에 따라 그룹을 만들어 메일 주소를 등록할 수 있는 주소록

이 매뉴얼 에서는 회사 주소록을 활성화하거나 비활성화하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정'을 사용할 수 있습니다.
* 이 매뉴얼의 절차로 회사 주소록을 활성화로 설정한 경우에도, Guest에 대해서는 [Guest로 로그인했을 때 주소록 표시 여부를 설정하는 방법](https://help.directcloud.net/admin_manual/user/guest_addressbook)의 설정이 적용됩니다.
* 주소그룹과 개인 주소록은 비활성화할 수 없습니다.
* 이 매뉴얼의 절차에서 회사 주소록을 비활성화로 설정한 경우에도 주소그룹과 개인 주소록은 계속 사용할 수 있습니다.
*   주소록은 다음 작업을 수행할 때 표시할 수 있습니다.

    * 사용자 페이지의 주소록 메뉴를 클릭했을 때(Web 브라우저만 해당)
    * 다음 화면에서 주소록을 표시했을 때:\
      링크 생성\
      첨부 파일 전송\
      업로드 요청 링크 전송\
      Guest 초대\
      Connect User 초대

    주소록에 등록된 주소는 수신자를 입력할 때의 자동 완성 후보로도 표시됩니다.

***
