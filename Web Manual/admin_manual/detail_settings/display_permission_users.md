# 폴더에 대한 접근 권한이 있는 사용자 정보를 사용자 페이지에 표시하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는, 관리자 페이지의 '공유 설정' ＞ '세부 기능 설정'을 표시하고 Shared Box의 사용자 표시 설정, Connect의 사용자 표시 설정, DLP의 사용자 표시 설정, Warm Storage의 사용자 표시 설정, Cold Storage의 사용자 표시 설정에서 폴더에 대해 접근 권한을 가진 사용자, 그룹, Guest, Connect User 정보를 사용자 페이지의 폴더 속성에 표시하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정'을 사용할 수 있습니다.
* 본 설정은 사용자 페이지에 로그인한 사용자에게 <마스터> 권한이 설정되어 있는 폴더에는 적용되지 않습니다.
* 폴더를 지정하여 설정을 적용할 수 없습니다.
* 사용자와 그룹을 나누어 설정할 수 없습니다.
* Shared Box의 사용자 표시 설정은 대상 사용자가 지정한 사용자로 설정되어 있는 폴더에 적용됩니다.
*   이 매뉴얼의 절차에서 사용자 페이지의 폴더 속성에 표시할 대상의 체크를 해제한 경우에도, 로그인 중인 사용자 본인의 정보는 표시됩니다.

    예를 들어, Guest에게 Guest 정보를 표시하지 않도록 설정한 경우에도, Guest로 로그인한 사용자의 정보는 폴더 속성에 표시됩니다.
* DirectCloud-CONNECT를 사용할 수 있는 경우, 상세 설정 메뉴에 Connect의 사용자 표시 설정이 표시됩니다.
* DirectCloud-SHIELD DLP를 계약한 경우, 상세 설정 메뉴에 DLP의 사용자 표시 설정이 표시됩니다.
* Warm Storage를 사용할 수 있다면 Warm Storage 메뉴가 표시됩니다.
* Cold Storage를 계약한 경우, Cold Storage 메뉴가 표시됩니다.
* Warm Storage, Cold Storage의 폴더에서는 Guest를 초대할 수 없기 때문에 Warm Storage의 사용자 표시 설정, Cold Storage의 사용자 표시 설정에는 Guest가 표시되지 않습니다.
* Cold Storage의 폴더에서는 그룹에 접근 권한을 부여할 수 없으므로 Cold Storage의 사용자 표시 설정에는 그룹이 표시되지 않습니다.
