# 일반 관리자가 사용할 수 있는 메뉴를 제한하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 ‘관리자 추가’ 메뉴에서 사용을 제한하고자 하는 메뉴를 관리 페이지에 표시하지 않도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '계정 관리'를 사용할 수 있습니다.
* ‘관리자’ 메뉴는 ‘계약 담당 관리자’ 권한을 가진 사용자 외에는 표시되지 않습니다.
* 계약 담당 관리자의 계정 정보를 알 수 없어 권한을 추가할 수 없는 경우에는 일반 관리자 계정으로 로그인한 후, ‘계정 관리’ > ‘사용자’ 메뉴에서 사용자의 편집 버튼을 클릭하고, ‘사용자 변경’ 화면에서 ‘URL 복사’를 클릭하여 비밀번호를 변경해 주시기 바랍니다.
* 계약 담당 관리자의 메뉴 사용은 제한할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '계정 관리' > '관리자 추가' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1811).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 관리자 추가/관리 설정에서 메뉴 사용을 제한하려는 일반 관리자 편집 버튼을 클릭합니다. '관리자 페이지 접근권한' 화면이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (1812).png" alt=""><figcaption></figcaption></figure>

&#x20;   &#x20;

3. 일반 관리자의 관리 페이지에 표시하지 않으려는 메뉴의 선택을 해제하고 '저장' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1813).png" alt=""><figcaption></figcaption></figure>



4. 저장한 내용으로 관리자 페이지 접근 권한 메뉴가 표시되는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (1814).png" alt=""><figcaption></figcaption></figure>
