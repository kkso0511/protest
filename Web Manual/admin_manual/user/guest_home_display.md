# Guest로 로그인했을 때 ‘Home’ 메뉴 표시 여부를 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 'Guest'관리 메뉴에서 Guest 사용자 페이지에  Home 메뉴를 표시할지 숨길지 선택하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '계정 관리'를 사용할 수 있습니다.
* Home 메뉴는 Web 브라우저, DirectCloud 드라이브, iOS 애플리케이션, Android 애플리케이션, 모바일 Web 브라우저에서 표시됩니다.\
  따라서 PC Agent는 Guest 기능 제한의 대상에 포함되지 않습니다.
*   Guest 기능 제한에서 Home을 '사용 안함'으로 설정하면 화면은 아래와 같이 표시됩니다.

    <table><thead><tr><th width="236">애플리케이션</th><th>영향</th></tr></thead><tbody><tr><td>Web 브라우저</td><td>왼쪽 메뉴와 시작 메뉴에 Home 이 표시되지 않도록 변경됩니다.</td></tr><tr><td>DirectCloud 드라이브</td><td>작업 트레이에 My Box 사용량(사용률) / 사용자 할당 용량, 코멘트, 최근 사용한 파일, 즐겨찾기, 공유된 파일, Link History, 링크로 업로드된 파일이 표시되지 않도록 변경됩니다.<br>단, 상태는 계속 표시됩니다.</td></tr><tr><td>iOS 애플리케이션</td><td>슬라이드 메뉴와 설정 화면의 시작 메뉴 설정에서 Home 이 표시되지 않도록 변경됩니다.</td></tr><tr><td>Android 애플리케이션</td><td>슬라이드 메뉴와 설정 화면의 시작 메뉴 설정에서 Home 이 표시되지 않도록 변경됩니다.</td></tr><tr><td>모바일 Web 브라우저</td><td>슬라이드 메뉴와 설정 화면의 시작 메뉴 설정에서 Home 이 표시되지 않도록 변경됩니다.</td></tr></tbody></table>
* Guest 기능 제한은 Guest별로 설정할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '계정 관리' > 'Guest' 메뉴를 선택하여 'Guest 설정' 탭을 표시합니다.

<figure><img src="../../.gitbook/assets/image (1730).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하여 Guest 기능 예외 설정을 표시합니다.

<figure><img src="../../.gitbook/assets/image (1824).png" alt=""><figcaption></figcaption></figure>



3. Guest 기능 제한에서 Home 설정을 지정하고 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1825).png" alt=""><figcaption></figcaption></figure>
