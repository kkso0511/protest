# 일반 관리자에게 계약 담당 관리자의 권한을 위임하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 계약 관리자로 관리자 페이지에 로그인하고, 관리자 메뉴에서 일반 관리자 권한을 가진 사용자에게 계약 담당 관리자 권한을 위임하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '계정 관리'를 사용할 수 있습니다.
* ‘관리자’ 메뉴는 ‘계약 담당 관리자’ 권한을 가진 사용자 외에는 표시되지 않습니다.
* 계약 담당 관리자의 계정 정보를 알 수 없어 권한을 추가할 수 없는 경우에는 일반 관리자 계정으로 로그인한 후, ‘사용자’ > ‘계정 관리’ 메뉴에서 사용자의 편집 버튼을 클릭하고, ‘사용자 변경’ 화면에서 ‘URL 복사’를 클릭하여 비밀번호를 변경해 주시기 바랍니다.
* 일반 관리자가 한 명도 등록되어 있지 않은 경우에는 계약 담당 관리자 권한을 위임할 수 없습니다.\
  [사용자에게 일반 관리자 권한을 추가하는 방법](https://help.directcloud.net/admin_manual/user/add_admin_permission)에 따라 일반 관리자를 추가한 후, 권한을 위임해 주시기 바랍니다.
* ‘사용자 변경’ 화면에서는 일반 관리자의 사용자 ID를 변경할 수 없습니다. 사용자 ID를 변경하려면 [사용자 ID를 변경하는 방법](https://help.directcloud.net/admin_manual/user/change_userid)의 절차를 따라 주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '계정 관리' > '관리자 추가' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1807).png" alt=""><figcaption></figcaption></figure>



2. '계약 담당 관리자' 설정에서 계약 관리자의 변경 버튼을 클릭합니다.\
   계약 담당 관리자 변경 화면이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (1808).png" alt=""><figcaption></figcaption></figure>

&#x20;    &#x20;

3. 계약 담당 관리자의 권한을 위임하려는 일반 관리자의 선택 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1809).png" alt=""><figcaption></figcaption></figure>



4. 확인 화면에서  OK 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2530).png" alt=""><figcaption></figcaption></figure>



5. 로그인할때 계약담당자였던 계정은 이제 일반관리자로 바뀌게되어어 '계정 관리' > '관리자 추가' 메뉴가 사라지고, 계약담당자 항목에서 사라지게 됩니다.
