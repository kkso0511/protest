# 사용자 정보 내보내기로 깨진 경우의 해결 방법 (일본어 문제로 한국불필요)

### 개요 <a href="#a03" id="a03"></a>

사용자 정보를 CSV 일괄 내보내는 방법의 단계를 통해 사용자 정보를 내보내는 경우 특수한 문자 등이 포함된 이름이 깨질 수 있습니다.\
이 절에서는 사용자 일괄 처리로 내보낸 CSV 파일이 깨진 경우의 해결 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자의 기능 제한으로 "계정 관리"의 권한이 필요합니다.
* Microsoft Office 2013 이전 버전은 유니코드를 지원하지 않으므로 해당 버전을 사용하는 경우 내보낼 파일의 문자 코드로 알맞게 선택해야 합니다.
* 사용자 일괄 처리 화면에서 내보내는 사용자 정보, 템플릿, 그룹 코드 CSV 파일의 문자 코드는 환경 설정 > 고급 설정 메뉴의 CSV 내보내기시 문자 코드 설정에서 설정됩니다. .\
  자세한 내용은 내보낼 CSV 파일의 문자 코드를 설정하는 방법을 참조하십시오.

### 절차 <a href="#a05" id="a05"></a>

사용자 목록 페이지는 문자 코드가 UTF-8로 고정되어 있으므로 UTF-8의 한자, 한글, 영문대 문자, 숫자, 공백, 다국어 문자, 기호 및 이모티콘 등에 해당합니다.\
문자 코드는가 "UTF-8 BOM 포함"으로 설정해주세요.

1. 계정 관리 > 사용자 메뉴를 선택하고 사용자 일괄 처리 버튼을 클릭합니다.
2. CSV 내보내기 버튼을 클릭하여 CSV 파일을 내보냅니다.
