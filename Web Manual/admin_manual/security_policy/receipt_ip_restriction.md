# 공유 링크·업로드 요청에 접근 가능한 IP 주소를 제한하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 ‘보안 정책’ > ‘접속 제어’ 메뉴의 ‘IP 제한’ 탭에서 공유 링크·업로드 요청에 대한 접근을, 지정한 글로벌 IP 주소로 제한하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* 글로벌 IP 주소가 입력되지 않은 상태에서는 ‘허용’ 라디오 버튼을 선택할 수 없습니다.
* 상태가 ‘허용’으로 설정되어 있는 경우, 등록된 글로벌 IP 주소를 모두 삭제할 수 없습니다.
* 등록 가능한 글로벌 IP 주소의 개수에는 제한이 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**링크 수신처 IP 주소 제한 사용**
{% endhint %}

1. '보안 정책' > '접속 제어' 메뉴를 선택하고 'IP 제한' 탭을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (615).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하여 '링크 수신처 IP 제한' 설정을 표시한 다음 '사용'을 선택합니다.\
   설정은 즉시 저장됩니다.

<figure><img src="../../.gitbook/assets/image (616).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**지정된 글로벌 IP 주소에서만 액세스 허용**
{% endhint %}

1. IP 주소 범위를 입력하고 추가 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (617).png" alt=""><figcaption></figcaption></figure>



2. 라디오 버튼의 '허가'을 선택합니다.

<figure><img src="../../.gitbook/assets/image (618).png" alt=""><figcaption></figcaption></figure>



3. 확인 화면에서 확인 버튼을 클릭합니다.\
   입력된 IP 주소로부터의 액세스만 허용되도록 설정됩니다.

<figure><img src="../../.gitbook/assets/image (619).png" alt=""><figcaption></figcaption></figure>





{% hint style="warning" %}
**지정된 글로벌 IP 주소의 액세스를 차단**
{% endhint %}

1. IP 주소 범위를 입력하고 추가 버튼을 클릭합니다.\
   입력된 IP 주소로부터의 액세스가 차단되도록 설정됩니다.

<figure><img src="../../.gitbook/assets/image (620).png" alt=""><figcaption></figcaption></figure>
