# 모바일 애플리케이션의 캐시를 자동 삭제하도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 ‘모바일 보안’ 메뉴에서 모바일 애플리케이션(iOS, Android)의 캐시가 자동으로 삭제되도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* ‘모바일 앱 보안 설정’은 Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자에게도 적용되지만, Connect User에는 적용되지 않습니다.
* ‘모바일 캐시파일 자동 삭제’는 기본적으로 ‘사용 안함’으로 설정되어 있습니다.
* ‘모바일 캐시파일 자동 삭제’를 ‘사용’으로 설정한 경우, 지정한 일수가 경과한 후 사용자가 모바일 애플리케이션에서 DirectCloud에 로그인하는 시점에 캐시가 삭제됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '보안 정책' > '모바일 보안'메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1616).png" alt=""><figcaption></figcaption></figure>



2. '모바일 캐시파일 자동 삭제'에서 '사용'을 선택하고 저장 버튼을 클릭합니다.
