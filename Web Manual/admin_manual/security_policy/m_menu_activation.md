# 모바일 메뉴의 각 설정 항목이 '사용 안 함'으로 고정된 경우 해결 방법

### 개요 <a href="#a03" id="a03"></a>

‘모바일 보안' 메뉴의 ‘이미지 다운로드’, ‘메일로 보내기’, ‘다른 앱으로 열기’, ‘다른 앱으로 보내기’, ‘오프라인 시 액세스’의 기본 설정은 ‘사용 안함’으로 설정되어 있으며, 필요에 따라 ‘사용’으로 변경할 수 있습니다.\
단, 다운로드 워크플로우를 사용하는 설정인 경우, 이러한 설정 항목은 비활성화되어 ‘사용 안함’으로 고정됩니다.\
이 매뉴얼에서는, 이러한 설정 항목을 ‘사용’으로 변경할 수 있도록 승인 워크플로우 설정을 변경하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* ‘모바일 앱 보안 설정’은 Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자에게도 적용되지만, Connect User에는 적용되지 않습니다.
* ‘보안’ > ‘모바일’ 메뉴에서 ‘이미지 다운로드’, ‘메일로 보내기’, ‘다른 앱으로 열기’, ‘다른 앱으로 보내기’, ‘오프라인 시 액세스’ 설정이 ‘사용 안함’으로 고정되면, 모바일 애플리케이션(iOS, Android)에서 ‘⁝’ 버튼을 탭했을 때 ‘이미지 저장’, ‘다운로드’, ‘오프라인에서 사용 가능’ 메뉴가 표시되지 않게 됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '워크플로우' > '설정' 메뉴를 선택합니다.

2. '워크플로우 기본설정'에서 '사용 안함'을 선택하고 저장 버튼을 클릭합니다.
