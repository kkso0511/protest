# 기기 제한을 설정하지 않았음에도 로그인을 할 수 없는 경우의 대처 방법

### 개요 <a href="#a03" id="a03"></a>

'보안 정책' > '접속 제어' 메뉴의 '기기 관리' 탭 에서 기기 인증을 설정하지 않았음에도 로그인이 되지 않는 경우에는 애플리케이션 이용 제한이 설정되어 있을 가능성이 있습니다.\
이 매뉴얼에서는 기기 관리가 비활성화된 상태에서 로그인할 수 없는 경우의 대처 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '보안 정책' > '접속 제어' 메뉴의 '접속 프로그램 제한' 탭으로 이동합니다.



2. '접속프로그램 제한'의 '사용할 응용프로그램'을 선택하고 저장 버튼을 클릭합니다.
