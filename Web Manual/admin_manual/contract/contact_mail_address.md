# 계약 갱신 알림 메일의 이메일 주소를 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 계약 갱신 알림 메일의 대상 이메일 주소를 확인하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 특히 없음

### 설명 <a href="#a05" id="a05"></a>

계약의 갱신일 전에 통지 메일은 관리자 페이지에서 「계약 담당 관리자」의 권한이 설정되어 있는 유저의 메일 주소에 발송됩니다.\
계약 정보를 확인하는 방법의 순서로 표시되는, 「회사 정보」에 등록되어 있는 메일 주소에는 도착하지 않습니다. 계약 갱신 통지 메일 수신자는 다음 방법 중 하나로 변경할 수 있습니다.

* 계약 담당 관리자 권한이 설정된 사용자의 이메일 주소변경
* 일반 관리자에게 계약 담당 관리자의 권한을 위양하는 방법의 순서로 통지 메일을 보내고 싶은 메일 주소가 등록되어 있는 일반 관리자에게 계약 담당 관리자의 권한을 설정합니다.
