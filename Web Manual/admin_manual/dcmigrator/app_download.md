# DCMigrator 애플리케이션을 다운로드 하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud에서는 사내 파일 서버나 네트워크 하드디스크(NAS)에서 DirectCloud로 데이터를 이전하기 위한 데이터 이동 도구인 DCMigrator를 제공하고 있습니다.\
DCMigrator는 Windows PowerShell을 사용하여 파일과 폴더를 DirectCloud로 업로드합니다.\
이 매뉴얼에서는 DCMigrator를 다운로드하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* **DCMigrator의 라이선스에 대하여**\
  DCMigrator(이하 본 애플리케이션) 및 그 복제본에 대한 소유권은 모두 주식회사 다이렉트클라우드(이하 당사)가 보유합니다.\
  당사의 사전 승인 없이 본 애플리케이션의 전부 또는 일부를 형식이나 수단을 불문하고 복제, 개변, 재배포, 다운로드 또는 전송하는 행위를 금지합니다.
*   DCMigrator의 사용 환경은 아래 표를 참고해 주세요.

    <table><thead><tr><th width="133.20001220703125">구분</th><th>특정</th></tr></thead><tbody><tr><td>OS</td><td>윈도우 11 (64비트판)<br>윈도우 10 (64비트판)<br>윈도우 10 (32비트판)<br>윈도우 서버 2016 / 2019 / 2022</td></tr><tr><td>CPU</td><td>Intel 또는 AMD CPU(4코어 이상 2.00GHz 이상 포함)</td></tr><tr><td>메모리</td><td>4.00GB(8.00GB 이상 권장)</td></tr></tbody></table>
* DCMigrator를 이용해 데이터를 이전하려면, DirectCloud 측에서 DCMigrator를 사용할 수 있는 권한을 활성화해야 합니다. 사용을 시작하기 전에 DirectCloud 상담팀에게 DCMigrator 이용 신청을 해주시기 바랍니다.
* 데이터 이동을 수행할 때에는 시작하기 전에 매번 애플리케이션 다운로드 페이지를 확인하여, 최신 버전의 DCMigrator로 데이터 이동을 진행해 주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**DCMigrator 다운로드**
{% endhint %}

1. 다이렉트 클라우드 홈페이지의 문의하기 또는 실시간 문의하기 채팅을 통해 신청해주세요
2. 담당자를 통해 사용자 확인후 DCMigrator다운로드 링크를 보내드립니다.



{% hint style="warning" %}
**DCMigrator 사용 준비**
{% endhint %}

1. 다운로드한  DCMigrator를 PC 바탕 화면으로 이동합니다.
2. DCMigrator의 파일 이름을 DCMigrator.exe로 변경합니다.
