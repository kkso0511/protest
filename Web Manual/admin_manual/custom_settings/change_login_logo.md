# 사용자 로그인 페이지 로고를 변경하는 방법 (미제공)

### 개요 <a href="#a03" id="a03"></a>

이 섹션에서는 사용자 지정 설정 메뉴의 사용자 지정 도메인 설정에서 사용자 지정 도메인을 설정한 경우 사용자 로그인 페이지의 로고를 변경하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 [관리자의 기능 제한](https://boxmanager.directcloud.jp/setting/manager) 으로 "공유"를 선택해야 합니다.
* 기본 도메인의 로고 변경은 비즈니스 플랜 이상에서 사용할 수 있습니다.
* 웹 애플리케이션 로고만 변경할 수 있습니다.
* 업로드 가능한 로고의 이미지 크기는 가로 280픽셀 x 세로 200픽셀 JPG 파일 또는 24비트 PNG 파일입니다.

### 절차 <a href="#a05" id="a05"></a>

#### 로고 이미지 변경

1. 공유 > 사용자 정의 설정 메뉴를 선택합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001DWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000PnGs)

2. [사용자 로그인 페이지의 하위 도메인을 설정하는 방법](https://help.directcloud.jp/s/article/operation-of-administrator-function11161) 또는 [사용자 로그인 페이지의 기본 도메인을 설정하는 방법](https://help.directcloud.jp/s/article/operation-of-administrator-function11162) 에 따라 사용자 지정 도메인을 설정합니다.
3. "사용자 정의 로고"의 "+" 표시를 클릭하고 로고로 사용할 이미지 파일을 선택합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001DWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000PvJV)

선택한 이미지 파일이 "사용자 정의 로고"에 반영됩니다.

4. 저장 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001DWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000PvJk)

맞춤 도메인 설정이 저장됩니다.

#### 변경된 로고 확인

로그인 페이지 로고가 변경되었는지 확인합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001DWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000PvKE)

#### 로고 이미지 삭제

1. 맞춤 로고 설정에서 업로드한 이미지 파일 아래의 재설정 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001DWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000PvKY)

2. 확인 화면에서 확인 버튼을 클릭합니다.

![이미지](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001DWI\&feoid=00N2w00000JMb2T\&refid=0EMBB000000PvKn)

이미지 파일이 삭제되고 사용자 정의 로고가 초기 상태로 돌아갑니다.
