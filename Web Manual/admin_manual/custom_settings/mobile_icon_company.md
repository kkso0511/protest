# Mobile APP의 아이콘과 회사이름을 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 커스텀 설정 메뉴의 커스텀 로고 설정에서 이미지 파일을 업로드하여 iOS 애플리케이션, Android 애플리케이션, 모바일 버전 Web 브라우저의 아이콘을 변경하는 방법을 설명합니다. 또한, 아이콘 옆에 표시되는 회사명을 변경하는 방법도 함께 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정' 메뉴를 사용할 수 있습니다.
* 업로드 가능한 아이콘 이미지의 크기는 가로 150픽셀 × 세로 150픽셀의 JPG 파일 또는 24비트 PNG 파일입니다.
* 회사명으로 설정할 수 있는 문자의 최대 길이는 20자입니다.
* 10MB 이상인 로고 파일은 등록할 수 없습니다.
* 이 매뉴얼의 절차로 커스텀 로고 설정을 삭제하는 경우 [Web, PC Agent, 관리자 페이지의 로고 이미지를 변경하는 방법](https://help.directcloud.net/admin_manual/custom_settings/change_logo)에서 설정한 Web/PC Agent/관리자 페이지 로고도 함께 초기화되므로 주의해야 합니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**맞춤 설정 화면 표시**
{% endhint %}

1. '공유설정' > '맞춤형 설정' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (2024).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**로고 이미지 변경**
{% endhint %}

1. '로고 설정'에서 '모바일용 심볼'의 '+' 마크를 클릭하고 아이콘으로 사용할 이미지 파일을 선택합니다.

<figure><img src="../../.gitbook/assets/image (2314).png" alt=""><figcaption></figcaption></figure>

선택한 이미지 파일이 아이콘에 반영됩니다.

<figure><img src="../../.gitbook/assets/image (2315).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**회사명 변경**
{% endhint %}

1. '로고 설정' 에서 '모바일앱용 회사명 텍스트' 입력란에 변경하려는 회사 이름을 입력하고 '저장' 버튼을 클릭합니다. 회사명이 저장됩니다.

<figure><img src="../../.gitbook/assets/image (2316).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**변경된 아이콘과 회사명 확인**
{% endhint %}

iOS 애플리케이션, Android 애플리케이션, 모바일 버전 Web 애플리케이션에서 아이콘과 회사명이 변경된 것을 확인합니다.

**\[모바일 애플리케이션]**

<figure><img src="../../.gitbook/assets/image (2317).png" alt=""><figcaption></figcaption></figure>



**\[모바일 Web]**

<figure><img src="../../.gitbook/assets/image (2318).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**회사명 삭제**
{% endhint %}

1. 맞춤 로고 설정에서 '모바일용 회사명 텍스트에' 입력된 문자를 제거하고 등록 버튼을 클릭하세요.\
   회사 이름이 삭제되고 회사 이름이 초기 상태로 돌아갑니다.

<figure><img src="../../.gitbook/assets/image (2319).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**로고 이미지 삭제**
{% endhint %}

1. 맞춤 로고 설정에서 업로드한 이미지 파일 아래의 초기화 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2320).png" alt=""><figcaption></figcaption></figure>



2. 확인 화면에서 확인 버튼을 클릭합니다.\
   이미지 파일이 삭제되고 아이콘이 초기 상태로 돌아갑니다.
