# Web, PC Agent, 관리자 페이지의 로고 이미지를 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 커스텀 설정 메뉴의 커스텀 로고 설정에서 이미지 파일을 업로드하여 web 브라우저, PC Agent, 관리자 페이지의 로고를 변경하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정' 메뉴를 사용할 수 있습니다.
* 업로드 가능한 로고 이미지의 크기는 가로 130픽셀 × 세로 30픽셀의 JPG 파일 또는 24비트 PNG 파일입니다.
* 10MB 이상인 로고 파일은 등록할 수 없습니다.
* 이 매뉴얼의 절차로 커스텀 로고 설정을 삭제하는 경우, 모바일 애플리케이션과 모바일 버전 Web 브라우저의 설정한 모바일용 아이콘과 모바일용 회사명도 초기화되므로 주의해야 합니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**맞춤 설정 화면 표시**
{% endhint %}

1. '공유 설정' > '맞춤형 설정' 메뉴를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1982).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**로고 이미지 변경**
{% endhint %}

1. 로고 설정에서 'Web/PC Agent/관리자페이지용 로고'에서 '+ 아이콘'을 클릭하고 로고로 사용할 이미지 파일을 선택합니다.

<figure><img src="../../.gitbook/assets/image (323).png" alt=""><figcaption></figcaption></figure>

선택한 이미지 파일이 로고에 반영됩니다.

<figure><img src="../../.gitbook/assets/image (325).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
**NOTE**\
로고 이미지를 다시 선택하려면, 설정되어 있는 'Web/PC Agent/관리자페이지용 로고'를 클릭한 뒤 로고로 사용할 이미지 파일을 선택합니다.
{% endhint %}



2. '등록' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (326).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**변경된 로고 확인**
{% endhint %}

Web 브라우저, PC Agent, 관리자 페이지, 공유 링크의 비밀번호 입력 및 수신 화면에서 로고가 변경되었는지 확인합니다.

\

{% hint style="warning" %}
**로고 이미지 삭제**
{% endhint %}

1. 로고 설정에서 업로드한 이미지 파일 아래의 초기화 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (333).png" alt=""><figcaption></figcaption></figure>



2. 확인 화면에서 확인 버튼을 클릭합니다.\
   이미지 파일이 삭제되고 로고가 초기 상태로 돌아갑니다.
