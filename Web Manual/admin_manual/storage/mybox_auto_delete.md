# My Box의 파일을 자동으로 삭제하도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

My Box에 저장되어 있는 파일의 자동 삭제를 설정하는 방법에는, 사용자 페이지에서 '파일 기한' 기능을 이용하는 방법과, 관리 페이지에서 자동 삭제 설정을 하는 방법이 있습니다.\
이 매뉴얼에서는, 관리 페이지의 'Storage' 메뉴에서 파일의 자동 삭제를 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '용량 관리' 항목을 사용할 수 있습니다.
* 'My Box 파일 자동 삭제' 설정은, 기본적으로 '사용 안함'으로 설정되어 있습니다.
* 자동 삭제의 대상은 파일만 해당됩니다. 폴더는 자동 삭제할 수 없습니다.
* 자동 삭제가 실행되는 시간은 지정할 수 없습니다.
* 아래 작업으로 파일이 갱신된 다음 날을 기산일로 하여, 기산일을 포함해 지정한 기간이 경과한 뒤, 다음 날 1시 30분에 자동 삭제됩니다.
  * 업로드
  * 온라인 편집
  * 복사
  *   이동:

      파일을 이동하더라도 갱신 일시는 변경되지 않지만, 파일은 갱신된 것으로 판단됩니다.
  * 아래 절차에 따른 파일 암호화:\
    파일의 보안등급을 변경하는 방법\
    사용자 페이지에서 파일의 보안등급을 변경하는 방법
  * 아래 절차에 따른 파일 암호화 해제:\
    파일의 암호화를 해제하는 방법\
    사용자 페이지에서 파일의 암호화를 해제하는 방법
* 자동 삭제 설정을 하기 전에 저장된 파일의 자동 삭제는 다음과 같이 실행됩니다.
  * 저장일로부터 경과한 일수가 설정된 일수를 초과한 경우:\
    설정한 당일은 일수로 계산하지 않으며, 다음 날 오전 1시 30분에 자동 삭제가 실행됩니다.
  * 저장일로부터 경과한 일수가 설정된 일수를 초과하지 않은 경우:\
    설정한 당일 역시 일수로 계산하지 않으며, 남아 있는 일수가 모두 경과한 뒤의 다음 날 오전 1시 30분에 자동 삭제가 실행됩니다.
  *   예:\
      설정 일시가 2022년 1월 11일, 저장일이 2022년 1월 14일, 자동 삭제 설정이 3일 후인 경우

      | 기산일                                |                                                     |
      | ---------------------------------- | --------------------------------------------------- |
      | <p>2022년 1월 15일<br>(저장일의 다음 날)</p> | <p>2022년 1월 18일 1시 30분<br>(기산일을 포함해 3일 후의 다음 날)</p> |
* 저장 기간을 기준으로 실행에서 ○개월 후란, ○개월 후의 같은 날짜를 의미합니다.
  *   예: 설정 일시가 2022년 1월 11일, 저장일이 2022년 1월 14일, 자동 삭제 설정이 1개월 후인 경우

      | 기산일                                | 자동 삭제일                                             |
      | ---------------------------------- | -------------------------------------------------- |
      | <p>2022년 1월 15일<br>(저장일의 다음 날)</p> | <p>2022년 2월 16일 1시 30분<br>(1개월 후의 같은 날짜의 다음 날)</p> |
* 저장 기간을 기준으로 실행에서 ○개월 후에 기산일과 같은 날짜가 존재하지 않는 경우, 다음 달 1일에 삭제됩니다.
  *   예: 설정 일시가 2022년 1월 11일, 저장일이 2022년 1월 30일, 자동 삭제 설정이 1개월 후인 경우

      | 기산일                                | 자동 삭제일                                         |
      | ---------------------------------- | ---------------------------------------------- |
      | <p>2022년 1월 31일<br>(저장일의 다음 날)</p> | <p>2022년 3월 1일 1시 30분<br>(1개월 후의 말일의 다음 날)</p> |
* 자동 삭제가 완료되었을 때 전송되는 알림 메일의 수신자를 지정할 수는 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**자동 삭제 설정하기**
{% endhint %}

1. '용량관리' > '스토리지 현황'을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (291).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 ' My Box 파일 자동 삭제' 설정에서 다음을 설정합니다.

<figure><img src="../../.gitbook/assets/image (292).png" alt=""><figcaption></figcaption></figure>

| 항목           | 설명                                                                                                                                                                                        |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| My Box 자동 삭제 | My Box의 파일을 자동으로 삭제할지 여부를 선택합니다.                                                                                                                                                          |
| 삭제 설정        | <p>드롭다운 리스트에서 My Box의 파일이 업데이트된 후 자동으로 삭제되기까지의 기간을 선택합니다.<br>파일의 업데이트 날짜, 삭제의 기산일, 삭제되기까지의 경과 기간에 대해서는 제한·보충 사항을 참조해 주십시오.</p>                                                            |
| 자동 삭제 처리 방법  | <p>자동으로 삭제된 파일을 어떻게 처리할지 선택합니다.<br>관리자 또는 사용자의 휴지통으로 이동할지, 완전히 삭제할지를 선택할 수 있습니다.</p><p>휴지통을 사용하는 설정을 하고, 저장 기간을 설정하는 방법의 절차에서 사용자의 휴지통을 사용하지 않음으로 설정한 경우, 삭제 시 동작은 파일을 완전히 삭제로 고정됩니다.</p> |



2. 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (293).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**자동 삭제 완료 이메일 확인**
{% endhint %}

My Box의 파일이 자동으로 삭제되면 사용자에게 "자동삭제가 진행되었습니다." 라는메일이 전송됩니다.
