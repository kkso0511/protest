# 공유 링크 생성 화면을 이전 버전으로 변경하는 방법(미표시)

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 "링크 옵션 설정"에서 공유 링크 만들기 화면을 이전 버전으로 변경하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* 구버전의 링크 작성 화면에서는, 「링크 메일 송신」 「링크 메일 송신의 문서 미리보기」 「메일 옵션」은 대상외가 됩니다.
* 2020년 5월 19일 업데이트 이후 발행된 회사 ID에는 이 항목이 표시되지 않으며 모두 새 버전으로 표시됩니다.\
  구 버전을 원하시는 경우는, 「[도입 후의 문의](https://directcloud.jp/support/qna_after)」폼으로부터 신청해 주세요.

### 절차 <a href="#a05" id="a05"></a>

1. 보안 > 파일 전송 설정 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1640).png" alt=""><figcaption></figcaption></figure>

2. 링크 옵션 설정에서 링크 화면에서 이전 버전을 선택하고 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1641).png" alt=""><figcaption></figcaption></figure>
