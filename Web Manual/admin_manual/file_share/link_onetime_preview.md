# 공유 링크 생성 화면에 1회용 미리보기 옵션이 표시되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 ‘링크 생성’ 화면에 ‘1회용 미리보기 링크’ 옵션이 표시되도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* ‘1회용 미리보기 링크’ 옵션은 기본적으로 ‘사용함’으로 설정되어 있습니다.
* ‘링크 생성’ 화면에서 ‘일회성 열람 옵션’에 체크하면, 이미 저장되어 있는 설정값과 관계없이 링크 생성 옵션이 아래와 같이 고정됩니다. 이 설정값은 변경할 수 없습니다.
  * 수신 방법: 미리보기만 제공
  * 업로드: 사용 안함
  * 만료: 당일
  * 비밀번호: 사용(고정)
  * 접속 횟수 제한: 1회

***

### 절차 <a href="#a05" id="a05"></a>

**\[관리자 페이지에서의 설정]**

1. '보안 정책' > '반출 정책' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1656).png" alt=""><figcaption></figcaption></figure>



2. '링크생성 시 기본값 설정'의 '1회용 미리보기 링크'을 '사용함'으로 선택하고 '저장' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1695).png" alt=""><figcaption></figcaption></figure>



**\[사용자 페이지에서 설정]**

1. 사용자 페이지에서 '링크 생성' 화면을 표시하고 '1회용 미리보기 링크'가 표시되는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (1696).png" alt=""><figcaption></figcaption></figure>
