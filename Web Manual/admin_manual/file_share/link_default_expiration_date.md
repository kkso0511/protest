# 공유 링크 생성 화면에 표시되는 만료일의 기본값을 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 링크 옵션 설정에서 링크 작성 화면에 표시되는 만료 날짜의 기본값을 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* 링크 옵션 설정은 Guest, LDAP 연계 및 SAML 연계로 등록된 사용자에게도 적용되지만, Connect User에게는 적용되지 않습니다.
* 만료일의 기본값은 7(7일 후)로 고정됩니다.
* 사용자가 만료일의 기본값을 지정할 수는 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

**관리 페이지에서의 조작**

1. '보안 정책' > '반출 정책' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (106).png" alt=""><figcaption></figcaption></figure>



2. '링크 생성시 기본값 설정' 중 만료일에 '항상 사용'또는'사용'을 선택하고 만료 기본값에 만료 날짜 수를 입력합니다.

<figure><img src="../../.gitbook/assets/image (375).png" alt=""><figcaption></figcaption></figure>



3. 링크 옵션 설정에서 저장 버튼을 클릭합니다.<br>

**\[사용자 페이지에서의 조작]**

1. 링크 생성 화면에서 만료 날짜의 기본값이 반영되는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (2259).png" alt=""><figcaption></figcaption></figure>
