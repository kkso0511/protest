# 공유 링크 생성 화면에서 지정할 수 있는 접속 횟수 제한의 최대값을 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 ‘링크 옵션 설정’에서 링크 생성 화면에서 지정할 수 있는 접근 횟수 제한의 최대값을 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* ‘링크 옵션 설정’은 Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자에게도 적용되지만, Connect User에는 적용되지 않습니다.
* 관리자 측에서 설정할 수 있는 접근 횟수 제한의 최대값은 ‘99’(99회)입니다.

***

### 절차 <a href="#a05" id="a05"></a>

**\[관리자 페이지에서의 설정]**

1. '보안 정책' > '반출 정책' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (96).png" alt=""><figcaption></figcaption></figure>



2. 링크 옵션 설정의 '접속횟수제한'에서 '항상 사용' 또는 '사용'을 선택하고 접속 횟수의 최대값을 입력합니다.

<figure><img src="../../.gitbook/assets/image (373).png" alt=""><figcaption></figcaption></figure>



3. 링크 옵션 설정에서 저장 버튼을 클릭합니다.



**\[사용자 페이지에서 작업]**

1. 링크 생성 화면에서 최대 접속 횟수 제한을 반영하는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (95).png" alt=""><figcaption></figcaption></figure>
