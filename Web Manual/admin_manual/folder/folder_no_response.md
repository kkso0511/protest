# 폴더를 클릭해도 반응이 없는 경우 해결 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 폴더를 클릭해도 반응이 없는 경우의 대응 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '공유 설정' 메뉴를 사용할 수 있습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. Web 브라우저의 자동 번역 기능을 비활성화합니다.
2. Web 브라우저의 캐시를 모두 삭제합니다.
3. [관리자 페이지에 로그인하는 방법](https://help.directcloud.net/admin_manual/login/admin_login)의 절차에 따라 다시 관리 페이지에 로그인합니다.
