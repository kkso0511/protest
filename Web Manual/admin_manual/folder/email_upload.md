# 전용 이메일 주소를 통해 특정 폴더에 파일을 업로드하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud에서는 폴더마다 고유의 업로드 전용 이메일 주소를 설정하여, 이메일에 파일을 첨부해 파일을 업로드할 수 있습니다. 이 이메일 주소는 DirectCloud에서 자동으로 생성됩니다.\
이 매뉴얼에서는 폴더를 선택하여 파일 업로드 전용 이메일 주소를 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼에서는 Shared Box 메뉴에서의 설정 방법을 설명하고 있지만, Connect 관리, DLP 관리 메뉴에서도 동일한 절차로 설정할 수 있습니다.
* Warm Storage와 Cold Storage에서는 업로드 전용 이메일 주소 기능을 사용할 수 없기 때문에 Warm Storage 관리, Cold Storage 관리 메뉴에서 폴더를 선택해도 폴더 속성에 업로드 전용 이메일 주소가 표시되지 않습니다.
* 관리자 권한이 부여된 사용자는 관리자 페이지의 '공유 설정'을 사용할 수 있습니다.
* 업로드 전용 이메일 주소는 폴더 속성에서 6단계 폴더까지 설정할 수 있습니다.
* 업로드 전용 이메일 주소로 파일을 업로드할 때, 하나의 이메일에 첨부할 수 있는 파일 수는 최대 100개입니다.
*   하나의 이메일에 첨부할 수 있는 파일의 최대 크기 제한은 사용 중인 메일 서버의 설정에 따라 달라집니다.

    예를 들어 Gmail의 경우 첨부 파일이 25MB를 초과하면 파일이 Google 드라이브로 업로드되고, 이메일에는 Google 드라이브에 업로드된 파일의 링크 URL이 포함됩니다. 이 경우 DirectCloud는 첨부 파일을 직접 수신할 수 없기 때문에, 로그 > 업로드 전용 이메일 수신 이력 메뉴의 업로드 결과에는 실패(첨부 파일 누락)으로 기록됩니다.
* 업로드 전용 이메일 주소가 수신한 메일의 이력은 업로드 전용 이메일 주소가 수신한 메일 이력을 확인하는 방법에서 확인할 수 있습니다.
* 업로드 전용 이메일 주소를 사용해 파일을 업로드했을 때 발생한 오류는 사용자 페이지에서 확인할 수 없습니다. 반드시 업로드 전용 이메일 주소가 수신한 메일 이력을 확인하는 방법의 절차에 따라 관리자 페이지에서 확인해야 합니다.
* 업로드 전용 이메일 주소를 통해 파일이 업로드된 폴더에 동일한 이름의 파일이 이미 있는 경우, 동일 파일명 처리 방식 설정과 관계없이 파일 이름 끝에 숫자가 자동으로 추가됩니다.
* 업로드 전용 이메일 주소의 메시지 헤더는 연도 표기 방식(2자리, 4자리 모두)에 대응합니다.
* 수신 주소에 다음 문자가 포함되어 있는 경우, 첨부 파일 전송이 실패할 수 있습니다:\
  ()<>\[]:;@,.

***

### 절차 <a href="#a05" id="a05"></a>

1. '공유설정' > 'Shared Box 관리' 메뉴를 클릭하고 파일을 업로드할 전용 이메일 주소를 설정할 폴더를 선택합니다.

<figure><img src="../../.gitbook/assets/image (2309).png" alt=""><figcaption></figcaption></figure>



2. 화면을 아래로 스크롤하고 폴더 속성 설정에서 '업로드용 이메일' 설정에서 '수정' 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2310).png" alt=""><figcaption></figcaption></figure>



3. 업로드용 이메일에서 '사용'을 선택하고 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (2311).png" alt=""><figcaption></figcaption></figure>



4. 업로드 전용 이메일 주소가 생성되었는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (2312).png" alt=""><figcaption></figcaption></figure>



5. 업로드 전용 이메일 주소 오른쪽에 있는 복사 버튼을 클릭하면 클립보드에 이메일 주소가 복사됩니다.

<figure><img src="../../.gitbook/assets/image (2313).png" alt=""><figcaption></figcaption></figure>
