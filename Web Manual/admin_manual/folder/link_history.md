# 관리자 페이지의 파일 목록에서 링크 이력을 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Shared Box 메뉴에서 파일 목록을 표시하고 첫 번째 계층 이후 폴더에 저장된 폴더 또는 파일의 링크 이력을 확인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '공유 설정' 메뉴를 사용할 수 있습니다.&#x20;
* 1계층 폴더는 링크를 생성할 수 없기 때문에 링크 기록에는 0으로 표시됩니다.
* 폴더 목록과 파일 목록에 표시되는 폴더와 파일은 한 페이지에 50건씩 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '공유 설정' > 'Shared Box' 관리 메뉴를 선택하고 1계층 이후 폴더를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1855).png" alt=""><figcaption></figcaption></figure>



2. 항목 설정에서 파일목록 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1854).png" alt=""><figcaption></figcaption></figure>



3. 링크 이력을 확인하고자 하는 파일의 '링크 이력'에 표시된 숫자를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1856).png" alt=""><figcaption></figcaption></figure>

&#x20;     링크 이력 화면이 표시됩니다.

<figure><img src="../../.gitbook/assets/image (1857).png" alt=""><figcaption></figcaption></figure>
