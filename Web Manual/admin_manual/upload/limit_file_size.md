# 업로드 가능한 파일의 크기를 제한하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 '보안 정책' > '반입 정책' 메뉴의 업로드 제한 설정에서 업로드 가능한 파일을 최대 크기로 제한하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '보안 정책'에 접근 가능합니다.
* ‘파일 단위 용량 제한’은 Guest, LDAP 연동 및 SAML 연동으로 취득한 사용자에게도 적용됩니다.
* ‘파일 단위 용량 제한’은 My Box와 Shared Box뿐만 아니라 Connect 폴더, 문서 관리 폴더, DLP 폴더, Warm Storage, Cold Storage에도 적용됩니다.
* 본 매뉴얼 에서 설정한 용량 기준 업로드 제한은 일반 업로드 외에도 아래 방법으로 업로드하는 경우에도 적용됩니다.
  * 공유 링크의 업로드 기능을 사용하여 파일을 업로드
  * 업로드 요청에 파일을 업로드
  * 업로드 전용 메일을 통해 파일을 업로드
* ‘파일 단위 용량 제한’에서 입력할 수 있는 최대값은 각 플랜에서 업로드 가능한 파일 크기의 상한까지입니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. '보안 정책' > '반입 정책' 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/image (1633).png" alt=""><figcaption></figcaption></figure>



2. 업로드 제한 설정의 '1 File 용량 제한'에서 '사용함'을 선택하고 업로드할 수 있는 파일의 최대 크기를 메가바이트 단위로 입력합니다.

<figure><img src="../../.gitbook/assets/image (1634).png" alt=""><figcaption></figcaption></figure>



3. 저장 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1635).png" alt=""><figcaption></figcaption></figure>
