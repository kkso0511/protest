# \[Web, Drive] Home 메뉴에서 최근의 조작 이력을 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Web 브라우저의 Home 화면과 DirectCloud 드라이브의 설정 화면에서 최근 작업 이력을 확인하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 폴더에 대한 접근 권한이 삭제된 경우, Home 메뉴에 나열되어 있던 아래 항목의 파일 및 폴더 경로 정보는 사라집니다.\
  코멘트\
  즐겨찾기\
  최근 사용한 파일\
  공유된 파일\
  링크로 업로드된 파일
* 폴더의 접근 권한이 업로더로 변경된 경우, 해당 폴더에 저장된 파일은 Home 메뉴 항목에 표시되지만 미리보기는 할 수 없습니다.
* 관리 페이지의 Guest 기능 제한에서 Home이 비활성화로 설정된 경우, Guest에게는 아래 항목이 표시되지 않습니다.
  * Web 브라우저\
    Home 메뉴
  * DirectCloud 드라이브\
    My Box의 사용량(사용률) / 사용자 할당 용량\
    코멘트\
    최근 사용한 파일\
    즐겨찾기\
    공유된 파일\
    Link History\
    링크로 업로드된 파일
* Warm Storage를 사용할 수 있는경우 사용자 페이지에 Warm Storage 메뉴가 표시됩니다.
* Cold Storage를 계약한 경우, 사용자 페이지에 Cold Storage 메뉴가 표시됩니다.
* 자동 이동 내역은 Warm Storage 또는 Cold Storage 중 하나라도 이용할 수 있는 경우에 표시됩니다.
* Cold Storage로 이동된 파일은 Home 메뉴에 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 확인**
{% endhint %}

1. Web 브라우저에 로그인합니다.\
   Home 메뉴에 다음과 같은 최근 작업 내역이 표시됩니다.

<table><thead><tr><th width="169">항목</th><th>설명</th></tr></thead><tbody><tr><td>코멘트</td><td>추가된 파일 코멘트가 표시됩니다. 최대 20건 표시됩니다.</td></tr><tr><td>최근파일</td><td>최근에 사용한 파일이 표시됩니다. 최대 20건 표시됩니다.</td></tr><tr><td>즐겨찾기</td><td>즐겨찾기에 등록한 파일이 표시됩니다. 최대 20건 표시됩니다.</td></tr><tr><td>공유받은 파일</td><td>다른 사용자가 업로드한 파일이 표시됩니다. 최대 20건까지 표시됩니다. 업로더 접근 권한이 부여된 사용자에게는 표시되지 않습니다.</td></tr><tr><td>Link History</td><td>사용자가 직접 발행한 공유 링크의 이력이 표시됩니다. 최대 20건까지 표시됩니다.</td></tr><tr><td>링크로 받은 파일</td><td>공유 링크에 업로드 기능이 허용된 경우, 링크 수신 화면에서 업로드된 파일이 표시됩니다. 링크 수신 화면에 파일이 업로드될 때마다 썸네일이 하나씩 생성되며, 썸네일은 최대 20건까지 표시됩니다. 한 번에 여러 파일이 업로드된 경우, 썸네일 왼쪽 하단에 ‘+1’과 같이 업로드된 나머지 파일 개수가 표시됩니다. 숫자를 클릭하면 업로드된 파일 이름을 확인할 수 있습니다.</td></tr></tbody></table>



{% hint style="warning" %}
&#x20;**DirectCloud 드라이브에서 확인**
{% endhint %}

1. DirectCloud 드라이브에 로그인하고 작업 트레이에서 DirectCloud 드라이브 아이콘을 클릭합니다.\
   DirectCloud 드라이브 설정 화면이 표시됩니다.
2. 다음 탭을 클릭합니다.\
   각 최근의 작업 이력이 표시됩니다.
