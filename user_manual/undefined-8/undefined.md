# 사용자 페이지에서 업로드 요청의 이용 내역을 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지에서 업로드 요청을 이용한 내역을 확인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 본인 외의 사용자가 업로드 요청을 이용한 내역은 확인할 수 없습니다.
* 업로드 요청 내역 메뉴에서는 비활성화된 업로드 요청 링크를 활성화할 수 없습니다.
* 관리자가 업로드 요청의 내역을 삭제하면 업로드 요청 내역 메뉴에 표시되지 않습니다.
* 업로드 요청의 내역은 기본적으로 날짜와 시간의 내림차순으로 표시됩니다.\
  헤더의 수신인, 제목, 날짜와 시간을 클릭하여 정렬할 수 있습니다.\
  단, 폴더와 파일 목록과는 달리 마지막으로 정렬한 설정이 저장되지 않으므로 Web 브라우저를 다시 실행하면 기본 상태로 돌아갑니다.\
  정렬 방법은 폴더와 파일 목록을 정렬하는 절차와 동일합니다.\
  자세한 내용은 [\[Web\] 폴더 및 파일 목록을 정렬하는 방법](https://help.directcloud.net/user_manual/undefined-3/web-2)을 참조해주시기 바랍니다.
* 업로드 요청 내역 상단에 표시되는 헤더에서는 표시 너비를 변경할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 내역 확인**
{% endhint %}

1. 'File Request' 메뉴를 클릭합니다.\
   업로드 요청의 내역이 나열됩니다.

2. 상세 내용을 확인하려는 업로드 요청 내역의 오른쪽에 있는 버튼을 클릭합니다.

3. 수신 폴더의 수신 화면에 업로드된 파일이 있는 경우 대상 이메일 주소, 업로드된 파일 이름, 업로드된 시간 및 IP 주소 및 메시지가 표시됩니다. 받은 폴더로 이동을 클릭하면 업로드된 파일이 저장된 폴더로 이동합니다.
