# 차단된 프로그램을 허용 프로그램으로 등록하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud의 기본 기능인 랜섬웨어 방지 기능을 사용하면, 짧은 시간에 여러 번 파일을 변경하는 프로그램을 랜섬웨어로 판단하여 자동으로 차단할 수 있습니다.\
아래 매뉴얼의 절차에 따라 프로그램이 차단되면 '차단된 프로그램'에 자동으로 등록되며, 프로그램 차단 이력에 로그가 기록됩니다.

* [랜섬웨어 방지 기능을 통해 보호하려는 파일의 확장자를 지정하는 방법](https://help.directcloud.net/admin_manual/undefined-9/undefined-1)
* [랜섬웨어 방지 기능에서 프로그램이 차단되기까지의 변경 횟수를 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-9/undefined-2)

또한 DirectCloud 드라이브로 로그인한 상태에서 프로그램이 차단되면, "알려지지 않은 프로그램의 파일 변조 시도로 접근을 차단했습니다." 라는 메시지가 표시됩니다.\
이 매뉴얼에서는 메시지 화면에서 차단된 프로그램을 허용 프로그램으로 등록하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 기능을 사용하기 위해서는 [랜섬웨어 방지 기능을 사용하는 방법](https://help.directcloud.net/admin_manual/undefined-9/undefined)에서 랜섬웨어 방지가 활성화되어 있어야 합니다.
* 랜섬웨어 방지 기능은 모든 요금제에서 추가 비용 없이 이용할 수 있습니다.
* DirectCloud 드라이브의 사용자 페이지에서 프로그램이 차단된 이력을 확인하실 수 있습니다.\
  자세한 내용은 [DirectCloud 드라이브에서 프로그램이 차단된 내역을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-10/directcloud)을 참고해주시기 바랍니다.
* 랜섬웨어 대책으로 차단되는 대상은 DirectCloud 드라이브에 저장된 파일을 변경한 프로그램만 해당됩니다.
* 랜섬웨어 대책으로 차단되는 것은 프로그램에 의해 파일이 편집되었을 때에 한합니다.\
  파일을 열기만 한 경우에는 변경으로 판단되지 않습니다.
* 한 번 프로그램이 차단 프로그램으로 등록되면, 동일한 프로그램이 파일을 열기만 해도 변경으로 판단됩니다.
* 디지털 서명이 포함된 프로그램은 이 기능으로 차단할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

DirectCloud 드라이브에 로그인했을 때 랜섬웨어 대책에 의해 "알려지지 않은 프로그램의 파일 변조 시도로 접근을 차단했습니다." 라는 메시지가 표시된 경우, 아래 절차를 통해 해당 프로그램을 허가 프로그램으로 등록할 수 있습니다.

1. '프로그램 허용' 버튼을 클릭합니다.<br>
2.  프로그램을 허용할 조건을 설정한 뒤 확인 버튼을 클릭합니다.

    <table><thead><tr><th width="194">항목</th><th>설명</th></tr></thead><tbody><tr><td>해시 값 일치</td><td>해시값이 일치할 때 프로그램의 파일 접근을 허가합니다.<br>이 항목을 선택하면 프로그램의 해시값이 자동으로 설정됩니다.</td></tr><tr><td>파일 경로 일치</td><td>파일 경로가 일치할 때 프로그램의 파일 접근을 허가합니다.<br>이 항목을 선택한 경우, 텍스트 박스에 파일의 경로를 입력해야 합니다.</td></tr></tbody></table>

차단된 프로그램이 “허용된 프로그램”에 등록되고 “허용 프로그램에 추가되었습니다.”라는 확인 화면이 표시됩니다.<br>

3. 확인 버튼을 클릭합니다.<br>
4. [DirectCloud 드라이브에서 차단/허가된 프로그램을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-10/directcloud-1) 으로 "허가된 프로그램"을 확인합니다.
