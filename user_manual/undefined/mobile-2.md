# \[Mobile] 2단계 인증 기능을 사용하여 사용자 페이지에 로그인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 및 Android 애플리케이션에서 일회용 비밀번호를 사용하여 사용자 페이지에 로그인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 페이지의 '보안정책' > '접속 제어' 메뉴에서 2단계인증 탭의 2단계인증의 사용을 선택해야 합니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**2단계 인증 코드를 전송하기**
{% endhint %}

1. 로그인 화면에서 회사 ID, 사용자 ID, 비밀번호를 입력하고 로그인 버튼을 누릅니다.\
   2단계 인증 선택 화면이 표시됩니다.



2. 이메일로 인증을 누르면 사용자 정보에 등록된 이메일 주소로 안내 메일이 도착합니다.\
   2단계 인증 화면에서 이메일로 받은 2단계 인증 코드를 입력하고 확인 버튼을 누릅니다.



3. 2단계 인증 코드를 저장하려면 저장 버튼을 누릅니다.\
   2단계 인증 코드가 저장됩니다.
