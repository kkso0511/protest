# \[Mobile] 저장된 로그인 정보로 DirectCloud를 사용하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 회사코드, 아이디 및 비밀번호를 입력하지 않고 사용자 페이지에 로그인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 페이지의 '보안 정책' > ' 로그인 설정 ' 메뉴의 '자동 로그아웃' 설정에서 '자동 로그아웃'이 '사용됨'으로 설정된 경우 액세스 토큰이 만료된 '60일'보다 빨리 로그아웃됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션으로 로그인**
{% endhint %}

1. 로그인 화면에서 "로그인 상태 유지"의 토글 스위치를 켜고 "로그인" 버튼을 클릭합니다.\
   로그인에 성공하면 입력한 회사코드, 아이디, 비밀번호가 기억됩니다.



2. 두 번째 이후에 로그인 화면에 액세스하면 회사코드, 아이디, 비밀번호를 입력하지 않고 로그인할 수 있습니다.



{% hint style="warning" %}
**Andorid 애플리케이션으로 로그인**
{% endhint %}

1. 로그인 화면에서 로그인 상태 유지를 선택하고 로그인 버튼을 클릭합니다.\
   로그인에 성공하면 입력한 회사코드, 아이디, 비밀번호가 기억됩니다.



2. 두 번째 이후에 로그인 화면에 액세스하면 회사코드, 아이디, 비밀번호를 입력하지 않고 로그인할 수 있습니다.
