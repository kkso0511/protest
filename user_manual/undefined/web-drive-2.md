# \[Web, Drive] 2단계 인증 기능을 사용하여 사용자 페이지에 로그인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 2단계 인증을 이용하여 사용자 페이지에 로그인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 페이지의 보안정책 > 접속 제어 메뉴에서 2단계인증 탭의 2단계인증의 사용을 선택해야 합니다.
* 관리 페이지에서 2단계 인증의 유효 기간 설정에서 입력한 기간 유효가 선택되어 있는 경우에는 일정 기간 동안 2단계 인증 코드 입력을 생략할 수 있습니다. 설정된 기간이 지나면 애플리케이션을 실행했을 때 자동으로 로그아웃됩니다. 사용자는 2단계 인증을 통해 다시 로그인해야 합니다.
* 2단계 인증 코드 입력을 생략할 수 있는 기간에 대한 자세한 내용은 관리자에게 문의해 주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저**
{% endhint %}

**\[2단계 인증 코드를 전송하기]**

1. 로그인 화면에서 회사코드, 아이디, 비밀번호를 입력하고 로그인 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1228).png" alt=""><figcaption></figcaption></figure>

2. 2단계 인증 화면이 표시되며 등록된 이메일로 전송된  인증코드를 입력하는 화면이 표시됩니다. 인증코드를 입력하고 확인 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1227).png" alt=""><figcaption></figcaption></figure>

3. 현재 사용중인 브라우저를 신뢰하는 경우일 '신뢰함' 버튼을  클릭합니다.

<figure><img src="../../.gitbook/assets/image (1229).png" alt=""><figcaption></figcaption></figure>

일회용 비밀번호가 저장됩니다.\
일회용 비밀번호 입력을 생략할 수 있는 기간에 대한 자세한 내용은 사용자가 일회용 비밀번호를 사용하여 로그인을 강제하는 방법 의 "이중 인증 유효 기간" 설정 설명을 참조하십시오.



**\[이메일로 인증하는 경우]**

"이메일로 인증"을 클릭하면 사용자 정보에 등록된 이메일 주소로 알림 이메일을 받게 됩니다.

<figure><img src="../../.gitbook/assets/image (1230).png" alt=""><figcaption></figcaption></figure>



{% hint style="info" %}
**DirectCloud 드라이브(Windows)**
{% endhint %}

**2단계 인증 코드를 전송하기**

1. 로그인 화면에서 회사코드, 아이디, 비밀번호를 입력하고 로그인 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1232).png" alt=""><figcaption></figcaption></figure>

사용자 정보에 등록된 이메일 주소로 알림 이메일을 받게 됩니다.

<figure><img src="../../.gitbook/assets/image (1234).png" alt=""><figcaption></figcaption></figure>

이메일로 전송된 인증코도를 입력하여 로그인 합니다.&#x20;

<figure><img src="../../.gitbook/assets/image (1233).png" alt=""><figcaption></figcaption></figure>

관리자 페이지의 보안정책 > 접속 제어 메뉴의 2단계 인증 탭에서 2단계 인증 유효 기간 설정에서 입력한 기간 사용을 선택한 경우 '이 프로그램을 신뢰 하시겠습니까? '라는 확인 화면이 나타납니다.



2. DirectCloud 드라이브 프로그램을 신뢰하는 프로그램으로 저장하여, 5일간 현재 기기에서 2단계 인증 절차를 생략할 수 있습니다. \
   원하시는 경우 '신뢰함' 버튼을 클릭합니다.
