# 사용자 페이지의 표시 언어를 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지의 설정 화면에서 사용자 페이지의 표시 언어를 변경하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼의 절차에 따라 설정한 언어는 사용자 페이지의 표시 언어뿐만 아니라, 사용자에게 발송되는 알림 이메일에도 반영됩니다.
* 사용자 페이지의 표시 언어를 변경하더라도, 아래 항목은 변경되지 않습니다.
  * 사용자명
  * 폴더명
  * 파일명
* 표시 언어를 한국어로 변경한 경우, ‘마이박스’, ‘공유’, ‘Connect’, ‘링크의 이력’, ‘수신 폴더 이력’, ‘첨부 파일 이력’, ‘휴지통’은 아래와 같이 표시됩니다.
  * 내폴더: My Box
  * 공유폴더: Shared Box
  * Connect: Connect
  * 링크의 이력: Link History
  * 파일 업로드 요청 이력: File Request
  * 파일첨부 메일발송 이력: Mail
  * 휴지통: Trash
* 사용자 페이지의 표시 언어를 변경하더라도, 알림 이메일 본문에 기재된 아래 정보는 변경되지 않습니다.
  * 사용자명
  * 회사명
  * 폴더 경로
  * 파일명
  * 코멘트 내용
* 아래 알림 이메일을 수신할지 여부를 설정하려면, [사용자 페이지에서 알림 메일의 수신 여부를 설정하는 방법](https://help.directcloud.net/user_manual/undefined-13/undefined-3)을 참고하시기 바랍니다.
  * 사용자 로그인
  * 파일 업로드
  * 파일 코멘트 추가
  * 폴더 속성 자동 삭제

***
