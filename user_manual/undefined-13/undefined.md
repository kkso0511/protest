# 사용자 페이지의 폴더 메뉴 유형을 변경하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 설정 화면에서 사용자 페이지의 폴더 메뉴 유형을 변경하는 방법에 대해 설명합니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. 사용자 페이지에 로그인하여 다음 방법 중 하나로 설정 화면을 표시합니다.\
   &#x20;❶ 사용자 이름 클릭\
   &#x20;❷ 설정 버튼 클릭



2. 설정 화면의 목록에서 '메뉴 타입'을 목록, 트리 중에서 선택합니다.

<table><thead><tr><th width="158">항목</th><th>설명</th></tr></thead><tbody><tr><td>목록형식                     </td><td>"My Box" "Shared Box" 메뉴 이름만 표시됩니다.<br>바로 아래 폴더는 표시되지 않습니다.</td></tr><tr><td>트리형식</td><td>"My Box" "Shared Box" 바로 아래에 만들어진 폴더가 트리 형식으로 표시됩니다.</td></tr></tbody></table>

3. 확인 버튼을 클릭합니다.<br>
4. 설정이 업데이트되고 폴더 메뉴 유형이 변경됩니다.<br>
5. 사용자 페이지가 선택한 폴더 메뉴 유형으로 표시되는지 확인합니다.
