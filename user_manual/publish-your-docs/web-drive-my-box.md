# \[Web, Drive] My Box의 사용량을 확인하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지에서 My Box의 사용량을 확인하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 사용자에게 용량을 할당하는 방법에 대해서는 [사용자에게 할당된 My Box 용량을 변경하는 방법](https://help.directcloud.net/admin_manual/undefined-8/my-box-3)을 참고해 주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 확인**
{% endhint %}

1. Web 브라우저에 로그인합니다.
2. 사용자 페이지에 표시된 사용자 이름 아래에 My Box의 할당량과 사용량이 표시됩니다.



{% hint style="warning" %}
&#x20;**DirectCloud 드라이브에서 확인**
{% endhint %}

1. DirectCloud 드라이브에 로그인한 후, 작업 트레이에서 DirectCloud 드라이브 아이콘을 클릭합니다.\
   설정 화면이 표시됩니다.
2. 설정 화면에 표시된 사용자 이름 아래에 My Box의 할당량과 사용량이 표시됩니다.
