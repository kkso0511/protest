# \[Mobile] DirectCloud가 지원하는 Microsoft Excel의 사용자 정의

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 및 Android 애플리케이션이 지원하는 Microsoft Excel의 사용자 정의에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* <업로더> 접근 권한이 부여된 사용자를 제외한 사용자는 파일 미리보기 기능을 이용할 수 있습니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/web)을 참고해 주시기 바랍니다.

***
