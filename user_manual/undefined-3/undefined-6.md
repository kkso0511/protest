# 온라인 편집 화면에서 코멘트 기능을 이용하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 온라인 편집 화면에서 코멘트 기능을 사용하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 온라인 편집을 사용하려면, [파일을 다운로드하지 않고 온라인으로 편집할 수 있도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-6/undefined-3)에 따라 온라인 편집 기능이 활성화되어 있어야 합니다.
* <마스터>, <전체권한>, <편집자>, <편집자-> 접근 권한이 부여된 사용자가 온라인 편집을 이용할 수 있습니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/web), [접근 권한의 종류와 DirectCloud 드라이브 에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/directcloud)을 참고해 주시기 바랍니다.
* 온라인 편집에서는 100MB 이하의 파일을 최대 5명이 동시에 편집할 수 있습니다.
*   Microsoft Office가 설치되어 있지 않은 PC에서도 Office 파일을 편집할 수 있습니다.

    온라인 편집에 대응하는 확장자는 다음과 같습니다.\
    .xls, .xlsx, .doc, .docx, .ppt, .pptx
* 확장자가 .doc, .ppt, .xls인 구형 형식의 Office 파일을 온라인 편집하려면, 편집 화면을 표시하기 위해 새로운 형식(.docx, .pptx, .xlsx)으로 변환한다는 확인 대화상자에서 확인 버튼을 클릭해야 합니다.
* 텍스트 파일과 매크로 파일은 온라인 편집을 사용할 수 없습니다.
* <편집자-> 이상의 접근 권한이 부여되어 있으면 다른 사용자가 작성한 코멘트에 대한 답글을 입력할 수 있습니다.
* 파일을 다운로드하면 온라인 편집에서 추가한 코멘트도 함께 보존됩니다.
* DirectCloud-SHIELD로 암호화된 파일도, 온라인 편집에 대응하는 확장자라면 말풍선 코멘트를 삽입할 수 있습니다.
* Office 파일에 코멘트를 추가한 경우 표시되는 사용자 이름은 다음과 같습니다.
  * 온라인 편집에서 코멘트를 추가한 경우: DirectCloud 사용자명
  * Microsoft Office에서 코멘트를 추가한 경우: Microsoft Office 계정명

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 코멘트 기능 사용**
{% endhint %}

&#x20;**\[코멘트 추가]**

1. [파일을 다운로드하지 않고 온라인으로 편집할 수 있도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-6/undefined-3)의 'Web 브라우저에서 온라인 편집 화면을 여는 절차'에 따라 온라인 편집 화면을 표시합니다.
2. 코멘트를 추가하려는 범위 또는 페이지를 선택한 후 오른쪽 클릭하여 코멘트 추가를 선택합니다.
3. 코멘트를 입력한 뒤 추가를 클릭합니다.\
   코멘트가 추가됩니다.<br>

**\[코멘트에 답장]**

1. Microsoft Office 제품인 경우 온라인 편집에 추가한 코멘트에 회신할 수 있습니다.\
   코멘트이 반영된 셀을 클릭한 후 답장을 클릭합니다.
2. 코멘트을 입력하고 제출 표시를 클릭합니다.\
   코멘트가 반영됩니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 코멘트 기능 사용**
{% endhint %}

**\[코멘트 추가]**

1. [파일을 다운로드하지 않고 온라인으로 편집할 수 있도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-6/undefined-3)의 'DirectCloud 드라이브에서 온라인 편집 화면 열기'에 설명된 절차에 따라 온라인 편집 화면을 표시합니다.
2. 코멘트을 추가할 범위 또는 페이지를 선택하고 마우스 오른쪽 버튼을 클릭 > 코멘트 추가를 선택합니다.
3. 코멘트를 입력한 뒤 추가를 클릭합니다.\
   코멘트가 추가됩니다.<br>

**\[코멘트에 답장]**

1. Microsoft Office 제품인 경우 온라인 편집에 추가한 코멘트에 회신할 수 있습니다.\
   코멘트이 반영된 셀을 클릭한 후 답장을 클릭합니다.
2. 코멘트을 입력하고 제출 표시를 클릭합니다.\
   코멘트가 반영됩니다.
