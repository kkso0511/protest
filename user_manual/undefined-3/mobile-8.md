# \[Mobile] 폴더 및 파일을 즐겨 찾기로 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 DirectCloud에 저장된 폴더와 파일을 즐겨찾기로 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일을 즐겨찾기로 등록하려는 사용자는 해당 파일에 대해 <업로더>를 제외한 접근 권한이 부여되어 있어야 합니다.
* 모든 사용자는 폴더를 즐겨찾기로 설정할 수 있습니다.
* Shared Box, My Box, Connect, DLP 메뉴 아래의 폴더와 파일을 즐겨찾기에 등록할 수 있습니다.
* DLP 메뉴는 DirectCloud의 옵션인 DirectCloud-SHIELD DLP를 계약한 경우에만 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**즐겨찾기 등록**
{% endhint %}

1. 즐겨찾기로 설정하려는 폴더 또는 파일이 저장된 폴더를 탭합니다.
2. 즐겨찾기로 설정하려는 폴더 또는 파일의 오른쪽 가장자리에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000NWw\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 즐겨찾기를 탭합니다.\
   즐겨찾기에 등록된 폴더나 파일의 시작 부분에는 노란색 ☆ 마크가 표시됩니다.



{% hint style="warning" %}
**즐겨찾기 참조**
{% endhint %}

1. 왼쪽 상단의 3선 아이콘을 탭한 다음 즐겨찾기 버튼을 탭합니다.\
   즐겨찾기에 등록된 폴더 및 파일 목록이 표시됩니다.
2. 즐겨찾기 목록에서 파일을 클릭하면 파일을 미리 볼 수 있습니다.\
   미리보기 화면의 왼쪽 상단에 있는 즐겨찾기 버튼을 탭하면 즐겨찾기 목록으로 돌아갑니다.
3. 즐겨찾기 목록에서 폴더를 클릭하면 즐겨찾기로 등록된 폴더로 이동합니다.



{% hint style="warning" %}
**즐겨찾기 해제**
{% endhint %}

1. 즐겨찾기 목록에서 즐겨찾기를 취소하려는 폴더 또는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000NWw\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
2. 표시되는 메뉴에서 즐겨찾기 해제를 탭합니다.\
   즐겨찾기 메뉴 목록에서 삭제한 폴더나 파일의 표시가 사라집니다.\
   또, 해제한 폴더나 파일의 선두로부터, 노란색☆마크가 사라집니다.
