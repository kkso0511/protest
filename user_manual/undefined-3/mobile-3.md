# \[Mobile] 파일을 업로드하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션을 사용하여 DirectCloud에 파일을 업로드하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일을 업로드하는 사용자에게는 <마스터>, <전체 권한>, <편집자>, <편집자->, <프리뷰어＋>, <업로더>의 접근 권한이 부여 돼야 합니다.&#x20;
* 사용자 API '파일 업로드'의 용량 상한은 '5GB'입니다.
* '업로드 파일 크기 제한'을 사용하도록 설정하면 업로드할 파일의 용량 한도가 설정된 숫자로 제한됩니다.
* 폴더에 저장할 수 있는 파일의 총 용량이 제한되어 있는 경우 업로드 시 폴더 내 파일의 총 용량이 제한 용량을 초과하면 “할당 용량을 넘고 있기 때문에 파일을 추가할 수 없습니다.”라는 오류 메시지가 표시되며, 초과 이후의 파일은 업로드 되지 않습니다.
* 업로드하는 파일 이름에 이전 글꼴의 문자가 포함되어 있으면 일부 문자가 자동으로 새 글꼴로 변환됩니다.
* 승인 워크플로우가 활성화 된 경우, ‘결재선 미지정사용자자’ 설정에서 ‘기능을 제한함’이 선택된 경우, 워크플로 신청자로 설정된 사용자가 아닌 경우에는 파일을 업로드할 수 없습니다.
* DirectCloud AI가 활성화된 폴더에 파일을 업로드하는 경우, 벡터 데이터가 생성됩니다. 벡터 데이터 생성 시에는 파일 내 텍스트 용량 등에 따라 요금이 발생할 수 있으므로 주의가 필요합니다.
* 업로드 대상 폴더에 동일한 이름의 파일이 이미 저장되어 있는 경우, 관리자 페이지의 '반입 정책' 설정에 따라 저장됩니다.\
  단, 아래의 방법으로 파일을 업로드한 경우에는 파일 이름 끝에 자동으로 숫자가 추가됩니다.
  * 공유 링크의 업로드 기능을 이용해 파일 업로드
  * 수신 폴더에 파일 업로드
  * 업로드 전용 메일을 통해 파일 업로드
* DirectCloud AI가 활성화된 폴더에 동일한 이름의 파일을 업로드하는 경우의 동작은 다음과 같습니다.
  * iOS 애플리케이션 또는 Android 애플리케이션에서 파일을 업로드하는 경우:\
    파일 이름의 끝에 자동으로 ‘-복사본’이 추가되어 별도의 파일로 업로드 됩니다.
  * 공유 링크 업로드 기능, 수신 폴더, 업로드 전용 메일 주소를 이용해 업로드하는 경우:\
    자동으로 파일 이름 끝에 숫자가 추가되어 별도의 파일로 업로드 됩니다.
* 업로드 전용 이메일 주소로 파일을 업로드했을 때의 에러는 사용자 페이지에서 확인할 수 없습니다.
* 파일을 덮어쓰기 저장하면 파일 버전이 업데이트됩니다.
* iOS 앱에서 '카메라' 앱이 실행되지 않는 경우 다음 단계를 시도해 보세요.\
  1\. iOS의 '설정'을 탭\
  2\. 'DirectCloud'를 탭\
  3\. '카메라'의 토글 스위치를 ON
* iOS 애플리케이션에서 '라이브러리에서 선택'이 표시되지 않는 경우 다음 단계를 시도해 보세요.\
  1\. iOS의 '설정'을 탭\
  2\. 'DirectCloud' > '사진'을 탭\
  3\. '모든 사진'을 탭
* Android 애플리케이션을 사용하는 경우 '라이브러리에서 선택'에서 이미지나 동영상 등의 미디어 파일을 업로드하면 업로드된 파일 이름이 Android 사양에 따른 숫자로 변경됩니다. 여러 파일을 선택하면 파일 이름이 숫자로 변환됩니다.\
  이를 방지하려면 '파일 목록에서 선택'으로 업로드하는 것이 좋습니다.
* iOS 전용 이미지 확장자인 heic, heif 파일을 업로드시, JPEG 파일로 변환하여 업로드합니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 파일 업로드**
{% endhint %}

**\['업로드' 화면 표시]**

1. 파일을 업로드할 폴더를 탭합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001OgD\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)' 버튼을 탭한 다음 표시된 메뉴에서 '업로드'를 탭합니다.\
   업로드 화면이 표시됩니다.



**\[사진이나 동영상을 촬영하여 업로드]**

{% hint style="info" %}
**NOTE**\
Apple 사의 이미지 파일인 heic, heif 파일을 iOS(iOS App, Safari)에서 업로드할 경우, JPEG 파일로 변환되어 업로드됩니다.
{% endhint %}

1. 업로드 화면에서 사진 또는 동영상 찍기를 탭합니다.\
   "카메라" 응용 프로그램이 시작됩니다.
2. 사진을 찍은 경우 오른쪽 하단에 있는 '사진 사용'을 탭합니다.\
   동영상을 촬영한 경우 오른쪽 하단의 '동영상 사용'을 탭합니다.\
   파일이 업로드됩니다.



**\[라이브러리에서 선택하여 업로드]**

1. 업로드 화면에서 라이브러리에서 선택을 탭합니다.\
   라이브러리에서 선택 화면이 표시됩니다.
2. 라이브러리에서 파일을 선택하고 오른쪽 상단의 마침을 탭합니다.\
   파일이 업로드됩니다.



**\[음성 녹음 및 업로드]**

1. 업로드 화면에서 녹음을 탭합니다.\
   녹음 화면이 표시됩니다.
2. 마이크 버튼을 탭합니다.\
   녹음이 시작됩니다.
3. 정지 아이콘을 탭합니다.\
   녹음이 중지됩니다.
4. 오른쪽 상단의 '업로드'를 탭합니다.\
   텍스트 상자에 파일 이름을 입력하고 확인을 탭합니다.\
   파일이 업로드됩니다.



{% hint style="warning" %}
**Android 애플리케이션에서 파일 업로드**
{% endhint %}

**\['업로드' 화면 표시]**

1. 파일을 업로드할 폴더를 탭합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000001OgD\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)' 버튼을 탭한 다음 표시된 메뉴에서 '업로드'를 탭합니다.\
   업로드 화면이 표시됩니다.



**\[사진을 찍고 업로드]**

1. 업로드 화면에서 사진 찍기를 탭합니다.\
   "카메라" 응용 프로그램이 시작됩니다.
2. 오른쪽 하단의 사진 사용을 탭합니다.\
   파일이 업로드됩니다.



**\[동영상을 촬영하고 업로드]**

1. 업로드 화면에서 동영상 촬영을 탭합니다.\
   "카메라" 응용 프로그램이 시작됩니다.
2. 오른쪽 하단의 '동영상 사용'을 탭합니다.\
   파일이 업로드됩니다.



**\[라이브러리에서 선택하여 업로드]**

1. 업로드 화면에서 라이브러리에서 선택을 탭합니다.\
   라이브러리에서 선택 화면이 표시됩니다.
2. 라이브러리에서 파일을 선택하고 오른쪽 상단의 마침을 탭합니다.\
   파일이 업로드됩니다.



**\[음성 녹음 및 업로드]**

1. 업로드 화면에서 음성 녹음을 탭합니다.\
   녹음 화면이 표시됩니다.
2. 마이크 버튼을 탭합니다.\
   녹음이 시작됩니다.
3. 정지 아이콘을 탭합니다.\
   녹음이 중지됩니다.
4. 오른쪽 상단의 '업로드'를 탭합니다.\
   텍스트 상자에 파일 이름을 입력하고 예를 탭합니다.\
   파일이 업로드됩니다.



**\[파일 목록에서 선택하여 업로드]**

1. 업로드 화면에서 파일 목록에서 선택을 탭합니다.\
   파일 목록에서 선택 화면이 표시됩니다.
2. 라이브러리에서 파일을 선택하고 오른쪽 상단의 마침을 탭합니다.\
   파일이 업로드됩니다.
