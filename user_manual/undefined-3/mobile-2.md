# \[Mobile] 파일을 다운로드하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 DirectCloud에 저장된 파일을 다운로드하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 파일을 다운로드하는 사용자에게는 <마스터>, <전체권한>, <편집자>, <다운로더>에 대한 접근권한이 부여되어야 합니다.
* iOS 또는 Android 애플리케이션에서는 여러 파일을 다운로드할 수 없습니다.
*   모바일 애플리케이션의 '이미지 저장' 메뉴에서 다음 확장명 이외의 파일을 다운로드할 수 없습니다.

    <table data-full-width="false"><thead><tr><th>파일형식</th><th>확장자</th></tr></thead><tbody><tr><td>JPEG</td><td>jpg, jpeg</td></tr><tr><td>PNG</td><td>png</td></tr></tbody></table>
* 모바일 앱의 '다음 방법으로 열기' 메뉴에서 파일을 다운로드하는 경우 다음 응용 프로그램을 각각 설치해야 합니다.
  * iOS 애플리케이션 : '파일' 애플리케이션
  * Android 애플리케이션: 'File Commander' 애플리케이션
* DirectCloud에 저장된 ZIP 파일을 다운로드할 때 용량 한도는 없습니다.
* 다운로드가 완료되면 알림 이메일이 전송되지 않습니다.
* 워크플로우가 활성화 된 경우, ‘결재선 미지정사용자’ 설정에서 ‘기능을 제한함’이 선택된 경우, 워크플로우 신청자로 설정된 사용자가 아닌 경우에는 파일을 다운로드할 수 없습니다.
* 워크플로우가 활성화 되어 있으며 ‘워크플로우 기본 설정’ 화면의 ‘다운로드’에 체크가 되어 있는 경우, ‘보안정책’ > ‘모바일 보안’ 메뉴의 ‘메일 전송’, ‘다른 앱에서 열기’ 설정이 ‘사용하지 않음’으로 고정되며, 모바일 애플리케이션에 ‘내보내기’와 ‘이미지 저장’이 표시되지 않습니다.
* 이 매뉴얼의 절차를 따라 파일을 다운로드하려면, 관리자 페이지의 '보안정책' > '모바일 보안' 메뉴에서 다음과 같이 설정되어 있어야 합니다.
  * 이미지 파일을 다운로드하는 경우: ‘이미지 저장’이 ‘사용함’으로 설정되어 있어야 합니다.
  * 이미지 이외의 파일을 다운로드하는 경우: ‘다음 방법으로 열기’가 ‘사용함’으로 설정되어 있어야 합니다.
* 관리자 페이지에서 다운로드 워크플로우가 활성화되어 있는 경우, 모바일 애플리케이션에서 ‘⁝’ 버튼을 탭했을 때 ‘이미지 저장’ 및 ‘내보내기’ 메뉴는 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 파일 다운로드**
{% endhint %}

**\[이미지 파일 다운로드]**

1. 다운로드하려는 파일이 저장된 폴더를 누릅니다.
2. 다운로드하려는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000MyL\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 누릅니다.
3. 표시되는 메뉴에서 '이미지로 저장'을 탭합니다.\
   선택한 이미지가 저장됩니다.



**\[이미지 파일 이외의 다운로드]**

1. 다운로드하려는 파일이 저장된 폴더를 누릅니다.
2. 다운로드하려는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000MyL\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 누릅니다.
3. 표시되는 메뉴에서 내보내기를 누릅니다.
4. '다음 방법으로 열기'를 누릅니다.
5. ""파일"에 저장"을 누른 후 저장 위치를 ​​지정하여 저장합니다.

**\[이미지 파일 이외의 다운로드]**

1. 다운로드하려는 파일이 저장된 폴더를 누릅니다.
2. 다운로드하려는 파일의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000MyL\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 누릅니다.
3. 표시되는 메뉴에서 내보내기를 누릅니다.

5. File Commander로 공유에서 한 번만 누르고 저장 위치를 ​​지정하고 저장합니다.
