# \[Web] 폴더 및 파일 목록을 정렬하는 방법

### 개요 <a href="#a03" id="a03"></a>

사용자 페이지에서는 My Box, Shared Box, 문서관리, Connect, DLP, Warm Storage, 휴지통, 즐겨찾기, 최근 이용한 파일 메뉴와 검색 결과에 표시된 폴더 및 파일 목록을 이름, 크기, 업데이트 날짜 또는 등록 날짜, 종류 기준으로 오름차순 또는 내림차순으로 정렬해 표시할 수 있습니다.\
이 매뉴얼에서는 사용자 페이지에서 폴더와 파일 목록을 표시하고 정렬하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 목록에 폴더와 파일이 함께 표시되는 경우에는 폴더가 먼저, 파일이 다음 순서로 표시됩니다.
* Web 브라우저의 기본 설정에서는 폴더와 파일 목록이 아래 기준으로 표시됩니다.
  * My Box, Shared Box, Connect 폴더, DLP 폴더, Warm Storage, 즐겨찾기, 휴지통 메뉴:\
    이름 기준 오름차순
  * 최근 사용한 파일 메뉴, 검색 결과:\
    날짜 기준 내림차순
* 검색 결과의 전체, 본문 탭에서는 폴더와 파일 목록을 정렬할 수 없습니다.
* 아래의 메뉴 및 검색 결과에서는 표시되는 정보를 정렬할 수 없습니다.
  * Link History
*   정렬의 기본 항목인 이름, 크기, 업데이트 날짜 또는 등록 날짜, 종류 기준으로 정렬하면 해당 정렬 설정은 브라우저 단위로 저장됩니다. 정렬 설정은 모든 메뉴와 폴더에서 공통으로 적용됩니다.\
    또한 동일한 브라우저에서 다른 사용자 계정으로 사용자 페이지에 로그인한 경우에도 정렬 설정이 그대로 유지됩니다.

    예를 들어 Shared Box에서 정렬을 설정하면 My Box와 문서관리 폴더에서도 동일한 정렬 방식이 적용됩니다.
* My Box, Shared Box, 문서관리 폴더, Connect 폴더, DLP 폴더, Warm Storage, 즐겨찾기, 휴지통 메뉴에서는 마지막으로 적용한 정렬 설정이 저장됩니다.\
  반면 최근 사용한 파일 메뉴 및 검색 결과에서는 마지막 정렬 설정이 저장되지 않습니다.
* 목록 상단에 표시되는 헤더에서 항목명 사이 경계에 마우스를 올렸을 때 표시되는 아이콘을 좌우로 드래그하면 각 항목의 표시 폭을 변경할 수 있습니다.
*   My Box, Shared Box, Connect, DLP, Warm Storage의 1단계 폴더에서는 변경한 항목 표시 폭이 저장됩니다.\
    이 설정은 정렬 설정과 마찬가지로 브라우저 단위로 저장됩니다.

    Shared Box, Connect, DLP, Warm Storage 메뉴를 클릭했을 때 표시되는 폴더 목록에서도 항목 폭을 변경할 수 있지만, 해당 설정은 저장되지 않습니다.

    최근 사용한 파일, 즐겨찾기, 휴지통 메뉴, 검색 결과, 문서 속성 검색 결과에서도 항목 폭을 변경할 수 있으나 설정은 저장되지 않습니다.
* 정렬 설정 및 항목 표시 폭 설정은 브라우저 캐시를 삭제하면 기본 표시 상태로 되돌아갑니다.
* DirectCloud 드라이브의 폴더 및 파일 목록 정렬은 Windows 탐색기의 기준을 따릅니다.\
  단, 휴지통은 DirectCloud에서 관리되는 기능이므로 이 매뉴얼과 동일한 방법으로 정렬할 수 있습니다.\
  휴지통 조작 방법은 Web 앱·드라이브 사용자 휴지통의 폴더 및 파일을 확인·조작하는 방법을 참조해주시기 바랍니다.
* DirectCloud-CONNECT를 이용할 수 있는 경우, 사용자 페이지에 Connect 메뉴가 표시됩니다.
* DirectCloud-SHIELD DLP를 계약한 경우, 사용자 페이지에 DLP 메뉴가 표시됩니다.
* Warm Storage를 사용할 수 있는경우 사용자 페이지에 Warm Storage 메뉴가 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**폴더와 파일 목록 정렬**
{% endhint %}

이 매뉴얼에서는 다음과 같은 일반적인 작동 절차를 설명합니다.

* My Box
* Shared Box
* 휴지통 메뉴
* 즐겨찾기 메뉴
* 최근 사용한 파일 메뉴
* 검색 결과의 파일 이름, 코멘트, 태그, 문서 속성 탭

폴더 및 파일 목록 위에 표시되는 헤더 항목은 선택한 메뉴와 폴더에 따라 다릅니다. 또한 정렬할 항목에 따라 표시되는 순서와 같은 규칙이 달라집니다. 자세한 내용은 하단에 있는 '정렬할 항목'을 참조하십시오.\
아래에서는 Shared Box 화면을 예로 설명합니다.

1. 폴더와 파일을 정렬하고 표시할 메뉴와 폴더를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1366).png" alt=""><figcaption></figcaption></figure>

&#x20;      폴더와 파일 목록이 표시됩니다.



2. 목록 상단에 표시되는 헤더에서 정렬할 항목을 클릭합니다.\
   정렬 대상 항목은 '이름', '크기', '날짜', '확장자'입니다.\
   여기에서는 예로 이름을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1367).png" alt=""><figcaption></figcaption></figure>

클릭한 항목의 내림차순으로 폴더와 파일 목록이 정렬되고 항목 이름 옆에 표시된 버튼이 ∨로 바뀝니다.

<figure><img src="../../.gitbook/assets/image (1369).png" alt=""><figcaption></figcaption></figure>



3. 다시 2단계와 같은 항목을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1370).png" alt=""><figcaption></figcaption></figure>

다시 클릭한 항목의 오름차순으로 폴더와 파일 목록이 정렬됩니다.\
항목 옆에 표시된 버튼이 “∧”로 바뀝니다.

<figure><img src="../../.gitbook/assets/image (1371).png" alt=""><figcaption></figcaption></figure>



4. 그런 다음 3단계, 4단계와 다른 항목 이름을 클릭합니다.\
   여기에서는 예로 '크기'를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1372).png" alt=""><figcaption></figcaption></figure>

이미 적용된 항목의 '정렬'이 취소되고 새로 클릭한 항목의 '오름차순'으로 목록이 정렬됩니다.\
적용된 항목이 내림차순이면 새로 클릭한 항목도 내림차순으로 표시됩니다.

<figure><img src="../../.gitbook/assets/image (1373).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**정렬 대상 항목**
{% endhint %}

정렬의 기본 항목에는 '이름', '크기', '업데이트 날짜 및 시간' 또는 '등록 날짜 및 시간', '확장자'가 있습니다.\
헤더에 표시된 항목을 클릭하면 오름차순 또는 내림차순으로 정렬됩니다.\
이 절에서는 정렬할 헤더 항목을 클릭할 때 오름차순/내림차순 표시 순서에 대해 설명합니다.
