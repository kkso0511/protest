# 재해 대비·원격지 백업 사이트에서 파일을 다운로드하는 방법

### 개요

이 매뉴얼 에서는 재해 대비·원격지 백업 옵션을 계약한 환경에서 재해 대비·원격지 백업 사이트에 로그인하여 파일을 다운로드하는 방법을 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자 권한이 부여된 사용자는 관리자 페이지에서 '용량 관리' 항목에 접근가능 합니다.
* 재해 대비·원격지 백업 사이트를 이용하려면 DirectCloud의 유료 옵션인 재해 대비·원격지 백업을 계약해야 합니다.\
  자세한 내용은 DirectCloud에문의해주시기 바랍니다.
* 재해 대비·원격지 백업 사이트에서 가능한 작업은 파일 다운로드뿐입니다.\
  파일 업로드나 온라인 편집에는 대응하지 않으므로 파일 버전 관리는 할 수 없습니다.
* DirectCloud의 재해 대비·원격지 백업 사이트는 재해 등의 사유로 본 서비스 사이트에 로그인할 수 없는 경우에도 파일을 계속 사용할 수 있도록 하는 것을 목적으로 하기 때문에, 재해 대비·원격지 백업 사이트에서는 파일이 자동으로 복원되지 않습니다.\
  백업 사이트에서 다운로드하여 수정한 파일은 본 서비스 사이트가 복구된 후 직접 업로드해야 합니다.
* 관리자 권한이 없는 사용자가 재해 대비·원격지 백업 사이트에 로그인하려면, 관리 페이지의 Storage > Storage 용량 메뉴에서 재해 대비·원격지 백업 항목의 사용자 접근 허가 설정이 사용 또는 장애·재해 시에만 사용으로 설정되어 있어야 합니다.\
  자세한 내용은 관리자 권한이 없는 사용자에게 재해 대비·원격지 백업 사이트 접근을 허용·비허용으로 설정하는 방법을 참조해주시기 바랍니다.
* 다음에 해당하는 경우에는 재해 대비·원격지 백업 사이트에 접근할 수 없습니다.
  * 오전 4:00\~6:00 사이(재해 대비·원격지 백업 사이트와 본 서비스 사이트 간 데이터베이스 동기화가 실행되는 시간대)
  * 사전에 공지된 업데이트 및 유지보수로 인한 서버 일시 중지
  * 복구 시간이 1시간 이내로 예상되는 일시적 장애
* 오사카 리전 및 싱가포르 리전의 재해 대비·원격지 백업 사이트와 도쿄 리전의 본 서비스 사이트 간 데이터베이스 동기화는 오전 4:00\~6:00에 실행됩니다.\
  이 때문에 DirectCloud 관리 페이지에서 설정한 내용이 재해 대비·원격지 백업 사이트에 즉시 반영되지는 않습니다.\
  상기 시간대의 동기화 실행 이후 재해 대비·원격지 백업 사이트에 설정이 반영되었는지 확인해주시기 바랍니다.
* 재해 대비·원격지 백업 사이트에서는 Shared Box 메뉴 아래에 저장된 파일을 다운로드할 수 있습니다.
* DirectCloud-CONNECT를 이용할 수 있는 경우, 재해 대비·원격지 백업 사이트의 Connect 메뉴에 저장된 파일을 다운로드할 수 있습니다.
* Connect 메뉴에서 다운로드할 수 있는 파일은 자사에서 생성한 폴더에 저장된 파일만 해당됩니다.
* Warm Storage를 사용할 수 있는경우 재해 대비·원격지 백업 사이트의 Warm Storage 메뉴에 저장된 파일을 다운로드할 수 있습니다.\
  단, Warm Storage에서 파일을 다운로드하려면 유료 옵션을 계약해야 하며, 또한 Warm Storage의 다운로드 기능이 활성화되어 있어야 합니다.
* 재해가 발생하여 DirectCloud 본 서비스 사이트를 사용할 수 없게 되면 Warm Storage 다운로드 기능(유료 옵션)을 계약하지 않은 경우에도 재해 대비·원격지 백업 사이트에서 Warm Storage 파일을 다운로드할 수 있게 됩니다.
* Cold Storage를 계약한 경우, 재해 대비·원격지 백업 사이트에 Cold Storage 메뉴가 표시됩니다.\
  그러나 재해 대비·원격지 백업 사이트에서는 Cold Storage 파일을 다운로드할 수 없습니다.
* 재해 대비·원격지 백업 사이트는 Web 브라우저에서만 로그인할 수 있습니다.
* 파일을 다운로드하기 위한 접근 권한 및 접근 권한 레벨은 본 서비스 사이트의 설정을 따릅니다.\
  단, DirectCloud 관리 페이지에서 변경된 설정은 재해 대비·원격지 백업 사이트에 즉시 반영되는 것이 아니라, 도쿄 리전 데이터베이스와의 동기화 후에 반영됩니다.\
  앞서 안내한 동기화 시간대(오전 4:00\~6:00)에 유의해주시기 바랍니다.
*   재해 대비·원격지 백업 사이트에서는 다음 항목을 선택하여 다운로드할 수 없습니다.

    * 폴더
    * 복수 파일

    다만, 재해가 발생하여 본 서비스 사이트를 이용할 수 없게 된 경우에는 여러 파일을 선택하여 다운로드할 수 있으며, 이때 ZIP 형식으로 압축되어 다운로드됩니다.

    기타 파일 다운로드 관련 제한 및 참고 사항은 Web 앱·드라이브 파일 다운로드 방법을 참조해주시기 바랍니다.
* DirectCloud-SHIELD IRM으로 암호화된 파일을 재해 대비·원격지 백업 사이트에서 다운로드하는 경우, 암호화가 해제된 후 원본 상태 그대로 다운로드됩니다.
* 재해 대비·원격지 백업 사이트의 검색에서는 검색 범위로 My Box, Shared Box, 문서관리, Warm Storage, Cold Storage를 지정하여 폴더 및 파일을 검색할 수 있습니다.\
  검색 절차는 Web 앱·드라이브에서 폴더 및 파일을 이름이나 문서 내 텍스트 등으로 검색하는 방법을 참조해주시기 바랍니다.

### 절차

1.  재해 대비·원격지 백업 사이트의 URL에 접속한 뒤, 계약 중인 회사 ID, 사용자 ID, 비밀번호를 입력하여 로그인합니다.<br>

    **NOTE**:\
    재해 대비·원격지 백업 옵션을 계약할 때 선택한 리전에 따라 접속 URL이 다릅니다.\
    오사카 리전: [https://osa.directcloud.jp/](https://osa.directcloud.jp/)\
    싱가포르 리전: [https://sgp.directcloud.jp/](https://sgp.directcloud.jp/)\
    \
    재해 대비·원격지 백업 사이트에 로그인하면 “재해 대비·원격지 백업 사이트에 로그인하고 있습니다. 파일 다운로드만 지원합니다.”라는 메시지가 표시됩니다.
2.  다운로드할 파일이 저장된 폴더로 이동한 뒤, 다운로드하려는 파일에 체크하고 다운로드 버튼을 클릭합니다.<br>

    **NOTE**:\
    오른쪽 클릭 메뉴 또는 오른쪽의 ▼ 버튼을 눌러 표시되는 메뉴에서 다운로드를 선택할 수도 있습니다.
