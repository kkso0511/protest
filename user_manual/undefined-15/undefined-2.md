# 링크 생성 화면에 링크 복사 및 로컬 메일 아이콘이 표시되지 않는 경우 해결 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 '링크 생성' 화면에 '링크 복사'와 '외부 메일러 열기' 아이콘이 표시되지 않는 경우의 대처 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* '링크 복사버튼'과 '로컬메일 발송 버튼' 아이콘이 표시되도록 설정하려면, 관리자 기능 제한에서 '보안 정책'에 접근 가능해야 합니다.

***

### 절차 <a href="#a05" id="a05"></a>

'링크 생성'화면에 '링크 복사버튼', '로컬메일 발송 버튼'아이콘이 표시되지 않는 경우 관리 페이지에서\
&#x20;'보안 정책' > '반출 정책' 메뉴의 '링크 발송방법 제어'가 다음과 같이 설정 되었기 때문입니다.

* '링크 발송방법 제어'에서 '링크 복사버튼 표시'에 체크가 되어 있지 않은 경우
* '링크 발송방법 제어'에서 '로컬메일 발송 버튼 표시'에 체크가 되어 있지 않은 경우
* '수신자 인증 의무사용'에서 '사용함'이 선택되어 있는 경우<br>

아래 절차에 따라 설정 상태를 확인하고, 아이콘이 표시되도록 설정해 주시기 바랍니다.

1. [링크 생성 화면에서 링크 복사를 사용하지 않도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-1/undefined-12)의 절차에 따라, '링크 복사'에 체크가 해제되어 있지 않은지 확인합니다.\
   체크가 해제되어 있는 경우에는 '링크 복사'에 다시 체크합니다.
2. [링크 생성 화면에서 로컬메일 사용을 제한하지 않도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-1/undefined-13)의 절차에 따라, '외부 메일러 열기'에 체크가 해제되어 있지 않은지 확인합니다.\
   체크가 해제되어 있는 경우에는 '외부 메일러 열기'에 다시 체크합니다.
3. '링크 발송방법 제어'의 '링크 복사버튼 표시'와 '로컬메일 발송 버튼 표시'가 회색으로 비활성화되어 있는 경우에는, '수신자 인증 의무사용'을 '사용안함'으로 선택합니다.
