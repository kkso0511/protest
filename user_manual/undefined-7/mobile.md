# \[Mobile] 폴더나 파일을 이름이나 문서내 텍스트 등으로 검색하는 방법

### 개요 <a href="#a03" id="a03"></a>

iOS 또는 Android 애플리케이션의 검색 기능에서는 검색 조건을 설정하여 My Box나 Shared Box에 저장된 폴더와 파일을 이름이나 태그로 검색할 수 있습니다.\
또한 문서 내에 포함된 텍스트를 대상으로 전체 텍스트 검색도 가능합니다.\
이 매뉴얼에서는 iOS 애플리케이션과 Android 애플리케이션에서 검색을 수행하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* iOS 또는 Android 애플리케이션에서는, 이 매뉴얼의 절차에 따라 설정한 검색 조건이 DirectCloud에 로그인되어 있는 동안 유지됩니다.\
  DirectCloud에서 로그아웃하면 설정한 검색 조건은 초기화됩니다.
* 폴더나 파일을 업로드한 사용자 이름으로는 검색할 수 없습니다.
* 검색 대상에서 ‘문서 내 텍스트’를 선택해 전체 검색을 수행하면, ‘AND 검색’을 선택했더라도 아래와 같은 결과가 표시될 수 있습니다.\
  예시) ‘클라우드 스토리지 추천’으로 검색한 경우
  * 문서 내에 ‘클라우드 스토리지’와 ‘추천’이 모두 포함된 문서가 검색 결과에 표시됨
  * 문서 내에 ‘클라우드 스토리지’만 포함된 문서가 검색 결과에 표시됨
  * 문서 내에 ‘추천’만 포함된 문서가 검색 결과에 표시됨
  * 문서 내에 ‘클라우드’, ‘스토리지’, ‘추천’이 각각 포함된 문서가 검색 결과에 표시됨
* 검색 결과는 검색 대상별로 표시할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**모바일 애플리케이션으로 검색**
{% endhint %}

1. 'My Box' 또는 'Shared Box' 바로 아래의 폴더를 표시하고 상단의 '검색' 영역을 탭합니다.\
   검색 화면이 표시됩니다.
2. 검색 설정 버튼을 탭합니다.\
   검색 조건 설정이 표시됩니다.
3. 아래 항목을 설정하고 오른쪽 상단의 '닫기'를 탭합니다.

4. 검색어를 입력하고 키보드의 오른쪽 하단에 있는 '검색' 버튼을 탭합니다.\
   검색 결과가 표시됩니다.
5. 검색 설정을 수정하고 검색하려면 '검색 설정'을 다시 탭합니다.\
   3단계에서 설정한 내용이 표시됩니다.
6. 검색 설정을 수정하고 검색 키워드를 다시 입력한 다음 키보드의 오른쪽 하단에 있는 검색 버튼을 탭합니다.



**\[검색 조건 설정 저장]**

* 검색 조건을 저장하려면\
  \
  검색 결과 상단에 표시되는 ‘+’ 버튼을 탭하면, 해당 검색 조건이 ‘빠른 검색(Quick Search)’에 추가됩니다.\
  이후 검색 화면을 다시 열고 ‘빠른 검색’에 등록된 검색 조건을 탭하면, 바로 검색 결과가 표시됩니다.\
  이 설정은 로그아웃해도 삭제되지 않습니다.\
  삭제하려면, 추가된 검색 조건의 오른쪽 끝에 표시된 ‘✕’ 버튼을 클릭하세요.
* 검색 조건의 설정 내용을 유지하고 싶은 경우\
  \
  설정한 검색 조건은 DirectCloud에서 로그아웃하면 초기화됩니다.\
  로그아웃하더라도 설정 내용을 유지하려면, ‘검색 설정’ 화면 하단에 표시된 ‘설정 저장’ 토글 스위치를 켜면 됩니다.
