# 폴더를 다운로드하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 Web 브라우저나 DirectCloud 드라이브를 사용하여 DirectCloud에 저장된 폴더를 다운로드하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼에서는 Shared Box에서 작업하는 방법에 대해 설명하지만 Shared Box 이외의 폴더에 대해서도 비슷한 절차를 수행할 수 있습니다.
* 2계층 이후의 폴더를 다운로드할 수 있습니다.
* 폴더를 다운로드하는 사용자에게는 <마스터>, <전체권한>, <편집자>, <다운로더>의 접근권한이 부여되어야 합니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/web), [접근 권한의 종류와 DirectCloud 드라이브 에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/directcloud)을 참조하십시오.
* 폴더는 ZIP 형식의 파일로 압축되어 다운로드 됩니다.
* Web 브라우저에서는 드래그 앤 드롭으로 폴더를 다운로드할 수 없습니다.\
  PC Agent 또는 DirectCloud 드라이브를 사용하십시오.
* Shared Box, Connect, DLP, Warm Storage의 1계층 폴더를 모두 선택하여 폴더·파일을 일괄 다운로드할 수는 없습니다.
* 다운로드가 완료되더라도 알림 메일은 전송되지 않습니다.
* 워크플로우의 다운로드는 파일만을 대상으로 하기 때문에, 워크플로우를 활성화하는 방법에서 워크플로우의 기본 설정이 사용하기에 체크되어 있고, 워크플로우에 포함되지 않은 사용자가 제한되도록 설정되어 있는 경우에는 폴더를 선택하여 다운로드할 수 없습니다.
* DirectCloud-CONNECT를 이용할 수 있는 경우, 사용자 페이지에 Connect 메뉴가 표시됩니다.
* DirectCloud-SHIELD DLP를 계약한 경우, 사용자 페이지에 DLP 메뉴가 표시되며 DirectCloud 드라이브에 DLP 폴더가 표시됩니다.
* DLP 폴더에 저장된 폴더를 다운로드하려면, DLP 설정에서 사용자에게 다운로드를 허용하도록 설정하는 방법의 절차에 따라 다운로드 설정의 사용하기에 체크되어 있어야 합니다.
* Warm Storage를 이용할 수 있는경우, 사용자 페이지에 Warm Storage 메뉴가 표시되고 DirectCloud 드라이브에 Warm Storage 폴더가 표시됩니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 폴더 다운로드**
{% endhint %}

1. 다운로드할 폴더가 저장된 폴더를 클릭합니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 173913.png" alt=""><figcaption></figcaption></figure>



2. 다음 방법 중 하나로 폴더를 다운로드합니다.

❶ 다운로드하려는 폴더에 체크를 하고 상단의 '다운로드' 버튼을 클릭합니다. 여러 폴더와 파일을 선택할 수 있습니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 174305.png" alt=""><figcaption></figcaption></figure>

\
❷ 다운로드하고자 하는 폴더를 체크하고 우클릭하여 표시된 메뉴에서 \[다운로드]를 클릭합니다. 여러 폴더와 파일을 선택할 수 있습니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 174633.png" alt=""><figcaption></figcaption></figure>

\
❸ 다운로드할 폴더의 오른쪽에 있는 ▼ 버튼을 클릭한 후 표시된 메뉴에서 \[다운로드]를 클릭합니다. 여러 폴더와 파일을 선택할 수 없습니다.\
\
선택한 폴더가 ZIP 형식의 압축 파일로 다운로드됩니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 174740.png" alt=""><figcaption></figcaption></figure>



3. 압축 해제 소프트웨어를 사용하여 다운로드한 파일의 압축을 풉니다.\
   다운로드한 폴더가 표시됩니다.



{% hint style="warning" %}
**DirectCloud 드라이브에서 폴더 다운로드**
{% endhint %}

관리자 페이지의 '워크플로우' > '설정' 에서 워크플로우 기본 설정이 사용 안 함으로 설정된 경우 드래그 앤 드롭으로 폴더를 다운로드할 수 있습니다.

1. 다운로드할 폴더가 저장된 폴더를 클릭합니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 175156.png" alt=""><figcaption></figcaption></figure>



2. DirectCloud에 폴더를 남겨둔 채로 다운로드하려면 폴더를 드래그하여 PC의 폴더나 바탕 화면에 놓습니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 175534.png" alt=""><figcaption></figcaption></figure>

&#x20;      폴더가 복사됩니다



3. DirectCloud에 폴더를 남기지 않고 이동하려면 "Shift" 키를 누른 상태에서 PC의 폴더나 바탕 화면에 폴더를 끌어다 놓습니다.

<figure><img src="../../.gitbook/assets/스크린샷 2025-09-29 175739.png" alt=""><figcaption></figcaption></figure>

&#x20;     폴더가 이동됩니다
