# \[Mobile] 사용자 페이지에서 제 2 계층 이하의 폴더를 생성하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 앱 또는 Android 앱에서 2계층 이하의 폴더를 만드는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 2계층 이하 폴더를 생성하는 사용자에게는 <마스터> 혹은 <전체권한> 의 접근 권한이 부여되어 있어야 합니다.
* Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자, Connect User도 2계층 이하 폴더를 생성할 수 있습니다.
* 생성한 폴더에는 상위 폴더에 부여된 사용자의 접근 권한이 그대로 설정됩니다.
* 폴더 이름은 190자 이내로 입력할 수 있습니다.
* 폴더 이름에 사용할 수 있는 문자와 사용할 수 없는 문자는 다음과 같습니다.
  * 사용 가능한 문자\
    UTF-8의 한굴,  한자, 영대 소문자, 숫자, 공백, 다국어 문자, 기호 및 이모티콘 등
  * 사용할 수 없는 문자\
    \ / : \* ? “ < > |
* 만들 폴더의 이름에 이전 문자의 문자가 포함되어 있으면 일부 문자는 자동으로 새 문자로 변환됩니다.
* 생성할 수 있는 폴더의 깊이는 최대 65계층입니다.
* 하나의 폴더 안에 대량의 폴더나 파일을 저장할 경우 성능이 저하되므로, 한 폴더 내 폴더와 파일의 총합이 2,000개 이하가 되도록 유지해 주시기 바랍니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 두 번째 계층 이상의 폴더 만들기**
{% endhint %}

1. 공유 메뉴를 클릭하여 폴더를 만들려는 첫 번째 계층 이후의 폴더를 표시합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000uVF\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)'을 탭한 다음 표시된 메뉴에서 '폴더 만들기'를 탭합니다.\
   "작성할 폴더 이름을 입력하십시오."라는 화면이 나타납니다.
3. 폴더 이름을 입력하고 확인을 탭합니다.\
   폴더가 생성됩니다.



{% hint style="warning" %}
**Android 애플리케이션에서 두 번째 계층 이상의 폴더 만들기**
{% endhint %}

1. 공유 메뉴를 클릭하여 폴더를 만들려는 첫 번째 계층 이후의 폴더를 표시합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000uVF\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)'을 탭한 다음 표시된 메뉴에서 '폴더 만들기'를 탭합니다.\
   "폴더 만들기"라는 화면이 표시됩니다.
3. 폴더 이름을 입력하고 확인을 탭합니다.\
   폴더가 생성됩니다.
