# \[Mobile] 폴더를 삭제하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는, iOS 애플리케이션 또는 Android 애플리케이션에서 DirectCloud에 저장되어 있는 폴더를 삭제하는 방법을 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 모바일 애플리케이션에서는 1계층 폴더를 삭제할 수 없습니다.
* 2단계 이후의 폴더를 삭제하는 사용자에게는 <마스터>, <전체권한>의 접근 권한이 부여되어 있어야 합니다.
* 삭제 권한이 부여된 경우, 접근 권한이 없는 하위 폴더를 포함하더라도 상위 폴더를 삭제할 수 있습니다.
*   삭제된 폴더는 저장되어 있던 위치에 따라 삭제한 사용자의 휴지통 탭으로 이동합니다.

    * My Box 및 Shared Box의 폴더:\
      사용자의 휴지통 내 My Box / Shared Box / / DLP 탭으로 이동
    * Connect의 폴더:\
      사용자의 휴지통 내 Connect 탭으로 이동

    단, 사용자의 휴지통 탭 이름은 이용 가능한 옵션에 따라 변경될 수 있습니다.
* 사용자의 휴지통에 저장된 폴더를 삭제하면 관리자의 휴지통으로 이동합니다.
* 관리자 페이지에서는 '용량 관리'> '스토리지 현황' 메뉴에서 사용자의 휴지통에 저장된 폴더를 확인할 수 있습니다.
* 폴더를 한 번에 삭제할 때 삭제 가능한 폴더 수에는 제한이 없습니다.
* 삭제 상태 화면에는 세션 시간이 없으므로 처리 시간이 길어져도 타임아웃이 발생하지 않습니다.
*   삭제 처리 중 네트워크 오류가 발생한 경우에는\
    “삭제 중 타임아웃이 발생하여 더 이상 진행할 수 없습니다.”\
    라는 오류 메시지가 표시됩니다.

    이 경우 오류 메시지가 표시되기 전까지 처리된 폴더와 해당 폴더의 직속 하위 폴더는 삭제되지만, 처리 대기 중인 폴더는 삭제되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 폴더 삭제**
{% endhint %}

**폴더 삭제**

1. 삭제할 폴더가 저장된 폴더를 탭합니다.
2. 삭제할 폴더의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000RMg\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 삭제를 탭합니다.\
   확인 화면이 표시됩니다.
4. '확인'을 탭합니다.\
   삭제가 시작되고 상태 화면이 표시됩니다.\
   삭제가 완료되면 "삭제했습니다."라는 화면이 나타납니다.
5. '확인'을 탭합니다.\
   폴더가 삭제됩니다.<br>

**여러 폴더 삭제**

1. 삭제할 파일이 저장된 폴더를 탭합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000RMg\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)' 버튼을 탭한 다음 표시된 메뉴에서 '복수 선택'을 탭합니다.\
   폴더와 파일의 왼쪽에 선택 버튼이 표시됩니다.
3. 삭제할 폴더를 선택하고 오른쪽 하단의 휴지통 버튼을 탭합니다.\
   확인 화면이 표시됩니다.
4. '확인'을 탭합니다.\
   삭제가 시작되고 상태 화면이 표시됩니다.\
   삭제가 완료되면 "삭제했습니다."라는 화면이 나타납니다.
5. '확인'을 탭합니다.\
   여러 폴더가 삭제됩니다.



{% hint style="warning" %}
**Android 애플리케이션에서 폴더 삭제**
{% endhint %}

**폴더 삭제**

1. 삭제할 폴더가 저장된 폴더를 탭합니다.
2. 삭제할 폴더의 오른쪽 끝에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0BB0000000RMg\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 삭제를 탭합니다.\
   확인 화면이 표시됩니다.
4. '확인'을 탭합니다.\
   삭제가 시작되고 상태 화면이 표시됩니다.\
   삭제가 완료되면 "삭제했습니다."라는 화면이 나타납니다.
5. '확인'을 탭합니다.\
   폴더가 삭제됩니다.<br>

**여러 폴더 삭제**

1. 삭제할 폴더가 저장된 폴더를 탭합니다.
2. 원하는 폴더나 파일을 길게 누릅니다.\
   폴더와 파일을 여러 개 선택할 수 있습니다.
3. 삭제하려는 여러 폴더를 선택하고 오른쪽 하단의 휴지통 버튼을 탭합니다.\
   확인 화면이 표시됩니다.
4. '확인'을 탭합니다.\
   삭제가 시작되고 상태 화면이 표시됩니다.\
   삭제가 완료되면 "삭제했습니다."라는 화면이 나타납니다.
5. '확인'을 탭합니다.\
   여러 폴더가 삭제됩니다.
