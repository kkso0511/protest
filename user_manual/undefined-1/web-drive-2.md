# \[Web, Drive] 사용자 페이지에서 제 2 계층 이하의 폴더를 생성하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지에서 2계층 이하의 폴더를 생성하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 2계층 이하 폴더를 생성하는 사용자에게는 <마스터> 혹은 <전체권한> 의 접근 권한이 부여되어 있어야 합니다.
* Guest, LDAP 연동 및 SAML 연동으로 등록된 사용자, Connect User도 2계층 이하 폴더를 생성할 수 있습니다.
* DirectCloud 드라이브에서는 Connect 폴더가 표시되지 않습니다.\
  따라서 DirectCloud 드라이브에서는 Connect 폴더의 2계층 이하 폴더를 생성할 수 없습니다.
* 생성한 폴더에는 상위 폴더에 부여된 사용자의 접근 권한이 그대로 설정됩니다.
* 생성할 수 있는 폴더의 깊이는 최대 65계층입니다.
* 하나의 폴더 안에 대량의 폴더나 파일을 저장할 경우 성능이 저하되므로, 한 폴더 내 폴더와 파일의 총합이 2,000개 이하가 되도록 유지해 주시기 바랍니다.
* 폴더 이름은 190자 이내로 입력할 수 있습니다.
* Mac의 폴더 이름 길이는 255바이트로 제한됩니다. Mac 버전의 DirectCloud 드라이브를 사용하는 경우 주의해 주시기 바랍니다.
* 폴더 이름에 사용할 수 있는 문자와 사용할 수 없는 문자는 다음과 같습니다.
  * 사용할 수 있는 문자\
    UTF-8의 한자, 히라가나, 영문 대소문자, 숫자, 공백, 다국어 문자, 기호 및 이모지 등
  * 사용할 수 없는 문자\
    \ / : \* ? “ < > |
*   생성하는 폴더 이름에 구자체 문자가 포함된 경우 일부 문자는 자동으로 신자체로 변환될 수 있습니다.

    폴더 이름 끝에 반각 마침표(.) 또는 공백 문자가 포함된 폴더를 생성한 경우, Windows의 제한으로 인해 DirectCloud 드라이브에서는 해당 폴더를 사용할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저의 'Shared Box'에서  2계층 이하의 폴더 만들기**
{% endhint %}

1. 'Shared Box' 메뉴를 클릭하여 폴더를 만들려는 첫 번째 계층 이후의 폴더를 표시합니다.

<figure><img src="../../.gitbook/assets/image (1261).png" alt=""><figcaption></figcaption></figure>



2. 다음 방법 중 하나를 사용하여 새 폴더 화면을 표시합니다.

&#x20;     ❶ '새 폴더' 버튼을 클릭

<figure><img src="../../.gitbook/assets/image (1262).png" alt=""><figcaption></figcaption></figure>

&#x20;    ❷  폴더 / 파일 표시 영역에서 마우스 오른쪽 버튼을 클릭하고 표시된 메뉴에서 "새 폴더"를 클릭하십시오.

<figure><img src="../../.gitbook/assets/image (1263).png" alt=""><figcaption></figcaption></figure>

&#x20;     새 폴더 화면이 표시됩니다.



3. 폴더 이름을 입력하고 생성 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1264).png" alt=""><figcaption></figcaption></figure>



4. 폴더가 만들어졌는지 확인합니다.

<figure><img src="../../.gitbook/assets/image (1265).png" alt=""><figcaption></figcaption></figure>



{% hint style="warning" %}
**DirectCloud 드라이브의 "Shared Box"에서 2 계층 이하의 폴더 만들기**
{% endhint %}

1. 새 폴더를 생성하려는 폴더를 표시합니다.

<figure><img src="../../.gitbook/assets/image (1267).png" alt=""><figcaption></figcaption></figure>



2. 오른쪽 마우스를 클릭해 '새로 만들기' > '폴더' 를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (1268).png" alt=""><figcaption></figcaption></figure>



3. 폴더 이름을 입력하고 Enter 키를 누릅니다.

<figure><img src="../../.gitbook/assets/image (1269).png" alt=""><figcaption></figcaption></figure>

폴더가 생성됩니다.
