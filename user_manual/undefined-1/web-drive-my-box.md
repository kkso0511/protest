# \[Web, Drive] My Box를 이용하는 방법

### 개요 <a href="#a03" id="a03"></a>

DirectCloud에는 사용자 간 안전한 파일 공유에 적합한 Shared Box(공용 데이터 영역) 외에도, 사용자가 자신의 파일을 보관할 수 있도록 제공되는 My Box(개인 데이터 영역)가 있습니다.\
이 매뉴얼에서는 My Box를 이용하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 관리자가 [특정 그룹이나 사용자에게 My Box 사용을 허용하지 않도록 설정하는 방법](https://help.directcloud.net/admin_manual/images-and-media/my-box)으로 My Box 기능이 제외되어 있는 경우에는 My Box를 사용할 수 없습니다.
* 삭제된 사용자의 My Box는 복원할 수 없습니다.
* 사용자 페이지에서는 My Box의 파일을 자동 삭제하도록 설정할 수 없습니다. 관리 페이지에서는 [My Box의 파일을 자동으로 삭제하도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-8/my-box)에 따라 자동 삭제 설정을 적용할 수 있습니다.
* 아래 절차로 링크 생성 기능을 활성화하면 Shared Box와 My Box 모두에서 링크 생성이 활성화됩니다. My Box의 링크 생성만 개별적으로 비활성화할 수는 없습니다.
  * [그룹에 할당된 사용 권한을 변경하는 방법](https://help.directcloud.net/admin_manual/undefined-2/undefined-3)
  * [특정 사용자에게 예외적인 사용 권한을 부여하는 방법](https://help.directcloud.net/admin_manual/undefined-3/undefined-5)
* [사용자에게 할당된 My Box 용량을 변경하는 방법](https://help.directcloud.net/admin_manual/undefined-8/my-box-3)의 절차로 설정된 할당 용량을 초과하는 파일은 링크로 업로드할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**My Box 표시**
{% endhint %}

1. 사용자 페이지에서 My Box 메뉴를 클릭합니다.\
   My Box가 표시됩니다.



{% hint style="warning" %}
**My Box에서 할 수 있는 작업**
{% endhint %}

'My Box'에서는 'Shared Box'와 비슷한 작업을 수행할 수 있습니다.\
그러나 다음 기능은 사용할 수 없습니다.

* 폴더 속성 표시
* 업로드 요청
* Guest 초대
* 파일 잠금
