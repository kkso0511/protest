# \[Mobile] 폴더를 복사하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 또는 Android 애플리케이션에서 DirectCloud에 저장된 폴더를 DirectCloud의 다른 폴더에 복사하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 폴더를 복사하는 사용자에게는 <마스터>, <전체권한>, <편집자>의 접근 권한이 부여되어야 합니다.\
  자세한 내용은 [접근 권한의 종류와 Mobile 앱 / Mobile Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/mobile-mobile-web)을 참고해 주시기 바랍니다.
* 복사 화면에서 대상 폴더에 새 폴더를 만들려면 <마스터>, <전체권한>의 접근 권한이 부여되어야 합니다.
* 여러 개의 폴더·파일에 체크되어 있는 경우에는 위에서부터 순서대로 복사가 실행됩니다.
* 복사 대상 폴더에 동일한 이름의 폴더·파일이 있는 경우에는 '이미 같은 파일명이 존재하여 진행할 수 없습니다.' 라는 메시지가 표시되며 복사할 수 없습니다.
* 이미 생성된 링크가 있는 파일을 포함한 폴더를 복사하는 경우, 파일의 링크 정보는 복사되지 않지만, 폴더를 이동하는 경우에는 파일의 링크 정보가 유지됩니다.
* 부하로 인해 처리 속도가 저하될 수 있으므로, DirectCloud에서는 한 번에 1,000개 이상의 폴더 및 파일을 복사하는 것을 권장하지 않습니다.
*   여러 폴더와 파일을 선택하여 복사할 때, 한 번에 복사할 수 있는 하위 폴더 수는 총 10,000개입니다.

    10,001개 이상의 하위 폴더를 가진 부모 폴더를 복사하려고 하면,\
    “복사/이동 제한을 초과하여 더 이상 진행할 수 없습니다. (하위폴더 10,000개 초과).”\
    라는 오류 메시지가 표시됩니다. 이 경우 복사 작업은 중단됩니다.
*   복사 처리 중 상태 화면의 × 버튼을 누르거나, 세션 시간이 10분을 초과한 경우에는,\
    “복사/이동 중 타임아웃이 발생하여 더 이상 진행할 수 없습니다.”\
    라는 오류 메시지가 표시됩니다.

    이때 오류 메시지가 표시되기 전까지 처리된 폴더 및 폴더 내 저장된 폴더·파일은 복사되지만, 처리 대기 상태였던 폴더·파일은 복사되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 폴더 복사**
{% endhint %}

**폴더 복사**

1. 복사할 폴더가 저장된 폴더를 탭합니다.
2. 복사할 폴더의 오른쪽 가장자리에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000w7F\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 복사를 탭합니다.\
   복사 화면이 표시됩니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 새 폴더를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   복사가 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.<br>

**여러 파일 복사**

1. 복사할 폴더가 저장된 폴더를 탭합니다.
2. 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000w7F\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0u3)' 버튼을 탭한 다음 표시된 메뉴에서 '복수 선택'을 탭합니다.\
   폴더와 파일의 왼쪽에 선택 버튼이 표시됩니다.
3. 복사할 폴더를 선택하고 오른쪽 하단의 '' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000w7F\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g5Ni)버튼을 탭합니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 새 폴더를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   복사가 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.



{% hint style="warning" %}
**Android 애플리케이션에서 폴더 복사**
{% endhint %}

**폴더 복사**

1. 복사할 폴더가 저장된 폴더를 탭합니다.
2. 복사할 폴더의 오른쪽 가장자리에 있는 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000w7F\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g0ye)'을 탭합니다.
3. 표시되는 메뉴에서 복사를 탭합니다.\
   복사 화면이 표시됩니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 폴더 만들기를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   복사가 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.<br>

**여러 폴더 복사**

1. 복사할 폴더가 저장된 폴더를 탭합니다.
2. 원하는 폴더나 파일을 길게 누릅니다.\
   폴더와 파일을 여러 개 선택할 수 있습니다.
3. 복사하려는 여러 폴더를 선택하고 오른쪽 하단의 ' ![](https://help.directcloud.jp/servlet/rtaImage?eid=ka0Q80000000w7F\&feoid=00N2w00000JMb2T\&refid=0EMBB000000g5Ni)' 버튼을 누릅니다.
4. 대상 폴더를 탭합니다.
5. 필요한 경우 폴더 만들기를 탭하여 새 폴더를 만듭니다.
6. 오른쪽 상단의 '완료' 버튼을 탭합니다.\
   복사가 시작되고 상태 화면이 표시됩니다.
7. 완료 화면이 표시되면 '확인'을 탭합니다.
