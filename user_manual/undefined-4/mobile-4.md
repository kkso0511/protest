# \[Mobile] 링크가 만료되었거나 삭제되었을 때 알림 메일이 전송되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션의 링크 화면에서, 한 번도 접근이 없었던 링크가 만되거나 삭제된 경우에, 링크의 발신자와 수신자에게 알림 메일이 전송되도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼의 설정은 My Box의 파일을 선택하여 생성한 링크에도 적용됩니다.
* 접속 기록이 한 번도 없는 링크라도 알림 메일은 전송됩니다. 자세한 내용은 [\[Mobile\] 접근 내역이 없는 링크가 비활성화되었거나 삭제되었을 때 알림 메일이 전송되도록 설정하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile-5)을 참조해 주시기 바랍니다.
* 아래에 해당하는 경우, 링크는 만료됩니다.
  * [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에 기재된 링크 화면 기능에서 설정된 링크의 유효 기간이 만료된 경우
  * [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에 기재된 링크 화면 기능에서 설정된 링크 접근 횟수의 상한에 도달한 경우
* 아래 항목 중 하나에 해당하면 링크는 삭제됩니다.
  * [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에서 사용자가 링크를 삭제한 경우
  * [\[Mobile\] 링크가 만료되었을 때 자동으로 삭제되도록 설정하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile-8)의경우
  * [\[Mobile\] 링크 대상 파일이 업데이트 될 때 링크가 삭제되도록 설정하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile-9)의 경우
* 삭제된 링크를 다시 활성화할 수는 없습니다.
* 링크를 생성한 이후, 대상 폴더 또는 파일이 저장된 폴더에 대한 접근 권한이 삭제되면 링크로 접속하더라도 해당 폴더나 파일에 접근할 수 없습니다.
* 링크 생성자의 접근 권한이 삭제되면, 링크 수신자 역시 해당 폴더나 파일에 접근할 수 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 설정하기**
{% endhint %}

1. [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에 따라, 접근이 없는 링크가 만료되거나 삭제되었을 때 알림 메일이 전송되도록 설정하려는 폴더 또는 파일의 링크 화면을 엽니다.



2. 링크 화면을 아래로 스크롤하여 메일 옵션을 탭합니다.\
   이메일 옵션 화면이 표시됩니다.



3. 알림 설정의 '링크 만료/삭제된 경우 알림 메일' 항목에 체크한 후, 오른쪽 위에 있는 완료 버튼을 탭합니다.



{% hint style="warning" %}
**Android 애플리케이션에서 설정**
{% endhint %}

1. [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에 따라, 접근이 없는 링크가 만료되거나 삭제되었을 때 알림 메일이 전송되도록 설정하려는 폴더 또는 파일의 링크 화면을 엽니다.



2. 링크 화면을 아래로 스크롤하여 메일 옵션을 탭합니다.\
   이메일 옵션 화면이 표시됩니다.



3. 알림 설정의 "링크 만료/삭제된 경우 알림 메일" 항목에 체크한 후, 오른쪽 위에 있는 완료 버튼을 탭합니다.



{% hint style="warning" %}
**링크 종료 이메일 확인**
{% endhint %}

링크가 사용 중지되거나 삭제된 경우 보낸 사람과 받는 사람에게 "받은 링크가 종료되었습니다."라는 제목의 알림 메일이 전송됩니다.
