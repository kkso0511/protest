# \[Web, Drive] 링크를 수신할 때, 이메일 주소 입력을 통해 본인 확인을 진행하도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 링크를 받은 사용자가 링크의 수신 화면에 접근할 때, 이메일 주소 입력을 통해 본인 확인을 수행하도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 구버전의 링크 생성 화면에서는 이 설정을 사용할 수 없습니다.
* 이 매뉴얼의 설정은 My Box의 파일을 선택하여 생성한 링크에도 적용됩니다.
* 공유 링크 수신 화면에 접근할 수 있는 것은 링크 생성 화면의 수신자 항목에 입력한 이메일 주소와 일치하는 경우에만 가능합니다.
* 이 매뉴얼의 절차에 따라 메일을 전송하면, 이후 링크 생성 화면에서 이력 중 링크를 선택할 때 링크 복사, 외부 메일 열기, URL 변경 아이콘이 표시되지 않으며 링크 설정을 변경할 수 없게 됩니다.
* 공유 링크 수신 화면에 접근할 때 2단계 인증을 사용해 본인 확인을 수행하도록 설정할 수는 없습니다.
* 메일 옵션 설정은 생성한 링크를 메일로 전송하는 경우에만 적용됩니다. 생성한 링크를 복사하여 사용하는 경우, 예를 들어 SNS 등에 붙여넣어 공유하는 경우에는 적용되지 않습니다.
* 링크 생성 화면에 표시되는 언어는 프로필의 설정 화면에서 변경할 수 있습니다.
* 링크 생성 후, 대상 폴더 또는 파일이 저장된 폴더에 대한 사용자 접근 권한이 삭제되면 링크로 접속하더라도 해당 폴더나 파일에 접근할 수 없습니다.\
  또한 링크를 생성한 사용자의 접근 권한이 삭제되면, 링크 수신자도 해당 폴더나 파일에 접근할 수 없게 됩니다.
* [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/web)또는 [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/dirve)의 상세 설정에서 비밀번호가 설정된 경우, 먼저 비밀번호 입력 화면이 표시되며 이후 수신자 확인 화면이 나타납니다.
*   링크 접근 횟수는 세션 단위로 집계되며, 해당 횟수는 링크를 연 Web 브라우저의 Cookie로 저장됩니다.\
    따라서 동일한 사용자가 같은 세션에서 링크를 10회 열어도 접근 횟수는 1회로 계산됩니다.

    브라우저의 방문 기록을 삭제하거나, 브라우저를 연 시점으로부터 24시간이 지나면 해당 세션은 종료됩니다.
* 전송한 링크를 열 때, 두 번째 이후에도 비밀번호 입력을 요구하도록 설정할 수는 없습니다.
* 링크 생성 화면에서 URL을 클릭해도 링크 수신 화면의 접근 횟수에는 반영되지 않습니다.\
  다만 다음과 같은 경우에는 접근 횟수에 포함됩니다.
  * Google Chrome 이외의 브라우저를 사용하여 별도 탭에서 링크를 연 경우
  * 브라우저를 연 시점으로부터 24시간이 지난 경우

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**Web 브라우저에서 설정하기**
{% endhint %}

1. [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/web)에 따라, 공유 링크를 생성하려는 폴더 또는 파일의 링크 생성 화면을 엽니다.



2. 이메일 옵션을 클릭합니다.

3. '링크 접속 시 수신자 메일주소 입력' 항목에 체크한 후, 이메일 옵션 설정 화면 밖의 영역을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (192).png" alt=""><figcaption></figcaption></figure>

&#x20;      설정은 즉시 저장됩니다.

{% endhint %}

2. 이메일 옵션을 클릭합니다.

3. '링크 접속 시 수신자 메일주소 입력' 항목에 체크한 후, 이메일 옵션 설정 화면 밖의 영역을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (192).png" alt=""><figcaption></figcaption></figure>

2. 링크 메일을 받은 이메일 주소를 입력한 후 확인 버튼을 누릅니다.

<figure><img src="../../.gitbook/assets/image (188).png" alt="" width="563"><figcaption></figcaption></figure>
