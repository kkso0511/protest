# \[Web, Drive] 링크가 만료되었을 때 자동으로 삭제되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 다음과 같은 이유로 발송한 공유 링크가 만료 되었을 때 자동으로 삭제되도록 설정하는 방법을 설명합니다.

* 유효 기간이 만료됨
* 접속 횟수 제한에 도달함

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 본 매뉴얼은 My Box에서 파일을 선택하여 생성한 링크에도 동일하게 적용됩니다.
* 링크가 삭제되는 시점은 링크가 무효가 된 다음 날 0시입니다. 예를 들어, 오후 5시에 링크가 무효가 된 경우, 다음 날 0시(31시간 후)에 삭제됩니다.
* 아래 항목 중 하나라도 해당되면 링크는 삭제됩니다.
  * [\[Web\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/web) 또는 [\[Dirve\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/dirve)에서 사용자가 링크를 삭제한 경우
  * 본 매뉴얼의 절차에 따라 링크가 삭제된 경우
  * [\[Web, Drive\] 링크 대상 파일이 업데이트 될 때 링크가 삭제되도록 설정하는 방법](https://help.directcloud.net/user_manual/undefined-4/web-drive-8)에 따라 링크가 삭제된 경우
  * 링크를 생성한 사용자가 삭제된 경우
  * 사용자가 생성한 링크를 삭제하는 방법을 통해 관리자가 링크를 삭제한 경우
*   링크에 대한 접속 횟수는 세션 단위로 계산되며, 그 횟수는 링크를 열었던 Web 브라우저의 쿠키에 저장됩니다.\
    따라서 한 사용자가 동일한 세션에서 링크를 10회 열었더라도, 접속 횟수는 1회만 기록됩니다.

    브라우저의 방문 기록을 삭제하거나 브라우저를 연 후 24시간이 지나면 해당 세션은 종료됩니다.
* 발송한 링크를 열 때, 두 번째 이후에도 패스워드를 다시 입력하도록 설정하는 기능은 제공되지 않습니다.
* 링크 생성 화면에서 URL을 클릭해도, 링크 수신 화면의 접속 횟수에는 카운트되지 않습니다.\
  단, 아래의 경우에는 접속 횟수가 증가합니다.
  * Google Chrome 이외의 브라우저를 사용하여 다른 탭에서 링크를 열었을 경우
  * 브라우저를 연 후 24시간이 지난 경우
* 링크 생성 화면에서 표시되는 언어는 프로필의 설정 화면에서 변경할 수 있습니다.

***
