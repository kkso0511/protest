# Connect User를 초대하는 방법

### 개요 <a href="#a03" id="a03"></a>

Connect는 DirectCloud의 회사 ID를 보유한 고객사 간에 특정 폴더를 공유하는 기능입니다.\
이 매뉴얼 에서는 사내 사용자가 다른 회사 ID에 소속된 사용자를 Connect User로 초대하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* Connect User를 초대하려는 사용자에게는 <마스터>, <전체권한>의 접근 권한이 부여되어 있어야 합니다.\
  자세한 내용은 [접근 권한의 종류와 Web 브라우저에서 가능한 조작](https://help.directcloud.net/user_manual/undefined-5/web)을 참고해주시기 바랍니다.
* Connect User 초대는 Guest 초대와 달리, 초대 대상 사용자가 DirectCloud의 회사 ID를 보유하고 있어야 합니다.
* 초대받는 고객사에서는 _다른 회사의 DirectCloud 사용자로부터 Connect User 초대를 허용하는 방&#xBC95;_&#xC5D0; 따라 Connect 동기화를 활성화해두어야 합니다.
* 사내 사용자가 생성한 제1계층 폴더의 아이콘은 파란색으로 표시됩니다.\
  또한, 타사의 DirectCloud 사용자에게 초대된 제1계층 폴더의 아이콘은 노란색으로 표시됩니다.\
  제2계층 이후의 폴더는 생성한 DirectCloud 사용자가 자사/타사 여부와 상관없이 폴더 아이콘이 노란색으로 표시됩니다.
* 타사의 DirectCloud 사용자에게 초대된 제1계층 폴더에는 초대한 사용자가 소속된 회사명이 표시됩니다.
* Shared Box에서는 Connect User를 초대할 수 없습니다.
* Connect User 초대 가능 사용자 수에는 제한이 없습니다.
* 메일로 Connect User를 초대하는 경우든, 링크로 초대하는 경우든 Connect User 초대 URL의 유효 기간은 _Connect 등록용 URL의 유효 기간을 변경하는 방&#xBC95;_&#xC5D0;서 설정된 기간이 적용됩니다.
* 수신자, 제목, 내용을 입력한 뒤 Connect User 초대 화면의 오른쪽 상단에 있는 ✕ 버튼 또는 취소 버튼을 클릭하면 설정 내용이 임시 저장됩니다.\
  이후 동일한 폴더에서 Connect User 초대를 다시 시작하고, 표시된 확인 화면에서 확인 버튼을 클릭하면 임시 저장된 설정 내용을 불러올 수 있습니다.
* 비밀번호 끝에 반각 스페이스가 포함된 상태로 복사하여 비밀번호 입력 화면의 입력란에 붙여넣으면, 반각 스페이스는 자동으로 제거됩니다.
* Connect User에게 설정할 수 있는 접근 권한은 사용자 및 그룹과 동일합니다.\
  그러나 Connect User는 파일첨부 메일발송, 업로드 요청, Guest 초대, Guest 초대 승인, 접근 권한 설정, 폴더 속성 설정 기능을 사용할 수 없습니다.
* Connect User는 제1계층 폴더를 조작할 수 없으므로, Connect User에게 <마스터> 권한을 부여할 수 없습니다.
* Connect User 초대 시의 정보는 사용자 페이지에서 확인할 수 없습니다.\
  &#xNAN;_&#xB4F1;록된 Connect User를 확인하는 방&#xBC95;_&#xC758; 절차에 따라 관리 페이지에서 확인할 수 있습니다.
* Connect User 등록 화면의 사용자 ID에는 Connect User 초대 화면에서 수신자로 지정한 메일 주소에 대응하는 사용자 ID를 입력해야 합니다.
* Connect 폴더의 폴더 및 파일 목록은 헤더 항목의 이름, 크기, 갱신 일시, 종류를 클릭하여 오름차순 또는 내림차순으로 정렬할 수 있습니다.\
  자세한 내용은 [\[Web\] 폴더 및 파일 목록을 정렬하는 방법](https://help.directcloud.net/user_manual/undefined-3/web-2)을 참고해주시기 바랍니다.
* Connect 폴더의 제2계층 이하의 파일 목록에는 위 정렬 항목 외에 사용자 항목이 표시됩니다.\
  또한 제1계층 폴더에는 회사명, 접근 레벨, 접근 권한의 유효 기간이 표시됩니다.

***

{% hint style="warning" %}
**메일로 Connect User를 초대하기**
{% endhint %}

이 매뉴얼 에서는 메일로 Connect User를 초대하는 기본적인 절차에 대해 설명합니다.

1. Connect User 초대 화면에서 '메일로 초대' 탭을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (144).png" alt=""><figcaption></figcaption></figure>



2.  받는 사람에 Connect User 초대 메일을 전송할 메일 주소를 입력합니다.\
    입력란 외의 영역을 클릭하면 입력한 메일 주소가 반영됩니다.

    [회사·개인 주소록을 활성화하는 방법](https://help.directcloud.net/admin_manual/undefined-6/undefined-8)의 절차에 따라 주소록이 활성화되어 있는 경우, 오른쪽에 표시된 아이콘을 클릭하여 주소록에서 수신자 메일 주소를 선택할 수도 있습니다.

<figure><img src="../../.gitbook/assets/image (147).png" alt=""><figcaption></figcaption></figure>



3. 필요한 경우 제목과 내용을 수정합니다.

<figure><img src="../../.gitbook/assets/image (148).png" alt=""><figcaption></figcaption></figure>



4. 접근 권한 드롭다운 리스트에서 Connect User에게 할당할 권한을 선택합니다.

<figure><img src="../../.gitbook/assets/image (149).png" alt=""><figcaption></figcaption></figure>



5. 필요한 경우 Connect User의 유효기간을 설정합니다.

<figure><img src="../../.gitbook/assets/image (150).png" alt=""><figcaption></figcaption></figure>



6. 전송될 메일의 내용을 확인하려면 미리보기 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (151).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (152).png" alt=""><figcaption></figcaption></figure>

   메일이 발송 됩니다.

<figure><img src="../../.gitbook/assets/image (154).png" alt=""><figcaption></figcaption></figure>



**\[관리자 승인이 필요한 경우]**

관리자 설정에서 관리자 또는 <마스터>가 승인한 후 Connect User에게 초대 메일을 전송하도록 설정되어 있는 경우, <계약 담당 관리자>, <일반 관리자>, <마스터>가 Connect User 초대를 승인하면 "Connect User 초대가 승인되었습니다." 라는 제목의 알림 메일이 전송됩니다.\
또한 초대된 사용자에게는 Connect User 초대 메일이 전송됩니다.



{% hint style="warning" %}
**링크로 Connect User를 초대하기**
{% endhint %}

이 매뉴얼 에서는 Connect User를 초대하기 위한 URL을 생성하는 기본 절차에 대해 설명합니다.

1. Connect User 초대 화면에서 링크로 초대 탭을 클릭합니다.

<figure><img src="../../.gitbook/assets/image (133).png" alt=""><figcaption></figcaption></figure>



2. 메일 주소에 Connect User로 초대하려는 사용자의 메일 주소를 입력합니다.\
   Enter 키를 누르면, 이미 Connect User로 등록되어 있는 메일 주소인 경우 회사명과 이름이 회색으로 표시됩니다.\
   회색으로 표시된 텍스트 박스에는 문자를 입력할 수 없습니다.

{% hint style="warning" %}
**Connect User 등록 화면에서 Connect User 정보를 등록하기**

메일로 Connect User를 초대하는 절차에서 Connect User를 초대한 경우, 수신자로 설정된 사용자에게 "○○님이 당신을 초대했습니다." 라는 제목의 Connect User 초대 메일이 전송됩니다.\
메일을 받은 사용자는 아래 절차에 따라 Connect User 등록을 진행할 수 있습니다.

1. 다음 중 하나의 방법으로 Connect User 등록 화면을 표시합니다.

➊ Connect User 초대 메일을 열어 Connect User 초대 URL 링크를 클릭합니다.

<figure><img src="../../.gitbook/assets/image (142).png" alt="" width="563"><figcaption></figcaption></figure>



➋ 링크로 Connect User를 초대하는 절차에서 초대된 경우에는, 메일에 포함된 링크를 클릭합니다.



2. 'Connect 사용 시 제공·공유되는 정보에 대해동의합니다에.' 체크한 후, 회사코드, 사용자 ID, 비밀번호를 입력하고 'Connect 연결' 버튼을 클릭합니다.
