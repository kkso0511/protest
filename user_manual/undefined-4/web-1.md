# \[Web] 만료된 링크를 사용자 페이지에서 다시 활성화하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 사용자 페이지의 Link History 메뉴에서 생성된 링크 목록을 열어 링크 생성 화면을 표시한 뒤, 비활성화된 링크를 다시 활성화하는 절차에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 삭제된 링크는 다시 활성화할 수 없습니다.
* [\[Web, Drive\] 링크를 수신할 때, 이메일 주소 입력을 통해 본인 확인을 진행하도록 설정하는 방법](https://help.directcloud.net/user_manual/undefined-4/web-drive-2)의 절차에서 수신자 확인이 활성화되어 있는 경우, 링크의 초기화 버튼은 표시되지 않습니다.

***

### 절차 <a href="#a05" id="a05"></a>

1. 사용자 페이지에서 Link History 메뉴를 클릭합니다.\
   Link History가 표시됩니다.<br>
2.  편집하려는 링크의 오른쪽 끝에 표시된 링크 버튼을 클릭합니다.\
    링크 생성 화면이 표시됩니다.

    비활성화된 링크의 경우, 빨간 글씨로 "링크의 유효 기간이 종료되었습니다." 라고 표시됩니다.<br>
3. '초기화' 버튼을 클릭합니다.<br>
4. 확인 화면에서 확인 버튼을 클릭합니다.\
   링크가 다시 활성화됩니다.
