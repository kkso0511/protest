# \[Mobile] 공유 링크에 접근할 수 있는 횟수를 1회로 제한하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 모바일용 어플리케이션에서 '1회용 미리보기 옵션' 기능을 사용해 공유 링크에 접근할 수 있는 횟수를 1회로 제한하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 사전에 [공유 링크 생성 화면에 1회용 미리보기 옵션이 표시되도록 설정하는 방법](https://help.directcloud.net/admin_manual/undefined-1/1)의 절차에 따라, 1회용 미리보기 기능을 활성화해두어야 합니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 설정하기**
{% endhint %}

1. [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)의 절차에 따라 링크 생성 화면을 표시합니다.



2. 기본 설정 화면에서 '1회용 미리보기'에 체크합니다.\
   설정은 즉시 저장되며, 이미 저장된 설정값과 관계없이 링크 생성 옵션은 아래와 같이 고정됩니다.\
   \
   -수신방법: 프리뷰어\
   -업로드: 사용 안함\
   -만료일: 당일\
   -비밀번호: 항상 사용\
   -접근 횟수 제한: 1회



{% hint style="warning" %}
**Android 애플리케이션에서 설정**
{% endhint %}

1. [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)의 절차에 따라 링크 생성 화면을 표시합니다.



2. 기본 설정 화면에서 '1회용 미리보기'에 체크합니다.\
   설정은 즉시 저장되며, 이미 저장된 설정값과 관계없이 링크 생성 옵션은 아래와 같이 고정됩니다.\

   -비밀번호: 항상 사용\
