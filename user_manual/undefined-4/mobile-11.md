# \[Mobile] 계정이 없는 상대로부터 파일을 받는 방법(업로드 요청 기능)

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼에서는 iOS 애플리케이션 또는 Android 애플리케이션에서 수신 폴더 기능을 활용하여 DirectCloud 계정이 없는 상대로부터 파일을 수신하는 방법에 대해 설명합니다.

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 업로드 요청 기능을 사용하려면, 관리자 페이지의 사용권한 설정에서 ‘File Request’ 항목에 체크되어 있는 권한 세트가 그룹 및 사용자에게 부여되어 있어야 합니다.
* <마스터>, <전체권한>, <편집자>, <편집자－> 접근 권한이 부여된 사용자는 업로드 요청 기능을 이용할 수 있습니다.
* 폴더 업로드를 요청할 수 없습니다.
* '받는 사람', '제목', '내용'을 입력 한 후 '업로드 요청 링크 전송' 화면의 왼쪽 상단에 있는 '닫기' 버튼을 클릭하면 설정 내용이 임시 저장됩니다.\
  이후 동일한 폴더에서 업로드 요청 생성을 다시 시작하고, 표시된 확인 화면에서 ‘확인’ 버튼을 클릭하면 이전에 임시 저장된 설정 내용을 불러올 수 있습니다.
* 사용자 페이지의 업로드 요청 이력 메뉴와 관리자 페이지의 '로그 현황'> '업로드 요청 내역' 메뉴에서는, 비활성화된 업로드 요청 링크를 다시 활성화할 수 없습니다.

\[**업로드 요청 폴더 링크 보내기]**

1. 업로드 요청 화면에서 수신자 항목에 링크를 보낼 이메일 주소를 입력합니다.\
   주소록이 활성화되어 있는 경우, 오른쪽에 표시된 아이콘을 탭하여 주소록에서 수신자 이메일 주소를 선택할 수도 있습니다.
2. 필요에 따라 제목과 내용을 입력합니다.
3. 설정한 내용으로 링크를 전송하려면 ‘전송’ 버튼을 탭합니다.\
   업로드 요청 링크 URL이 전송됩니다.
