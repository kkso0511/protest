# \[Mobile] 링크 대상 폴더 파일이 업데이트 될 때 알림 메일이 전송되도록 설정하는 방법

### 개요 <a href="#a03" id="a03"></a>

이 매뉴얼 에서는 iOS 애플리케이션 또는 Android 애플리케이션의 링크 화면에서, 전송한 링크의 폴더 또는 파일이 업데이트되었을 때 링크의 발신자와 수신자에게 알림 메일이 전송되도록 설정하는 방법에 대해 설명합니다.

***

### 제한·보충사항 <a href="#a04" id="a04"></a>

* 이 매뉴얼의 설정은 My Box의 파일을 선택하여 생성한 링크에도 적용됩니다.
* 폴더 링크를 생성한 경우, 폴더 바로 아래의 파일이 업데이트되면 폴더 정보도 함께 업데이트되므로 알림 메일이 전송됩니다.
* 링크 생성 이후, 대상 폴더 또는 파일이 저장된 폴더에 대한 접근 권한이 삭제되면 링크로 접속하더라도 해당 폴더나 파일에 접근할 수 없습니다.\
  또한 링크 생성자의 접근 권한이 삭제되면, 링크 수신자도 해당 폴더나 파일에 접근할 수 없게 됩니다.
* 링크 수신 화면에서 파일을 미리보기 할 때 표시되는 날짜와 시간은 파일의 업데이트 시각이며, 이 날짜와 시간은 숨길 수 없습니다.\
  동일한 파일명으로 덮어쓰기 하면, 링크 수신 화면에서 표시되는 파일의 업데이트 시각도 변경됩니다.
* [파일을 다운로드하지 않고 온라인으로 편집하는 방법](https://help.directcloud.net/user_manual/undefined-3/undefined)에 따라 파일을 수정하면 파일이 덮어써지면서 업데이트 시각이 변경되지만, 이 매뉴얼의 절차에 따라 알림 메일을 전송하도록 설정할 수는 없습니다.

***

### 절차 <a href="#a05" id="a05"></a>

{% hint style="warning" %}
**iOS 애플리케이션에서 설정하기**
{% endhint %}

1. [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에 따라, 링크 대상 폴더 또는 파일이 업데이트되었을 때 알림 메일이 전송되도록 설정하려는 폴더 또는 파일의 링크 화면을 엽니다.



2. 위의 절차에 따라, 링크 대상 폴더 또는 파일이 업데이트되었을 때 알림 메일이 전송되도록 설정하려는 폴더 또는 파일의 링크 화면을 표시합니다. \
   이메일 옵션 화면이 표시됩니다.



3. '파일(폴더)이 갱신된 경우 알림' 항목을 ON으로 전환한 뒤, 오른쪽 위에 있는 완료 버튼을 탭합니다.



{% hint style="warning" %}
**Android 애플리케이션에서 설정**
{% endhint %}

1. [\[Mobile\] 링크 생성 화면을 표시하여 기본 기능을 확인하는 방법](https://help.directcloud.net/user_manual/undefined-4/mobile)에 따라, 링크 대상 폴더 또는 파일이 업데이트되었을 때 알림 메일이 전송되도록 설정하려는 폴더 또는 파일의 링크 화면을 엽니다.



2. 위의 절차에 따라, 링크 대상 폴더 또는 파일이 업데이트되었을 때 알림 메일이 전송되도록 설정하려는 폴더 또는 파일의 링크 화면을 표시합니다. \
   이메일 옵션 화면이 표시됩니다.



3. '파일(폴더)이 갱신된 경우 알림' 항목을 ON으로 전환한 뒤, 오른쪽 위에 있는 완료 버튼을 탭합니다.



{% hint style="warning" %}
**알림 이메일 확인**
{% endhint %}

링크 대상 폴더 또는 파일이 업데이트되면 링크의 보낸 사람과받는 사람에게 "링크 대상 파일 / 폴더가 업데이트되었습니다."라는 알림 메일이 전송됩니다.
